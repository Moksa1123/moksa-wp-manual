# 1.5 實務導覽：認識後台儀表板 (含「顯示選項」客製化)

嗨，歡迎來到這一堂課！當你第一次登入 WordPress 後台時，看到滿滿的選項和資訊，可能會覺得有點不知所措。別擔心，這是完全正常的！

這個後台，我們稱之為「**儀表板 (Dashboard)**」，它就是你網站的「**指揮中心**」。所有文章的撰寫、商品的上架、訂單的管理、外掛的安裝，都會在這裡完成。

這堂課的目標很簡單：帶你熟悉儀表板的環境，並教你一個非常實用、但常被新手忽略的技巧——「**顯示選項**」，讓你打造一個清爽、專屬於你的工作區。

---

### 儀表板三大核心區域

首先，我們先將整個畫面拆解成三個主要區塊，讓你對整體結構有個概念。

``

1.  **頂部工具列 (Top Admin Bar):**
    *   **位置：** 畫面最上方，橫跨整個寬度。
    *   **功能：** 提供最常用的快捷功能。無論你在後台的哪個頁面，甚至登入後在前台瀏覽網站，它都會存在。
    *   **常見項目：**
        *   WordPress Logo：提供 WordPress 相關的官方資源連結。
        *   你的網站名稱：快速切換到網站前台瀏覽。
        *   留言圖示：快速查看最新留言。
        *   **「+ 新增」**：非常實用！可以快速新增文章、頁面、媒體，未來安裝 WooCommerce 後，也可以在這裡快速新增商品或優惠券。
        *   右側的使用者資訊：編輯你的個人資料或登出。

2.  **左側主選單 (Left Main Menu):**
    *   **位置：** 畫面左邊的深色垂直區域。
    *   **功能：** 這是你網站的**核心導航**，所有管理功能都可以在這裡找到。
    *   **初始項目：**
        *   **文章：** 管理你的部落格文章、分類和標籤。
        *   **媒體：** 你的媒體庫，所有上傳的圖片、影片、文件都在這。
        *   **頁面：** 管理你的「關於我們」、「聯絡我們」等靜態頁面。
        *   **外觀：** 控制網站的佈景主題、選單、小工具等視覺外觀。
        *   **外掛：** 安裝、啟用、停用、刪除外掛的地方，是 WordPress 功能擴充的心臟。
        *   **設定：** 網站的各項基本設定，例如網站標題、時間格式、固定網址等。
    *   **Moksa 技巧：** 這個選單會隨著你安裝的外掛而「長大」。例如，當我們安裝 **WooCommerce** 後，你就會看到「WooCommerce」和「商品」等專屬選單；安裝 **Rank Math SEO** 後，就會出現 SEO 設定選單。熟悉這個區域是管理網站的基礎。

3.  **中央主要工作區 (Main Central Workspace):**
    *   **位置：** 整個畫面最主要的中央區域。
    *   **功能：** 你實際進行操作和查看內容的地方。當你點擊左側主選單的任何項目，對應的操作介面就會顯示在這裡。
    *   **儀表板首頁：** 在預設的儀表板首頁，這個區域會顯示各種「**資訊看板 (Dashboard Widgets)**」，提供網站的快速資訊摘要。

---

### (實作) 打造你的個人化儀表板：善用「顯示選項」

預設的儀表板資訊看板很多，但並非每一個都對你有用。有些看板（例如 WordPress 新聞）甚至會干擾你的注意力。現在，我們要學習如何整理這個工作區，讓它變得更有效率。

**步驟 1：找出並點擊「顯示選項」**

請將你的目光移到畫面的**右上角**，你會看到一個標示著「**顯示選項 (Screen Options)**」的按鈕。點擊它，一個設定面板會從上方滑出。

``

**步驟 2：勾選／取消勾選資訊看板**

在滑出的面板中，你會看到一系列的勾選框 (Checkbox)，每一個都對應著中央工作區的一個「資訊看板」。

*   **試試看：** 取消勾選「**歡迎**」和「**WordPress 活動與新聞**」。
*   **觀察：** 你會發現中央工作區對應的看板立刻消失了！

就是這麼簡單！你可以根據你的需求，自由決定要顯示哪些資訊。

``

**步驟 3：拖曳調整位置與順序**

除了隱藏看板，你還可以調整它們的顯示順序。

*   **操作：** 將滑鼠移動到任何一個資訊看板的標題列上。
*   **拖曳：** 按住滑鼠左鍵，你就可以自由地拖曳該看板到你想要的位置，例如將「**網站健康狀態**」拖曳到最上方。

``

**Moksa 技巧：**

作為一個電商網站的經營者，**效率就是一切**。我們建議你初期可以這樣設定你的儀表板：

*   **保留這些：**
    *   **網站健康狀態：** 快速了解網站是否有技術問題。
    *   **近況：** 讓你一眼看到目前有幾篇文章、頁面、商品。
    *   未來安裝 **WooCommerce** 後，會有一個「**WooCommerce 狀態**」的看板，顯示你的銷售額、訂單等核心數據，**請務必將它拖到最顯眼的位置**。
    *   未來安裝 **FluentCRM** 後，也會有專屬的看板，讓你追蹤 Email 行銷的成效。
*   **可以隱藏的：**
    *   **歡迎：** 這個看板在你熟悉後台後就不再需要。
    *   **快速草稿：** 我們習慣直接到「文章 > 新增文章」來撰寫，功能更完整。
    *   **WordPress 活動與新聞：** 這對日常營運幫助不大，可以隱藏以保持專注。

這個「**顯示選項**」的功能不僅存在於儀表板，它也會出現在**文章列表、商品列表、訂單列表**等許多後台頁面，讓你自訂要顯示哪些欄位。這是一個你必須熟練的強大技巧！
