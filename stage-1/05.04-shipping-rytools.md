# 5.4 (實作) 串接「RY Tools (Pro) for WooCommerce」物流外掛

你好，我是 Moksa 的首席課程導師。

在前面的章節，我們已經完成了 WooCommerce 的基本設定。然而，一個在地化的電商網站，最重要的就是提供符合消費者習慣的物流選項。在台灣，**超商取貨**（包含取貨付款）幾乎是所有電商網站的標準配備。

傳統上，要串接各大超商（7-11, 全家）的物流 API，流程非常繁瑣，需要申請、設定、測試，對新手來說門檻極高。

因此，Moksa 團隊經過多年的實戰與測試，強力推薦使用 **「RY Tools (Pro) for WooCommerce」** 這款由台灣開發者開發的專業物流外掛。它將所有複雜的串接流程打包好，讓我們只需要透過幾個簡單的點擊，就能快速為網站加上「7-11 超商取貨」、「全家超商取貨」、「宅配」等多種運送方式，完美符合台灣市場的需求。

這堂課，我們將一步步帶你完成 RY Tools 的安裝、啟用與核心設定。

---

### 課前準備

在開始前，請確保你已經完成以下準備：

1.  **已安裝 WooCommerce**：你的網站需要有 WooCommerce 核心外掛。
2.  **取得 RY Tools (Pro) 外掛**：請至 [RY Tools 官方網站](https://ry-plugin.com/ry-woocommerce-tools) 購買並下載 Pro 版本的外掛 `.zip` 檔。
3.  **取得授權金鑰 (License Key)**：購買後，你也會在官網的會員帳號中找到一組授權金鑰，等等會用到。

---

### 步驟 1：安裝與啟用 RY Tools 外掛

首先，我們要將下載好的外掛安裝到 WordPress 網站中。

1.  登入你的 WordPress 後台。
2.  在左側選單，找到「**外掛**」>「**安裝外掛**」。
3.  點擊頁面上方的「**上傳外掛**」按鈕。
4.  點擊「**選擇檔案**」，並選取你剛剛下載的 `ry-woocommerce-tools-pro.zip` 檔案。
5.  點擊「**立即安裝**」。
6.  安裝完成後，點擊「**啟用外掛**」。

``

### 步驟 2：啟用專業版授權金鑰

啟用外掛後，下一步是輸入你的授權金鑰，這樣才能解鎖所有 Pro 版的強大功能並接收後續的更新。

1.  在左側選單，找到「**WooCommerce**」>「**設定**」。
2.  切換到「**RY 工具**」分頁。
3.  在「**RY 工具**」的子分頁中，點選「**一般設定**」。
4.  找到「**專業版授權金鑰**」的欄位，將你從官網取得的金鑰貼上。
5.  點擊「**儲存設定**」按鈕。系統會自動驗證你的金鑰，成功後通常會顯示綠色的啟用訊息。

``

### 步驟 3：設定運送區域與新增運送方式

RY Tools 的運送方式必須被加入到 WooCommerce 的「運送區域 (Shipping Zone)」中才會生效。

1.  在左側選單，找到「**WooCommerce**」>「**設定**」。
2.  切換到「**運送方式**」分頁。
3.  如果你還沒有設定任何運送區域，請點擊「**新增運送區域**」。如果已經有，請點擊現有的區域（例如：「台灣」）下方的「**編輯**」。

    *   **區域名稱**：輸入一個好辨識的名稱，例如 `台灣`。
    *   **區域範圍**：在下拉選單中選擇 `台灣`。
    *   完成後點擊「**儲存設定**」。

4.  在運送區域的編輯畫面中，找到「**運送方式**」的區塊，點擊「**新增運送方式**」按鈕。

``

5.  此時會跳出一個彈出視窗，在下拉選單中，你會看到許多由 RY Tools 新增的選項，例如：
    *   **7-11 C2C 取貨付款**
    *   **7-11 C2C 純取貨**
    *   **全家 C2C 取貨付款**
    *   **全家 C2C 純取貨**
    *   **宅配 (RY Tools)**

    > **Moksa 技巧**：
    > *   **C2C vs B2C**：C2C (Customer-to-Customer) 是指個人賣家寄件，適合初期或個人賣家使用，申請快速。B2C (Business-to-Customer) 則是企業戶寄件，通常需要與超商物流簽約，適合有一定規模的商家。**初學者建議先使用 C2C 即可**。
    > *   **取貨付款 vs 純取貨**：這兩者的選擇，取決於你的金流設定。如果你有啟用「貨到付款」金流，就一定要提供「取貨付款」的物流選項。

6.  我們先選擇「**7-11 C2C 取貨付款**」，然後點擊「**新增運送方式**」。
7.  重複步驟 5 和 6，再新增「**宅配 (RY Tools)**」。

### 步驟 4：設定個別運送方式的費用

新增運送方式後，我們需要為它們設定費用。

1.  將滑鼠移到你剛剛新增的「7-11 C2C 取貨付款」上，點擊「**編輯**」。
2.  在設定畫面中，你可以調整幾個重要欄位：
    *   **方法標題**：這是顯示給顧客看的名稱，你可以維持預設，或修改成更親切的文字，例如 `7-11 超商取貨付款`。
    *   **運費**：直接輸入一個數字，例如 `60`。這就是這項運送方式的固定運費。
    *   **免運門檻**：這是 RY Tools Pro 的超實用功能！你可以直接在這裡設定滿額免運。例如，輸入 `1000`，代表訂單金額滿 1000 元時，此運送方式將自動變為免運。
    *   **金流方式**：你可以指定只有哪些金流方式可以使用這個物流。對於「取貨付款」，我們通常會勾選「**貨到付款 (ECPay)**」。

``

3.  設定完成後，點擊「**儲存設定**」。
4.  用同樣的方式，點擊「宅配 (RY Tools)」的「**編輯**」，並為它設定費用（例如 `100`）與免運門檻。

### 步驟 5：檢查前台結帳頁面

恭喜你！核心設定已經完成。現在，讓我們實際到前台看看效果。

1.  將任何一個商品加入購物車。
2.  前往「**結帳**」頁面。
3.  在「**運送方式**」的區塊，你現在應該能看到剛剛設定的「7-11 超商取貨付款」和「宅配」選項了。
4.  當你點選「7-11 超商取貨付款」時，下方會出現一個「**選擇取貨門市**」的按鈕。
5.  點擊按鈕，RY Tools 會載入一個互動地圖，讓顧客可以輕鬆搜尋並選擇他們方便取貨的 7-11 門市。
6.  選擇門市後，相關資訊會自動帶入結帳欄位。

``

這就是 RY Tools 強大的地方，它無縫整合了 WooCommerce 結帳流程與超商的電子地圖，大幅提升了使用者的購物體驗。

### 總結

在本章節中，我們成功地串接了台灣電商最重要的物流外掛 - RY Tools (Pro)。我們完成了：

*   **安裝與啟用外掛**，並輸入了專業版授權金鑰。
*   在 WooCommerce 的運送區域中，**新增了 7-11 與宅配的運送方式**。
*   為每個運送方式**設定了運費與免運門檻**。
*   驗證了前台結帳頁面，確認**電子地圖選擇門市**的功能正常運作。

現在，你的網站已經具備了專業、在地化的物流選項。在後續的「訂單管理」章節，我們將會教你如何透過 RY Tools，在接到訂單後，一鍵產生托運單並完成出貨流程。
