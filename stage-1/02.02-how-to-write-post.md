# 2.2 (實作) 如何發表一篇圖文並茂的「最新消息」

嗨，歡迎來到這個實作單元！在經營一個網站時，「最新消息」或「部落格」是你與顧客溝通、建立品牌形象、以及提升 SEO 自然搜尋流量最重要的管道。

今天，我們將一步步帶你走過從無到有，發表一篇專業、內容豐富、且對搜尋引擎友善的文章。我們將會運用 WordPress 內建的「**區塊編輯器 (Gutenberg)**」，並結合 Moksa 技術棧中的 **Kadence Blocks** 與 **Rank Math SEO** 外掛，讓你掌握現代化內容創作的流程。

---

### 步驟 1：進入文章編輯畫面

首先，我們需要新增一篇文章。

1.  登入你的 WordPress 後台。
2.  在左側的主選單中，找到「**文章**」。
3.  點擊「**文章**」>「**新增文章**」。

[截圖] - 顯示 WordPress 後台，滑鼠指標停在「文章」>「新增文章」的選單上。

這樣，你就會進入 WordPress 的核心 - **區塊編輯器**介面。這是一個非常直覺的編輯環境，所有內容，無論是文字、圖片、標題、按鈕，都是一個個獨立的「區塊」。

### 步驟 2：撰寫標題與內文

編輯器的第一個欄位，就是文章的靈魂。

1.  在最上方的「**新增標題**」欄位中，輸入你這篇文章的標題。例如：「Moksa 歡慶週年慶！全館商品限時 88 折優惠開跑」。
    *   **重要觀念**：這個標題不僅要吸引讀者，也會成為網頁的 H1 標籤和網址的一部分，對 SEO 至關重要。
2.  在標題下方，你會看到「**由此開始撰寫或輸入 / 來選擇區塊**」的提示。直接開始打字，WordPress 會自動為你建立第一個「**段落區塊**」。
3.  你可以自由地撰寫你的文章內文，每一段都會是一個獨立的段落區塊。

[截圖] - 顯示區塊編輯器介面，已填寫文章標題和一小段內文。

**Moksa 技巧**：在編輯器中，你可以直接輸入「`/`」符號，接著輸入區塊的名稱（例如 `/圖片` 或 `/標題`），就能快速新增區塊，大幅提升寫作效率！

### 步驟 3：善用「區塊」打造豐富版面

一篇「圖文並茂」的文章，不能只有文字。接下來，我們將學習如何插入最重要的幾種區塊。

#### 3.1 新增「標題」區塊 (H2, H3)

為了讓文章結構清晰、易於閱讀，我們需要使用「**標題區塊**」來建立小標題。

1.  在你想新增小標題的地方，按下 `Enter` 鍵新增一個區塊。
2.  點擊右邊的「`+`」號，或直接輸入 `/標題`。
3.  選擇「**標題**」區塊。
4.  預設是 H2，你可以點擊工具列來切換 H2、H3、H4 等不同層級。
    *   **重要觀念**：請依據文章的邏輯結構來使用標題，而不是為了外觀。H1 只能有一次 (就是你的文章主標題)，內文則由 H2、H3 依序構成。這對 SEO 和使用者體驗都非常好。

[截圖] - 顯示一個「標題區塊」，並突顯工具列上切換 H2/H3 的選項。

#### 3.2 插入「圖片」區塊

圖片是吸引讀者目光的關鍵。

1.  在你想插入圖片的地方，新增一個區塊。
2.  選擇「**圖片**」區塊。
3.  你可以選擇「**上傳**」新圖片，或從「**媒體庫**」中挑選已經上傳過的圖片。

[截圖] - 顯示一個「圖片區塊」，並有「上傳」和「媒體庫」的選項。

**Moksa 技巧**：在你上傳圖片的同時，我們安裝的 **ShortPixel** 外掛會在背景自動幫你壓縮圖片，確保網站速度。你只需要專注在上傳好圖片即可。

**上傳圖片後的 SEO 設定：**
當你選好圖片後，請務必點擊圖片，並在右側的設定欄中找到「**替代文字 (Alt Text)**」。

*   **替代文字 (Alt Text)**：這是 **SEO 的關鍵**，請務務必用簡短的文字描述這張圖片的內容，最好能包含你的文章關鍵字。例如：「Moksa 團隊週年慶祝活動合照」。這不僅能幫助 Google 理解圖片內容，也能在圖片載入失敗時顯示文字，並幫助視障人士理解網頁。

[截圖] - 顯示圖片區塊被選取，右側設定欄的「替代文字 (Alt Text)」欄位被高亮。

#### 3.3 使用 Kadence Blocks 增加互動按鈕

有時候，你希望引導讀者前往某個活動頁面或商品頁。這時，「**按鈕區塊**」就非常實用。

1.  新增一個區塊，搜尋並選擇「**Kadence / Button**」。
2.  點擊按鈕區塊，直接修改按鈕上的文字，例如：「立即前往活動頁」。
3.  在右側的設定欄中，你可以詳細設定按鈕的**連結 URL**、顏色、樣式、大小等。

### 步驟 4：設定文章分類、標籤與精選圖片

在文章內容完成後，我們還需要進行一些重要的後台設定。這些設定通常位於編輯器右側的「**文章**」分頁中。

1.  **分類 (Categories)**：為文章選擇一個主要的分類，例如「最新優惠」或「品牌故事」。分類有助於網站結構化，方便使用者尋找同類型文章。
2.  **標籤 (Tags)**：為文章加上幾個相關的關鍵字標籤，例如「週年慶」、「折扣」、「免運費」。標籤可以更細緻地描述文章內容。
3.  **精選圖片 (Featured Image)**：這是文章的「封面圖」。它會顯示在部落格列表頁、社群分享時的預覽圖等。點擊「**設定精選圖片**」並上傳或選擇一張最具代表性的圖片。**這是一個必做的步驟！**

[截圖] - 顯示編輯器右側的文章設定欄，高亮「分類」、「標籤」和「精選圖片」這三個區塊。

### 步驟 5：SEO 優化 (使用 Rank Math)

內容寫好了，最後一步就是讓它更容易被 Google 找到。

1.  將編輯器畫面往下捲動，你會看到 **Rank Math SEO** 的設定區塊。
2.  **焦點關鍵字 (Focus Keyword)**：在欄位中輸入你希望這篇文章在 Google 被搜尋到的**主要關鍵字**，例如「週年慶優惠」。
3.  **編輯摘要 (Edit Snippet)**：點擊這個按鈕，你可以自訂文章顯示在 Google 搜尋結果頁的**標題 (Title)**、**永久連結 (Permalink)** 和**描述 (Description)**。Rank Math 會提供即時預覽。盡量將你的焦點關鍵字自然地融入其中。

[截圖] - 顯示 Rank Math SEO 設定區塊，高亮「焦點關鍵字」和「編輯摘要」按鈕。

**重要觀念**：Rank Math 的分數是一個「參考指標」，幫助你檢查是否有遺漏重要的 SEO 項目。不需要執著於追求 100 分，重點是內容對你的讀者有價值。

### 步驟 6：預覽與發表

恭喜你，一切都準備就緒了！

1.  點擊右上角的「**預覽**」按鈕，選擇「**在新的分頁中預覽**」。請務必檢查文章在電腦和手機上的顯示效果是否都正常。
2.  確認一切無誤後，回到編輯器。
3.  點擊右上角藍色的「**發佈**」按鈕，再次確認後，你的文章就成功上線了！

---

### 總結

今天，你已經學會了在 WordPress 中發表一篇專業文章的完整流程。我們回顧一下重點：

*   使用「**區塊編輯器**」來組織內容結構。
*   善用「**標題區塊 (H2/H3)**」劃分段落，提升可讀性與 SEO。
*   上傳圖片時，務必填寫「**替代文字 (Alt Text)**」。
*   透過 **Kadence Blocks** 的「**按鈕**」等區塊增加互動性。
*   設定好文章的「**分類**」、「**標籤**」與「**精選圖片**」。
*   利用 **Rank Math** 設定「**焦點關鍵字**」與「**搜尋結果摘要**」。

熟練這個流程後，你就能持續為你的網站產出高品質的內容。下一堂課，我們將會探討更多進階的區塊應用！
