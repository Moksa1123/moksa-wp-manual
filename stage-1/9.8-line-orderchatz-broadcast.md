# 9.8 OrderChatz 主動推播功能介紹

嗨，歡迎來到這個章節！在之前的課程中，我們已經學會如何透過 OrderChatz 與顧客進行一對一的 LINE 客服溝通。然而，OrderChatz 最強大的功能之一，是它能夠讓你**主動出擊**，向你的 LINE 官方帳號好友們進行「**主動推播**」(Broadcast)。

這項功能徹底改變了傳統電商的行銷模式。相較於逐漸被忽略的 Email EDM，LINE 推播擁有**極高的開啟率**與**即時性**，是你與顧客建立深度連結、提升回購率的超級武器。

在這個單元，我們將深入探討 OrderChatz 的主動推播功能，從介面介紹、建立第一則推播，到各種實戰行銷情境的應用。

---

### 什麼是「主動推播」？

簡單來說，「主動推播」就是由**商家主動發起**，向**多位**或**全部** LINE 好友發送訊息的過程。

這與一對一的客服對話不同，客服是被動地等待顧客提問後才回覆。而推播則是主動的行銷或通知行為，例如：

*   發送本週新品上市通知。
*   宣告雙 11 全館折扣活動。
*   發送會員專屬的優惠券。
*   提醒顧客上次買的商品快用完了。

**Moksa 技巧：** 請務必區分 **OrderChatz (行銷/客服)** 與 **Order Notify (訂單通知)** 的不同。Order Notify 專注於「交易型」通知，例如訂單成立、已付款、已出貨，是自動化且一對一的。而 OrderChatz 的推播功能則專注於「行銷型」或「關懷型」訊息，是需要手動規劃與發送的群發訊息。

---

### (實作) 建立你的第一則主動推播 (SOP)

現在，讓我們實際操作一次，從頭到尾建立一則完整的推播訊息。

#### 步驟 1：進入主動推播後台

1.  從 WordPress 後台，找到左側選單的 `OrderChatz`。
2.  點擊下方的子選單「**主動推播**」。
3.  你會看到一個列表，顯示所有過去建立的推播紀錄，包含狀態、發送數量、開啟率等。
4.  點擊畫面右上角的「**新增推播**」按鈕。

``

#### 步驟 2：設定推播基本資訊

進入新增推播頁面後，你會看到幾個基本欄位需要填寫。

1.  **推播名稱：** 這是**內部管理用**的名稱，顧客不會看到。**Moksa 技巧：** 建議使用有系統的命名方式，方便日後追蹤成效，例如 `[2023-11-11] 雙11全館8折優惠券`。
2.  **推播時間：** 你可以選擇「**立即發送**」或是「**排程發送**」。排程功能非常實用，你可以預先規劃好一整個月的行銷活動，讓系統在指定時間自動發送。

``

#### 步驟 3：設定推播對象 (最重要的步驟！)

這是主動推播最核心、也最能體現價值的部份。亂槍打鳥式的推播只會造成顧客反感與封鎖，**精準的分眾**才是提升轉換率的關鍵。

OrderChatz 提供了強大的篩選器：

*   **全部好友：** 將訊息發送給所有已綁定 LINE 的顧客。除非是全站重大公告，否則不建議常用。
*   **根據條件篩選：** 這是我們的秘密武器！你可以設定一或多個條件，組合出你想要的目標客群。

以下是一些常見的**Moksa 實戰篩選情境**：

*   **鐵粉回饋：**
    *   `總消費金額` `大於` `10000`
    *   發送 VIP 專屬的感謝禮或高額折價券。
*   **喚醒沉睡顧客：**
    *   `最後下單時間` `在...之前` `90天前`
    *   發送「好久不見」的關懷訊息，並附上一個吸引人的回購優惠。
*   **新品交叉銷售：**
    *   `曾經購買過商品` `包含` `[A商品]`
    *   `曾經購買過商品` `不包含` `[A商品的配件B]`
    *   向買過 A 商品的顧客推銷最適合搭配的 B 配件。
*   **會員等級行銷 (需搭配 YITH Membership)：**
    *   你可以針對「黃金會員」、「白金會員」等不同等級的顧客，發送專屬的福利或活動通知。

``

**Moksa 技巧：** 篩選條件可以**多重組合**，例如「過去一年內消費滿 5000 元，且購買過特定分類商品的顧客」。請善用這個功能，讓你的每一則訊息都送到最對的人手上。

#### 步驟 4：編輯推播訊息內容

設定好目標對象後，就來編輯要發送的訊息吧！OrderChatz 支援 LINE 官方提供的多種訊息格式。

1.  **文字訊息：** 最單純的格式。你可以使用變數 `{{customer_name}}` 來自動代入顧客的名字，增加親切感。
2.  **圖片訊息：** 一張吸睛的圖片勝過千言萬語。你可以上傳設計好的活動 Banner。
3.  **彈性訊息 (Flex Message)：** **這是最強大、最推薦的格式！** Flex Message 讓你可以在一個訊息框內組合圖片、文字、按鈕，創造出類似卡片的豐富版面。你可以用它來製作：
    *   商品輪播卡 (Carousel)。
    *   圖文選單。
    *   帶有「**立即購買**」、「**領取優惠券**」按鈕的活動訊息。

``

**Moksa 技巧：** 你的推播內容一定要有一個明確的「**行動呼籲 (Call to Action, CTA)**」。不要只是告知，要引導顧客**下一步該做什麼**。例如，你可以結合 **Advanced Coupons** 的 **URL Coupon** 功能，產生一個「點擊連結即可自動套用 9 折」的網址，並將它放在 Flex Message 的按鈕中，大幅簡化顧客使用優惠的流程。

#### 步驟 5：測試與發送

在正式發送給數百甚至數千位顧客前，**務必進行測試**！

1.  在訊息編輯區的下方，有一個「**傳送測試訊息**」的欄位。
2.  輸入你自己的 LINE User ID (可以在 `OrderChatz > 好友列表` 中找到自己的帳號)。
3.  點擊發送，然後到你的手機上檢查訊息的排版、圖片、連結是否都正確無誤。
4.  確認一切完美後，回到頁面頂部，點擊「**儲存**」(若是排程) 或「**立即發送**」。

### 主動推播的行銷應用情境

學會操作後，更重要的是如何應用。以下是 Moksa 團隊推薦的幾種行銷情境：

1.  **新品上市/預購通知：** 篩選出曾經買過同類別商品的顧客，發送第一手新品消息。
2.  **節慶主題活動：** 如母親節、聖誕節，設計精美的圖文訊息，附上活動頁面連結與專屬優惠。
3.  **會員專屬日：** 每個月固定一天，篩選出特定等級的會員 (`YITH Membership`)，發送他們獨享的優惠或搶先購資格。
4.  **點數加倍活動 (需搭配 YITH Points and Rewards)：** 舉辦限時點數加倍送的活動，刺激顧客消費以累積更多點數。
5.  **內容行銷：** 不只是賣東西！你也可以分享實用的部落格文章、教學影片，與顧客建立信任感。

### 最後的提醒：Moksa 最佳實踐

*   **重質不重量：** 不要疲勞轟炸你的顧客。過於頻繁的推播只會換來封鎖。建議行銷類訊息一週最多 1-2 次。
*   **遵守 LINE 官方規範：** 避免發送垃圾訊息或違規內容，否則你的官方帳號可能面臨被停權的風險。
*   **提供真正的價值：** 在按下發送前，問問自己：「這則訊息對顧客有幫助嗎？」、「這個優惠夠吸引人嗎？」。永遠以顧客為中心。
*   **分析成效：** 定期回到推播列表，檢視每一則訊息的開啟率與點擊率，你將會慢慢找出你的顧客最喜歡什麼樣的內容。
