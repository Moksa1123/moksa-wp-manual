# 3.2 (實作) 運送方式設定（費率、免運、自取、運送區域）

哈囉，歡迎來到這個章節！物流設定是電商網站的靈魂之一，它直接影響了顧客的購物體驗與你的營運成本。一個清晰、有彈性的運費規則，可以有效提升轉換率，反之，則可能讓顧客在結帳前最後一刻放棄訂單。

在這個單元，我們將會從零開始，帶你完整設定 WooCommerce 的運送方式。我們會涵蓋最重要的「運送區域」觀念，並實作台灣電商最常見的幾種運費情境：固定費率、免運門檻、到店自取，以及透過 **Moksa 技術棧** 核心外掛 **RY Tools** 來整合的 **7-11/全家超商取貨**。

---

## 觀念解析：運送區域 (Shipping Zone)

在開始設定前，你必須先理解一個 WooCommerce 最重要的物流核心觀念：「**運送區域 (Shipping Zone)**」。

*   **什麼是運送區域？**
    *   它不是指「行政區」或「郵遞區號」，而是一個你**自訂的地理範圍**。
    *   你可以為這個「範圍」內的顧客，提供專屬的「**運送方式 (Shipping Method)**」與費率。

*   **舉例來說：**
    *   你可以建立一個名為「台灣本島」的運送區域，並在這個區域內提供「宅配到府 (費率 $100)」、「超商取貨 (費率 $65)」、「滿千免運」等多種運送方式。
    *   你可以再建立一個名為「澎湖、金門、馬祖」的運送區域，並只提供「郵局寄送 (費率 $150)」這個選項。
    *   當顧客在結帳頁面輸入地址時，WooCommerce 會自動判斷他屬於哪個「運送區域」，並只顯示該區域可用的「運送方式」。

理解這個觀念後，我們就可以開始動手設定了！

---

## 步驟一：建立你的第一個「運送區域」

我們來建立一個最常用的「台灣」運送區域。

1.  登入 WordPress 後台，從左側選單前往「**WooCommerce**」>「**設定**」。
2.  切換到「**運送方式**」分頁。
3.  點擊「**新增運送區域**」按鈕。

    ``

4.  在設定頁面中，依序填寫以下欄位：
    *   **區域名稱**：輸入一個你好辨識的名稱，例如「**台灣**」或「**台灣本島**」。這是給管理者看的，顧客不會看到。
    *   **區域**：在搜尋框中輸入「**台灣**」，並從下拉選單中選取。你可以進一步限定省市，但通常直接選取整個「台灣」即可。

    ``

5.  完成後，先別急著離開，直接點擊右下角的「**儲存設定**」，我們接著要為這個區域新增運送方式。

---

## 步驟二：在「台灣」區域內新增運送方式

成功建立「台灣」這個運送區域後，畫面會刷新，你會看到下方出現了「運送方式」的區塊。這裡就是我們要新增各種運費規則的地方。

``

點擊「**新增運送方式**」按鈕，你會看到一個彈出視窗，裡面包含了幾種預設的選項以及由 **RY Tools** 新增的超商選項。

``

接下來，我們會逐一實作最常見的幾種設定。

### 情境 A：設定「宅配」的固定費率 (統一費率)

這是最基本的設定，無論顧客買多少東西，運費都固定。

1.  在「台灣」運送區域的編輯畫面，點擊「**新增運送方式**」。
2.  從下拉選單中選擇「**統一費率**」，並點擊「**新增運送方式**」。
3.  你會看到清單上多了一行「統一費率」。將滑鼠移到上面，點擊「**編輯**」連結。
4.  在彈出的視窗中，進行以下設定：
    *   **方法標題**：這是顯示給顧客看的名稱，可以改成更友善的文字，例如「**黑貓宅配**」或「**新竹物流**」。
    *   **稅金狀態**：通常選擇「**應稅**」。
    *   **費用**：**最重要的欄位！** 請直接輸入運費金額，例如 `100` 或 `120`。**不需要**加上 `$` 或 `NT$` 符號。
5.  點擊「**儲存設定**」。

``

### 情境 B：設定「滿額免運」的規則 (免運)

免運是刺激顧客提高客單價最有效的行銷工具。

1.  同樣在「台灣」運送區域的編輯畫面，點擊「**新增運送方式**」。
2.  這次從下拉選單中選擇「**免運**」，並點擊「**新增運送方式**」。
3.  將滑鼠移到新增的「免運」上，點擊「**編輯**」。
4.  在彈出的視窗中，進行設定：
    *   **標題**：可以維持預設的「免運」。
    *   **免運要求**：這是**關鍵欄位**。從下拉選單中選擇「**需要最低訂單金額**」。
    *   **最低訂單金額**：在下方出現的欄位中，輸入你的免運門檻，例如 `1000` 或 `1500`。
5.  點擊「**儲存設定**」。

``

**Moksa 技巧：** 當顧客的購物車金額同時滿足「統一費率」和「免運」的條件時（例如消費滿 $1000），WooCommerce **預設只會顯示「免運」的選項**，自動隱藏需要付費的選項，非常智慧！

### 情境 C：設定「到店自取」的選項 (Local Pickup)

如果你的品牌有實體門市或工作室，提供自取服務可以省下運費與包材成本。

1.  在「台灣」運送區域的編輯畫面，點擊「**新增運送方式**」。
2.  從下拉選單中選擇「**到店自取**」，並點擊「**新增運送方式**」。
3.  點擊「**編輯**」來修改設定：
    *   **標題**：可以改成更清楚的說明，例如「**工作室自取 (請預約時間)**」。
    *   **費用**：通常自取是免費的，所以保留 `0` 即可。
4.  點擊「**儲存設定**」。

### 情境 D：設定「超商取貨」(7-11/全家)

這是台灣電商的標配！我們將使用 **RY Tools** 外掛來實現這個功能。

**前情提要：** 你必須已經安裝並啟用 **RY Tools - 7-11 and FamilyMart Logistics for WooCommerce** 外掛。相關的 API Key 設定會在後續的「金物流串接」章節有更詳細的教學，這裡我們先學習如何將它加入運送選項。

1.  在「台灣」運送區域的編輯畫面，點擊「**新增運送方式**」。
2.  你會在下拉選單中看到由 RY Tools 新增的選項，我們先選擇「**7-11 C2C**」，並點擊「**新增運送方式**」。
3.  重複上一個步驟，再次點擊「新增運送方式」，這次選擇「**FamilyMart C2C**」。
4.  現在你的運送方式清單應該會同時有 7-11 和全家了。分別點擊它們的「**編輯**」連結，進行設定：
    *   **Title (標題)**：可以改成顧客容易懂的名稱，例如「**7-11 超商取貨**」和「**全家超商取貨**」。
    *   **Cost (費用)**：輸入超商的運費，通常是 `60` 或 `65`。
5.  分別儲存兩個超商的設定。

``

完成以上所有設定後，你的「台灣」運送區域看起來應該會像這樣，包含了所有主流的運送選項：

## 步驟三：設定全域「運送選項」

最後，在「運送方式」分頁頂部，還有一個「**運送選項**」的子分頁，這裡有一些全域設定需要我們確認。

``

*   **計算方式：**
    *   **在購物車頁面啟用運費計算機**：**建議勾選**。這會讓顧客可以在購物車頁面預先試算運費，體驗較好。
    *   **在輸入地址前，隱藏運費**：**建議勾選**。在顧客還沒填寫地址時，先不要顯示任何運費資訊，避免造成混淆。
*   **運送目的地：**
    *   **預設為顧客帳單地址**：這是最常見的設定，維持預設即可。

設定完成後，記得點擊「**儲存設定**」。

恭喜你！你已經完成了 WooCommerce 最核心、也最重要的物流設定。

讓我們快速回顧一下本章節的重點：

1.  **先建立「運送區域」**：這是所有運費規則的基礎，我們先定義了一個「台灣」的範圍。
2.  **再新增「運送方式」**：我們在「台灣」這個區域內，新增了四種主流的運送方式：
    *   **統一費率**：適用於固定運費的宅配。
    *   **免運**：設定了滿 `1000` 元免運的門檻。
    *   **到店自取**：提供給有實體店的商家。
    *   **超商取貨**：透過 **RY Tools** 整合了 7-11 與全家。
3.  **檢查「運送選項」**：確保購物車的運費計算體驗是流暢的。
