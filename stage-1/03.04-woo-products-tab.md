# 3.4 商品頁籤設定 (商店頁面、商品評論)

嗨，歡迎來到這個章節！我是你在 Moksa 的課程導師。

在前面的課程中，我們已經學會如何新增商品。現在，我們要來探討如何管理與顧客購物體驗息息相關的幾個重要頁面與功能：**商店主頁面**、**商品評論系統**，以及單一商品頁面下方的**資訊頁籤**。

這些設定看似微小，但它們直接影響了顧客如何在你的網站上找到商品、如何信任你的商品品質，以及他們能多深入地了解商品細節。一個清晰、友善且資訊豐富的購物介面，是提升轉換率的關鍵。

### 第一部分：商店頁面設定

首先，我們需要告訴 WooCommerce，哪一個頁面是我們網站的「商店主頁」。這個頁面會自動陳列你所有的商品。

**步驟 1：前往 WooCommerce 商品設定**

1.  從 WordPress 後台，前往左側選單的 `WooCommerce` > `設定`。
2.  點擊上方的 `商品` 頁籤。
3.  確認你現在位於 `商品` 頁籤下的 `一般` 子分頁。

`[截圖：顯示 WooCommerce > 設定 > 商品 > 一般 的後台畫面]`

**步驟 2：指定商店頁面**

1.  在 `一般` 設定中，你會看到一個名為「**商店頁面**」的選項。
2.  點擊下拉選單，選擇你想要作為商品陳列主頁的頁面。
3.  **Moksa 技巧**：通常在安裝 WooCommerce 時，系統會自動幫你建立一個名為「商店」的頁面。除非你有特殊規劃，否則直接選擇這個「商店」頁面即可。這個頁面不需要用 Elementor 或 Kadence Blocks 編輯，WooCommerce 會自動接管它並輸出商品列表。
4.  設定完成後，記得滑到頁面最下方，點擊「**儲存設定**」。

`[截圖：圈選出「商店頁面」的下拉選單，並顯示已選擇「商店」頁面]`

**步驟 3：調整商店頁面顯示方式**

1.  在同一個 `商品` 頁籤下，切換到 `商品顯示` 子分頁。
2.  在這裡你可以調整幾個關鍵設定：
    *   **商店頁面顯示**：你可以決定商店主頁要顯示「商品」、「分類」，還是「分類與商品」。對於大多數電商網站，**我們建議選擇「顯示商品」**。
    *   **預設商品排序**：設定顧客進入商店頁面時，商品的預設排序方式。常見的選項有「依熱門度 (銷量) 排序」、「依最新商品排序」或「依價格排序」。

`[截圖：顯示「商品顯示」子分頁的設定選項]`

### 第二部分：商品評論設定

商品評論是建立**社會認同 (Social Proof)** 的最重要工具。真實的顧客回饋能大幅提升潛在顧客的購買意願。讓我們來看看如何設定它。

**步驟 1：找到評論設定區塊**

1.  同樣在 `WooCommerce` > `設定` > `商品` > `一般` 這個頁面。
2.  將頁面向下捲動，你會找到一個標題為「**評論**」的區塊。

**步驟 2：啟用並管理評論功能**

`[截圖：顯示「評論」區塊的所有設定選項]`

以下是各個選項的詳細說明：

*   **啟用商品評論**：
    *   `啟用商品評論`：**務必勾選此項**，才能開啟整個評論功能。
    *   `在評論中顯示「已驗證的購買者」標籤`：**強烈建議勾選！** 這會在實際購買過該商品的顧客評論旁，顯示一個「已驗證的購買者」標籤，大幅增加評論的可信度。
    *   `評論僅限「已驗證的購買者」可 남기`：這是一個更嚴格的設定。勾選後，只有登入帳號並實際購買過此商品的顧客才能留下評論。
        *   **Moksa 技巧**：這個設定可以**有效杜絕惡意或虛假的垃圾評論**。初期商品不多時，建議勾選此項以維持評論品質。

*   **商品評分**：
    *   `在評論中啟用星級評分`：允許顧客在留下文字評論的同時，給予 1 到 5 顆星的評分。**這是必備功能，請務必勾選**。
    *   `星級評分必須填寫，不可選填`：建議勾選，確保每一則評論都有對應的星級，方便其他顧客快速判斷。

設定完成後，別忘了點擊「**儲存設定**」。

**Moksa 技巧：如何審核評論？**

當有顧客留下新評論時，預設情況下需要由管理員審核。你可以到 WordPress 後台的 `留言` 區塊，看到所有等待審核的商品評論，你可以在這裡進行「核准」、「回覆」、「編輯」或「移至垃圾桶」。

`[截圖：顯示 WordPress 後台「留言」區塊，並有一則待審核的商品評論]`

### 第三部分：(實作) 管理單一商品頁籤

當顧客點進單一商品頁面時，下方通常會有「說明」、「其他資訊」、「評論」這幾個頁籤。讓我們來學習如何管理它們的內容。

`[截圖：一個 WooCommerce 單一商品頁面，並圈選出「說明」、「其他資訊」、「評論」這三個頁籤]`

**頁籤 1：說明 (Description)**

這個頁籤的內容，就是你在編輯商品時，於**主要內容編輯器**中所輸入的圖文資訊。你可以使用 Kadence Blocks 或 Elementor 把它設計得非常豐富，包含詳細的規格、使用情境、品牌故事等等。

`[截圖：編輯商品頁面，並指向主要的內容編輯器區塊]`

**頁籤 2：其他資訊 (Additional Information)**

這個頁籤的內容是**自動產生**的，它主要顯示：

1.  **商品重量與尺寸**：如果你在 `商品資料` > `運送方式` 中有填寫。
2.  **商品屬性**：你在 `商品資料` > `屬性` 中建立的任何自訂屬性（例如：顏色、材質、產地）。

**實作：新增一個「材質」屬性到頁籤中**

1.  編輯一個商品，捲動到 `商品資料` 區塊。
2.  切換到 `屬性` 頁籤。
3.  在「自訂商品屬性」下拉選單，點擊 `新增`。
4.  在「**名稱**」欄位輸入 `材質`。
5.  在「**值**」欄位輸入 `100% 純棉` (多個值請用 `|` 分隔)。
6.  **最重要的一步**：務必勾選「**在商品頁面顯示**」這個選項。
7.  點擊 `儲存屬性`，然後更新商品。

`[截圖：顯示在商品資料 > 屬性中，新增「材質」屬性的完整操作畫面]`

現在回到你的商品前台頁面，重新整理後點擊「其他資訊」頁籤，你就會看到剛剛新增的「材質」資訊了！

**頁籤 3：評論 (Reviews)**

這個頁籤會顯示所有已通過審核的顧客評論，以及讓顧客提交新評論的表單。它的開關與運作方式，完全由我們在**第二部分**所做的設定來決定。

---

**Moksa 技巧 (進階)：如何用程式碼調整頁籤？**

有時候，你可能會想要移除某個頁籤（例如，如果你的商品都沒有尺寸重量，想移除「其他資訊」頁籤），或是調整它們的順序。這通常需要一小段程式碼。

**警告：以下操作建議有程式基礎的學員進行，修改前請務必使用 UpdraftPlus 備份！**

你可以將以下程式碼片段，貼到你的佈景主題 `functions.php` 檔案的最後面：

```php
/**
 * MoksaWP 範例：移除商品頁面的「其他資訊」頁籤
 */
add_filter( 'woocommerce_product_tabs', 'moksa_remove_additional_info_tab', 98 );
function moksa_remove_additional_info_tab( $tabs ) {
    unset($tabs['additional_information']); // 移除「其他資訊」頁籤
    return $tabs;
}

/**
 * MoksaWP 範例：調整商品頁籤的順序
 * 數字越小，排序越前面
 */
add_filter( 'woocommerce_product_tabs', 'moksa_reorder_product_tabs', 98 );
function moksa_reorder_product_tabs( $tabs ) {
    $tabs['reviews']['priority'] = 5;          // 評論頁籤
    $tabs['description']['priority'] = 10;     // 說明頁籤
    // $tabs['additional_information']['priority'] = 15; // 其他資訊頁籤 (如果沒有移除的話)
    return $tabs;
}
```
