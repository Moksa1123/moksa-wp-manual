# 01.01 課程介紹：n8n 與 WordPress 自動化協作的價值

你好，歡迎來到課程四！我是你在 Moksa 的課程導師。

在前面的課程中，我們已經學會如何使用 WordPress 與 WooCommerce 打造出功能強大的電子商務網站。你可能已經是一位熟練的站長，每天處理著訂單、管理客戶、更新商品。但你是否曾感覺，有許多日常工作是重複且耗時的？例如：

*   每天手動將新的訂單資料複製到 Google Sheets 報表中。
*   每當有新會員註冊時，都要手動寄出一封歡迎信，並把他加入到特定的行銷名單。
*   當客戶完成某個線上課程時，需要手動頒發證書並給予一個專屬優惠券。

如果這些場景讓你心有戚戚焉，那麼這堂課將徹底改變你的工作模式。我們將引入一個強大的「自動化」工具 — **n8n** (發音為 n-eight-n)，並學習如何讓它與我們的 WordPress 網站完美協作，將你從重複的瑣事中解放出來。

---

### 什麼是 n8n？為什麼我們選擇它？

想像一下，你的網站和所有你使用的網路服務（像是 Gmail, Google Sheets, Slack, LINE, FluentCRM...）都是一個個獨立的樂高積木。它們各自功能強大，但彼此之間無法直接對話。

**n8n** 就是那個神奇的「**數位黏著劑**」或「**樂高連接器**」。它是一個工作流自動化工具，可以讓你用**視覺化**的方式，將這些不同的應用程式和服務串連起來，建立自動化的流程（我們稱之為 **Workflow**）。

`[截圖：n8n 的視覺化工作流編輯器介面，展示了幾個節點（Node）被線條連接起來的畫面]`

你可能會問，市面上有名的 Zapier 或 Make (前身為 Integromat) 也能做到，為什麼 Moksa 的技術棧特別推薦 n8n？

1.  **高度彈性與掌控權**：n8n 是開源軟體，這代表我們可以將它**自行託管 (Self-Hosted)** 在自己的主機上。這不僅大幅降低了長期成本（許多付費方案是根據「執行次數」收費），更重要的是，**你的所有商業數據都掌握在自己手中**，無需流經第三方平台，保障了資料的隱私與安全。
2.  **強大的功能**：n8n 的免費版本功能就已經非常強大，支援複雜的邏輯判斷、分支、合併等，幾乎沒有任何限制。對於開發者來說，甚至可以自行撰寫節點來擴充功能。
3.  **視覺化流程**：與寫程式碼相比，n8n 的拖拉式介面讓非技術背景的站長也能輕易上手，直觀地設計出自己想要的自動化流程。

### n8n 與 WordPress 協作的價值：一些真實應用場景

這門長達 30 小時的課程，價值在哪裡？讓我們透過幾個基於 **Moksa 技術棧** 的實際案例，來感受自動化能為你的事業帶來多大的效益。

#### 場景一：電商營運全自動 (WooCommerce)

當一筆新的 WooCommerce 訂單成立時，n8n 可以自動為你完成以下所有事情：

*   **觸發條件**：WooCommerce 訂單狀態變為「處理中」。
*   **自動化流程**：
    1.  **財務同步**：自動在 **Google Sheets** 的財務報表中新增一筆訂單紀錄，包含訂單編號、金額、商品、客戶資訊。
    2.  **倉儲通知**：自動發送一則訊息到 **Slack** 或 **Discord** 的 `#出貨頻道`，通知倉儲人員「有新訂單，請準備出貨！」，並附上訂單摘要。
    3.  **CRM 標籤**：在 **FluentCRM** 中找到該位客戶，並為他貼上「已購買-商品A」的標籤，方便未來進行精準行銷。
    4.  **LINE 客戶通知**：如果客戶有綁定 LINE (透過 **OrderChatz** 或類似工具)，自動透過 LINE Notify 發送一則「您的訂單 #XXXXX 已收到，我們將盡快為您出貨」的客製化訊息。

`[截圖：一個展示 WooCommerce 訂單觸發多個動作（Google Sheets, Slack, FluentCRM）的 n8n 工作流示意圖]`

**Moksa 技巧**：這個流程將財務、倉儲、行銷、客服四個環節無縫串連，**將原本可能需要 10-15 分鐘的人工操作，縮短為 0 秒**，並且 100% 避免了人為的複製貼上錯誤。

#### 場景二：智慧化客戶關係與行銷 (FluentCRM / YITH)

自動化不僅僅是處理訂單，更是維繫客戶關係的利器。

*   **觸發條件**：WordPress 新會員註冊。
*   **自動化流程**：
    1.  **分眾標籤**：檢查這位會員的註冊來源（例如，是不是從某個特定的聯盟行銷連結來的 by **Solid Affiliate**），在 **FluentCRM** 中為他貼上不同的標籤 (e.g., `from-affiliate-A`, `organic-signup`)。
    2.  **個人化歡迎信**：根據不同的標籤，透過 Gmail 或你的 SMTP 服務，發送一封**高度個人化的歡迎信**，而不是千篇一律的罐頭訊息。
    3.  **點數獎勵**：透過 API 呼叫，自動在 **YITH Points and Rewards** 系統中，為這位新會員贈送 100 點註冊獎勵點數，並發信通知他。

#### 場景三：無縫的線上學習體驗 (LearnDash)

如果你經營的是線上課程網站，自動化可以讓學員的體驗提升到新的層次。

*   **觸發條件**：學員在 **LearnDash** 中完成了「WordPress 新手入門」整門課程。
*   **自動化流程**：
    1.  **自動頒發證書**：觸發 LearnDash 自動生成證書 (這部分 LearnDash 內建，但 n8n 可以接手後續)。
    2.  **寄送祝賀信**：發送一封客製化的祝賀 Email。
    3.  **發放專屬獎勵**：在 WooCommerce 中**自動生成一個 8 折的優惠券**，券碼可以是 `CONGRATS-[學員姓名]`，並將這個專屬優惠券碼包含在祝賀信中，鼓勵他購買下一門進階課程。
    4.  **社群升級**：如果你的社群是用 **BuddyBoss** 搭建的，可以自動將這位學員加入到「畢業生」專屬社團，讓他獲得更高的社群權限。

### 在這門課程中，你將學到什麼？

這門課是理論與實作並重。我們將從零開始，帶你一步步成為自動化高手。

*   **基礎篇**：n8n 的核心概念（Workflow, Node, Trigger, Credentials）、環境安裝（雲端版 vs. 自託管）與介面操作。
*   **連接篇**：學習如何使用 **Webhook** 和 **REST API**，安全地將你的 WordPress/WooCommerce 網站與 n8n 連接起來。
*   **實戰篇**：親手打造數十個實用的自動化工作流，涵蓋電商出貨、客戶服務、會員經營、課程管理、行銷活動等。
*   **整合篇**：串接我們熟悉的各種工具，如 Google Sheets, Slack, Discord, LINE, Gmail, FluentCRM 等。
*   **進階篇**：學習錯誤處理、流程優化、排程任務，以及**Moksa 獨家的實戰技巧**，確保你的自動化系統穩定可靠。

### 課前準備

在開始下一章節之前，請確保你具備：

1.  一個可以實際操作的 WordPress & WooCommerce 網站。
2.  對 WordPress 後台的基本操作有一定熟悉度。
3.  最重要的一點：**一顆願意探索、動手嘗試，並享受「讓聰明的工作取代辛苦的工作」的開放心態！**

自動化的世界充滿無限可能。它不僅僅是省下時間，更是讓你將寶貴的精力，專注在**策略、創意與品牌發展**等更具價值的事物上。
