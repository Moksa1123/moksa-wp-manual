# 06.02 (實作) 導入「LearnDash」或「LifterLMS」

嗨，歡迎來到這個章節！在之前的課程中，我們已經學會了如何使用 WooCommerce 銷售實體商品和虛擬商品。但如果你的商業模式是「知識變現」，例如線上課程、教學內容或專業認證，那麼一個專業的「學習管理系統 (LMS, Learning Management System)」外掛就不可或缺了。

LMS 就像是為你的 WordPress 網站裝上一個功能強大的線上學校，它可以幫你：
*   **結構化課程內容：** 將課程拆分為章節、單元、測驗。
*   **追蹤學習進度：** 了解學員上到哪裡、完成了哪些部分。
*   **頒發證書與徽章：** 給予學員完成課程的成就感。
*   **管理學員與講師：** 輕鬆管理學生名單與課程講師權限。
*   **內容保護：** 確保只有付費的學員才能看到課程內容。

在 WordPress 生態系中，`LearnDash` 和 `LifterLMS` 是最頂尖的兩個選擇。這堂課的目標，就是帶你了解兩者的差異，並以其中一個為範例，完成從安裝、建立課程到整合 WooCommerce 銷售的完整流程。

---

### 核心觀念：如何選擇適合你的 LMS？

在我們動手安裝前，最重要的第一步是「選擇」。LearnDash 和 LifterLMS 都非常強大，但它們的設計哲學、定價模式和生態系略有不同。

以下是 Moksa 團隊為你整理的比較表格，幫助你做出最適合的決策：

| 特性 | LearnDash | LifterLMS |
| :--- | :--- | :--- |
| **定價模式** | **付費訂閱制**。購買後包含所有核心功能，無免費版。 | **核心功能免費**。進階功能 (如進階測驗、作業系統、特定金流整合) 需額外購買 Add-on。 |
| **課程建立器** | 提供非常直觀的「拖放式 (Drag & Drop)」課程建立器，管理章節、單元非常方便。 | 同樣提供視覺化的課程建立器，體驗良好，容易上手。 |
| **內建功能** | 功能完整，包含進階測驗、作業上傳、課程群組、Focus Mode (專注模式) 等。 | 免費版核心功能已相當夠用，包含課程、會員、基礎測驗等。**內建基礎金流 (Stripe/PayPal) 和會員功能**。 |
| **整合性** | **生態系極為龐大**，與市面上絕大多數外掛 (如 BuddyBoss, MemberPress, Gamipress) 都有深度整合。 | 整合性也非常好，但第三方開發的 Add-on 數量略少於 LearnDash。 |
| **WooCommerce 整合** | 需安裝官方免費的 `LearnDash WooCommerce Integration` 外掛來串接。 | 需要**付費**購買官方的 `LifterLMS WooCommerce` Add-on 才能串接。 |

#### **Moksa 技巧：我們的選擇建議**

*   **如果你的預算充足，且希望一開始就擁有最完整、最強大的功能和生態系**，我們推薦你選擇 **LearnDash**。它的社群支持和第三方整合是業界標竿，特別是與 **BuddyBoss** 的社群整合，可以打造世界級的線上學習社群體驗。
*   **如果你是剛起步，預算有限，希望先從免費的核心功能開始測試市場**，那麼 **LifterLMS** 會是你的好朋友。它的免費版已經能滿足基本的開課需求，等未來需要擴充時，再根據需求逐一購買 Add-on 即可。

接下來的實作，我們將以 **LearnDash** 作為主要示範，因為它的流程最能體現「LMS + WooCommerce」這種強大的組合。同時，我們也會快速帶你瀏覽 LifterLMS 的獨特之處。

### 實作教學：以 LearnDash 為例

假設我們已經決定使用 LearnDash，現在就來動手把它整合進我們的 MoksaWP 網站吧！

#### 步驟 1：安裝與啟用 LearnDash

LearnDash 是付費外掛，你需要先到官網購買並下載外掛的 `.zip` 檔案。

1.  登入 WordPress 後台，前往「**外掛**」>「**安裝外掛**」。
2.  點擊上方的「**上傳外掛**」按鈕。
3.  選擇你從 LearnDash 官網下載的 `learndash.zip` 檔案，並點擊「**立即安裝**」。
4.  安裝完成後，點擊「**啟用外掛**」。
5.  啟用後，LearnDash 通常會跳出一個**設定精靈 (Setup Wizard)**。**Moksa 技巧：** 我們強烈建議你跟著設定精靈走一遍，它會引導你完成一些基礎設定，並自動幫你建立必要的頁面 (如：課程列表、學員個人資料頁)。

``

#### 步驟 2：建立你的第一門課程

啟用後，你會在左側選單看到一個新的「**LearnDash LMS**」項目。現在，我們來建立一門叫做「WordPress 基礎入門」的課程。

1.  前往「**LearnDash LMS**」>「**課程 (Courses)**」。
2.  點擊「**Add New**」新增課程。
3.  **標題**欄位輸入「WordPress 基礎入門」。
4.  下方的編輯區可以輸入這門課程的詳細介紹，你可以使用 Kadence Blocks 或 Elementor 來設計這個頁面。
5.  切換到頁籤「**Builder**」。這裡是 LearnDash 的核心！你可以用拖放的方式建立課程架構。
    *   點擊「**New Lesson**」新增一個章節，例如「第一章：認識 WordPress」。
    *   在章節下方，點擊「**New Topic**」新增單元，例如「1-1 什麼是 WordPress」。
    *   在章節下方，點擊「**New Quiz**」新增測驗，例如「第一章隨堂測驗」。

``

6.  切換到頁籤「**Settings**」。這裡有非常多關於課程的重要設定。
    *   **Access Mode (存取模式):** 這是最重要的設定！
        *   `Open`: 免費課程，任何人都能看。
        *   `Free`: 免費課程，但需要登入/註冊帳號才能看。
        *   `Buy Now`: **這就是我們要用的！** 設定一個價格，學員需透過 LearnDash 內建的金流購買。
        *   `Recurring`: 訂閱制課程，例如每月收費。
        *   `Closed`: **這是整合 WooCommerce 的關鍵！** 將課程的銷售交給外部系統 (也就是 WooCommerce)，學員無法直接從課程頁面購買。
    *   **Course Prerequisites (先修課程):** 設定學員必須先完成哪門課才能修這門課。
    *   **Course Certificate (課程證書):** 如果學員完成課程，可以頒發一張證書給他 (證書需要先另外設計)。

#### 步驟 3：設定課程售價 (整合 WooCommerce)

我們的目標是使用 WooCommerce 的金流系統 (綠界 ECPay、LINE Pay) 來銷售課程。所以，我們需要將 LearnDash 課程與一個 WooCommerce 商品綁定。

1.  首先，前往「**LearnDash LMS**」>「**Add-ons**」，找到並安裝啟用 `LearnDash WooCommerce Integration` 這個官方免費的整合外掛。
2.  前往「**商品**」>「**新增商品**」。
3.  建立一個新的商品，**商品名稱**可以和課程一樣，例如「WordPress 基礎入門」。
4.  在「**商品資料**」區塊，將商品類型設定為「**虛擬商品 (Virtual)**」。
5.  設定「**一般**」頁籤中的「**售價**」，例如 `2,000` 元。
6.  你會發現左側多了一個「**LearnDash**」頁籤，點擊它。
7.  在「**Associated Courses**」欄位，**搜尋並選擇**我們剛剛建立的「WordPress 基礎入門」課程。
8.  完成後，點擊「**發佈**」。

``

**Moksa 技巧：** 現在發生了什麼事？

*   學員會在你的商店看到一個 2,000 元的「WordPress 基礎入門」商品。
*   他可以像購買一般商品一樣，將它加入購物車，並使用我們設定好的綠界信用卡、ATM 或 LINE Pay 結帳。
*   當 WooCommerce 系統確認訂單狀態為「**已完成**」時，`LearnDash WooCommerce Integration` 外掛會**自動**將這位顧客註冊進「WordPress 基礎入門」這門課程。
*   學員就可以在他的「我的帳號」頁面中，看到他購買的課程並開始學習了！

這個流程完美地結合了 WooCommerce 強大的金流、行銷 (優惠券、點數) 功能，以及 LearnDash 專業的學習管理功能。

### 快速導覽：LifterLMS 的不同之處

如果你選擇了 LifterLMS，其基本概念是相似的，但有幾個關鍵差異你需要知道：

1.  **安裝與設定精靈：** LifterLMS 的免費版可以直接從 WordPress 外掛庫搜尋 `LifterLMS` 來安裝。它的設定精靈做得非常出色，會引導你設定金流、會員頁面等，非常適合新手。
2.  **課程架構：** LifterLMS 的課程架構稱為 `Course > Section > Lesson`，對應到 LearnDash 的 `Course > Lesson > Topic`，只是名詞不同，概念是相通的。
3.  **內建金流與會員功能：** 這是最大的不同。LifterLMS 的**免費核心**就內建了會員系統和基礎金流 (PayPal, Stripe) 的串接。如果你不想要使用 WooCommerce，可以直接用它內建的系統來收款。
4.  **WooCommerce 整合：** 如前述，若你希望使用 WooCommerce 的完整金流與生態系 (例如綠界)，你需要為 LifterLMS 購買**付費**的 `LifterLMS WooCommerce` 整合外掛。

### 整合與下一步

導入 LMS 只是第一步，真正的威力在於將它與我們學過的其他工具結合：

*   **頁面編輯器 (Kadence/Elementor):** LearnDash 和 LifterLMS 都有提供專屬的 Gutenberg Blocks 或 Elementor Widgets，讓你可以輕鬆地在任何頁面插入「課程列表」、「學員儀表板」、「課程進度條」等元素。
*   **會員系統 (YITH Membership):** 你可以建立一個「VIP 年度會員」方案，購買此會員資格的學員，可以免費學習網站上的**所有**或**特定分類**的課程。這可以透過 YITH Membership 建立一個會員方案，並將該方案可以存取的內容設定為指定的 LearnDash 課程。
*   **社群功能 (BuddyBoss):** 這是進階玩法。整合 BuddyBoss 平台後，你可以為每一門 LearnDash 課程自動建立一個專屬的學員社團 (Group)，讓學員可以在裡面討論、交流、問問題，打造一個充滿互動的學習社群，大幅提升課程價值與學員完課率。

### 總結

恭喜你！你已經掌握了在 WordPress 網站中導入專業線上課程系統的核心知識。無論你選擇功能全面、生態系強大的 **LearnDash**，還是從免費版開始、彈性擴充的 **LifterLMS**，都為你的知識變現事業打下了堅實的基礎。

請務必親自動手建立一門測試課程，並完成與 WooCommerce 商品的綁定與購買流程。只有實際走過一遍，你才能深刻體會到這套自動化系統的魅力。在下個章節，我們將探討更多 LMS 的進階設定與行銷玩法！
