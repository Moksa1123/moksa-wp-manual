# 02.01 UI/UX 設計稿基礎 (Grid, Spacing, Typography)

嗨，歡迎來到課程二的第一個章節。在我們正式開始用頁面編輯器（例如 Kadence Blocks 或 Elementor）打造精美頁面之前，我們必須先打好根基。這個根基，就是網頁設計中最重要的三大元素：**網格 (Grid)**、**間距 (Spacing)** 與**字體排印 (Typography)**。

許多新手在建站時，常常會覺得「為什麼我做的網站看起來就是不專業？」、「為什麼版面看起來很亂？」。答案往往就出在這三個基礎觀念上。理解並掌握它們，是從「會用工具」進階到「能做出好設計」的關鍵一步。

這一章，我們將不只談理論，更會告訴你這些觀念如何在 WordPress 的實務中應用。

---

## 1. 網格系統 (Grid System)：網站的隱形骨架

想像一下蓋房子，你會需要鋼筋水泥作為結構支撐。**網格系統 (Grid System)** 就是網頁的鋼筋，它是一個看不見的結構，用來組織、對齊頁面上的所有內容（文字、圖片、按鈕等），確保版面看起來整齊、有條理。

### 為什麼網格系統這麼重要？

*   **一致性與秩序感：** 網格為你的設計提供了一個清晰的結構，讓所有元素都能完美對齊，避免視覺上的混亂。
*   **提升閱讀效率：** 有序的版面能引導使用者的視線，讓他們更容易找到想看的資訊。
*   **響應式設計 (RWD) 的基礎：** 在手機、平板、電腦等不同尺寸的螢幕上，網格系統讓我們可以輕鬆地重新排列內容，提供最佳的瀏覽體驗。

### Moksa 實戰技巧：在 Kadence Blocks 中應用網格

在 WordPress 的世界裡，我們最常用「**欄 (Column)**」來實現網格系統。Moksa 推薦的 **Kadence Blocks** 外掛中，核心的 **Row Layout (橫列排版)** 區塊就是你最好的朋友。

**步驟 1：新增 Row Layout 區塊**

*   在頁面編輯器中，點擊 `+` 按鈕，搜尋並新增「**Row Layout**」區塊。

``

**步驟 2：選擇欄位佈局**

*   新增後，Kadence 會立刻請你選擇一個初始的欄位佈局，例如「兩欄均分」或「三欄均分」。這就是最基礎的網格應用。

``

**步驟 3：調整欄位寬度與間隔**

*   點選整個 Row Layout 區塊，你可以在右側的區塊設定中，精確地調整每一欄的寬度百分比。
*   **Column Gutter (欄位間隔)** 則是用來設定欄與欄之間的空白距離，這也是「間距」的一部分。

``

> **Moksa 技巧：**
> 專業的網頁設計通常基於「**12 欄網格系統**」。雖然在 Kadence 中你不需要手動計算，但這個概念能幫助你理解佈局的彈性。例如：
> *   `50% / 50%` 的兩欄佈局，相當於 `6/12` + `6/12`。
> *   `33% / 33% / 33%` 的三欄佈局，相當於 `4/12` + `4/12` + `4/12`。
> *   `25% / 75%` 的主副欄佈局，相當於 `3/12` + `9/12`。
> 更重要的是，你必須點擊右下角的「平板」和「手機」圖示，**分別設定不同裝置上的欄位佈局**。例如，在電腦上是三欄，在手機上通常會設定為堆疊成單欄，以確保最佳瀏覽體驗。

## 2. 間距 (Spacing)：賦予設計呼吸的空間

如果網格是骨架，那麼**間距 (Spacing)** 就是設計的「呼吸空間」。適當的留白，不僅不是浪費空間，反而是區分資訊、引導視覺、提升質感的關鍵。

在 CSS (網頁的樣式語言) 中，間距主要分為兩種：

*   **外距 (Margin):** 元素**邊框以外**的空間，像是兩個箱子之間的距離。它會把其他元素推開。
*   **內距 (Padding):** 元素**邊框以內**、內容以外的空間，像是箱子裡物品與箱壁之間的緩衝空間。它會把內容往內推。

### Moksa 實戰技巧：在 Kadence Blocks 中設定間距

幾乎所有的 Kadence 區塊（Row Layout, Advanced Heading, Info Box...）都在設定面板中提供了完整的間距控制。

**步驟 1：選取要調整的區塊**

*   點選任何一個你想調整間距的區塊，例如一個「Info Box」。

**步驟 2：找到 Padding & Margin 設定**

*   在右側的區塊設定中，找到「**Spacing Settings**」或類似的選項。
*   你可以在這裡分別設定 **Padding (內距)** 和 **Margin (外距)**。你可以設定上下左右四個方向的數值。

``

> **Moksa 技巧：**
> **建立一致性的間距系統**。不要隨心所欲地設定 `13px`, `27px`, `54px` 這種隨機數字。專業的做法是建立一個基準值，並使用它的倍數。
> *   **業界常用的基準值是 8px**。
> *   你的間距設定應該盡量是 `8px`, `16px`, `24px`, `32px`, `48px`, `64px`...
> *   例如，小區塊的內距可以是 `16px`，大區塊（Section）的上下內距可以是 `64px`。
> *   **在 Kadence 中，你可以點擊數值旁邊的單位（px, em, rem, %），並使用這些有規律的數值來建立視覺上的一致性與和諧感。** 這樣做出來的網站才會顯得專業、有條理。

## 3. 字體排印 (Typography)：網站的聲音與個性

**字體排印 (Typography)** 遠不只是挑選一個好看的字體。它是一門關於如何運用文字，使其清晰、易讀、並具有吸引力的藝術。好的字體排印能建立品牌形象，傳達情感，並引導使用者完成操作。

### 關鍵元素：

1.  **字體選擇 (Font Family):**
    *   **原則：** 少即是多。整個網站最多使用 1-2 種字體就非常足夠了。一種用於**標題 (Headings)**，一種用於**內文 (Body)**。
    *   **推薦：** 使用 Google Fonts 提供的免費、高品質字體，例如 Noto Sans TC (思源黑體) 是最安全、泛用的中文內文字體。

2.  **字體階層 (Hierarchy):**
    *   使用不同的大小、粗細、顏色來區分資訊的重要性。這能幫助使用者快速掃描頁面，抓住重點。
    *   在 HTML 語意中，標題從 `H1` 到 `H6` 代表了從最重要到最次要的層級。**一個頁面應該只有一個 H1**，這對 SEO (搜尋引擎優化) 也至關重要。

3.  **字體大小 (Font Size):**
    *   內文大小建議至少 `16px`，在現代高解析度螢幕上，`17px` 或 `18px` 會更舒適。
    *   標題大小則應明顯大於內文，以建立對比。

4.  **行高 (Line Height):**
    *   指每一行文字垂直方向的距離。適當的行高能大幅提升長篇文章的易讀性。
    *   **經驗法則：** 內文的行高通常設定為字體大小的 `1.5` 到 `1.7` 倍。

### Moksa 實戰技巧：在 Kadence 中設定全域字體

**最重要的觀念：不要在單一頁面的單一區塊中去獨立設定字體！** 這樣會造成全站風格混亂且難以維護。你應該在一個統一的地方進行設定。

**步驟 1：進入 WordPress 自訂器**

*   從後台左側選單進入「**外觀**」>「**自訂**」。

**步驟 2：找到字體排印設定**

*   在自訂器中，找到「**General (一般)**」 > 「**Typography (字體排印)**」。
*   在 Kadence 主題中，這個選項非常強大。

``

**步驟 3：設定基礎與標題字體**

*   **Body Font (內文字體):** 在這裡設定你整個網站的預設內文字體、大小、行高等。
*   **Headings (H1-H6):** 分別設定各級標題的字體樣式。你可以讓標題繼承內文字體，只改變大小與粗細；或者為標題選擇另一種不同的字體。

> **Moksa 技巧：**
> 在你開始建立任何頁面**之前**，請務必先來到「自訂器」中，完成**全域色彩 (Global Colors)** 與**全域字體排印 (Global Typography)** 的設定。這是一個專業開發者的標準流程，能為你省下數十個小時的重複修改時間，並確保你的網站從頭到尾都擁有一致、專業的視覺風格。

### 總結

恭喜你！你已經學習了構成專業網頁設計的三大基石。

*   **網格 (Grid)** 提供結構與秩序。
*   **間距 (Spacing)** 提供呼吸與清晰度。
*   **字體 (Typography)** 提供個性與易讀性。

請牢記這些原則。在接下來的課程中，當我們開始實作頁面時，你會不斷地回來應用這些觀念。這正是區分一個普通網站與一個卓越網站的秘訣所在。
