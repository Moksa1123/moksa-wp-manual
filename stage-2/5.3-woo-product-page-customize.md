# 05.03 (實作) 客製化「單一商品頁面」 (使用 Kadence Elements)

嗨，歡迎來到 Moksa 的課程！在之前的章節中，我們已經學會了如何上架商品，並了解了 WooCommerce 預設的商品頁面長什麼樣子。雖然預設的版型功能完整，但它通常比較「公版」，缺乏品牌特色，也可能不是最適合你目標客群的瀏覽動線。

這就是為什麼客製化「單一商品頁面」如此重要。一個經過精心設計的商品頁面，可以有效提升使用者體驗、傳達品牌價值，最終提高轉換率。

在這堂實作課中，我們將會使用 **Moksa 技術棧**中的核心工具之一：**Kadence Pro** 的 **Elements (元素)** 功能，來從零開始，打造一個既美觀又具備高轉換潛力的客製化商品頁面。這個方法讓你不需要寫任何一行程式碼，就能透過視覺化的拖拉方式，完全掌握商品頁的每一個細節。

---

### 課前準備

在開始之前，請確保你的網站已經滿足以下條件：

1.  **已安裝並啟用 WooCommerce**：並且已經新增了至少 2-3 個不同的商品，包含商品圖片、價格、簡短描述與詳細描述，這樣我們才能即時預覽設計成果。
2.  **已安裝並啟用 Kadence Blocks (免費版) 與 Kadence Pro (付費版)**：因為 **Elements** 是 Kadence Pro 的核心功能，必須要有 Pro 版本才能使用。

準備好了嗎？讓我們開始動手吧！

---

### 步驟 1：建立一個新的 Kadence Element (元素)

首先，我們要建立一個專門用來取代預設 WooCommerce 商品頁版型的「樣板 (Template)」元素。

1.  從 WordPress 後台，前往左側選單的「**Kadence**」>「**Elements**」。
2.  點擊頁面上方的「**Add New**」按鈕。
3.  系統會詢問你要建立哪種類型的 Element。請選擇「**TEMPLATE**」。

``

---

### 步驟 2：設定 Element 的核心選項 (顯示位置與條件)

建立好 Element 之後，我們不會馬上開始設計畫面，而是要先設定最重要的規則：**「這個版型要取代哪個頁面？」** 以及 **「它應該在哪個條件下出現？」**。

1.  首先，在最上方的標題欄位，為這個 Element 取一個好辨識的名稱，例如「**客製化單一商品頁版型 - 全站通用**」。

2.  接著，看到右側的設定欄，點擊 **Element Settings** 的圖示 (一個小齒輪跟三條線)。

    ``

3.  找到「**Preview Settings**」區塊。點擊「Select Preview Post」，搜尋並選擇一個你已經建立好的商品。**Moksa 技巧：** 這個設定非常重要，它能讓你在後續的設計過程中，直接載入該商品的資料（如標題、圖片、價格）來即時預覽，大幅提升設計效率。

    ``

4.  接著是最關鍵的「**Placement (顯示位置)**」設定。點擊下拉選單，找到 **WooCommerce** 的區塊，然後選擇「**Replace Single Product**」。這個選項的意思就是「用我現在設計的這個版型，去完全取代掉預設的單一商品頁」。

    ``

5.  最後是「**Display Settings (顯示條件)**」。這裡決定了這個版型要在哪些商品頁面生效。
    *   在 **Show On** 的下拉選單中，選擇「**Single Products**」。
    *   右邊的條件預設會是「**All**」，代表套用到「所有商品」。
    *   **Moksa 技巧：** 未來你可以利用這個功能為**特定商品分類**建立專屬的版型。例如，「服飾類」的版型可以多一個尺寸表區塊；「電子產品類」的版型可以多一個規格比較表。你只需要再建立一個新的 Element，並將這裡的條件設定為 `Select -> Product Category -> [你的分類]` 即可。

    ``

設定完成後，點擊右上角的「**儲存草稿**」。

---

### 步驟 3：使用 Kadence Blocks 設計商品核心資訊區塊

現在，我們可以開始用 Kadence Blocks 自由地設計版面了。一個經典的商品頁佈局是左側為商品圖庫，右側為商品資訊與購買按鈕。

1.  在編輯器中，新增一個「**Row Layout**」區塊，並選擇「**兩欄 (均分)**」的結構。

2.  **設計左欄 (商品圖庫)：**
    *   點擊左欄中的 `+` 號。
    *   搜尋並新增「**Product Gallery**」這個 Kadence 專屬區塊。
    *   新增後，你會發現它自動抓取了我們在步驟 2 中設定的預覽商品的圖庫。你可以點擊這個區塊，並在右側設定欄調整圖片輪播、縮圖、放大鏡等外觀細節。

    ``

3.  **設計右欄 (商品資訊與購買)：**
    *   這是最重要的轉換區域，我們會使用 Kadence 強大的「**動態內容 (Dynamic Content)**」功能來自動抓取商品資料。
    *   **商品標題：** 新增一個「**Advanced Text**」區塊 (標示為 H1)。在區塊工具列上，點擊「**動態內容**」的圖示 (一個資料庫的樣子)，選擇 **Content** > **Post Title**。它就會自動變成目前預覽商品的標題。
        ``
    *   **商品價格：** 新增 Kadence 專屬的「**Product Price**」區塊。它會自動抓取價格，包含特價的樣式。
    *   **商品簡短描述：** 新增一個「**Advanced Text**」區塊。同樣使用「動態內容」功能，這次選擇 **Content** > **Post Excerpt**。
    *   **加入購物車按鈕：** 這是最重要的區塊！新增 Kadence 專屬的「**Add to Cart**」區塊。這個區塊會自動生成數量選擇器和「加入購物車」按鈕。你可以在右側設定欄客製化按鈕的顏色、樣式和文字。
    *   **商品 Meta 資訊：** 新增「**Product Meta**」區塊，它會自動顯示 SKU、商品分類等資訊。

完成後，你的右欄看起來應該會像這樣，充滿了各種從商品資料庫動態載入的資訊。

``

---

### 步驟 4：加入商品詳細內容與相關商品

核心購買區塊完成後，我們還需要把商品詳細描述、規格、評論，以及相關商品等資訊加回來。

1.  在剛剛建立的兩欄式「Row Layout」區塊下方，新增一個 Kadence 的「**Product Tabs**」區塊。
    *   這個區塊非常方便，它會自動抓取並生成 WooCommerce 預設的頁籤，包含「商品描述」、「其他資訊」(商品屬性) 和「評論」。

    ``

2.  在「Product Tabs」區塊下方，新增 Kadence 的「**Related Products**」區塊。
    *   這個區塊會自動根據演算法，顯示與當前商品相關的其他產品，有助於交叉銷售，提升客單價。
    *   你可以在右側設定欄調整要顯示的商品數量以及欄位數。

    ``

---

### 步驟 5：發佈與最終測試

恭喜你！你已經完成了客製化商品頁面的所有設計步驟。

1.  點擊右上角的藍色按鈕「**發佈**」。
2.  **清空快取：** 如果你的網站有安裝 **WP Rocket** 這類的快取外掛，請務必清空一次快取，確保前台能載入最新的版型。
3.  **實際測試：** 開啟你網站的任一商品頁面，你應該會看到它已經完全被我們剛剛設計的全新版型所取代。檢查所有動態內容是否正確顯示（標題、價格、圖片等），並試著點擊「加入購物車」按鈕，確認功能一切正常。

### 總結與 Moksa 進階技巧

透過 Kadence Elements，我們成功地擺脫了 WooCommerce 的預設版型限制，打造了一個獨一無二的商品頁面。這個工作流程不僅賦予你完全的設計自由，更重要的是，它讓你能根據行銷策略去優化使用者動線。

**Moksa 進階技巧分享：**

*   **整合 ACF 自訂欄位：** 如果你有使用 Advanced Custom Fields (ACF) 為商品新增額外的欄位（例如：材質、產地、注意事項），你可以在設計 Element 時，使用「Advanced Text」區塊的動態內容功能，直接選擇 `ACF Field`，將這些自訂欄位的資訊顯示在商品頁的任何位置！
*   **A/B 測試版型：** 對於追求極致轉換率的電商高手，你可以針對「同一個商品分類」設計兩種不同排版的 Element (例如 A 版按鈕在右、B 版按鈕置中)，然後搭配行銷工具進行 A/B 測試，用數據找出哪個版型最受你的顧客青睞。
*   **創造「登陸頁式」商品頁：** 你可以在這個版型中自由加入客戶見證、FAQ、影片介紹、品牌故事等區塊，將一個單純的商品頁，打造成一個資訊完整、說服力十足的銷售登陸頁 (Landing Page)。
