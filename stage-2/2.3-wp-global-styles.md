# 02.03 (實作) 將 UI Kit 導入 WP (全域色彩、字體)

嗨，歡迎來到 MoksaWP 網站設計與進階客製化課程！

在之前的章節中，我們理解了網站設計的前置作業，其中最重要的產出之一就是 **UI Kit (使用者介面工具包)**。UI Kit 定義了整個網站的視覺一致性，包含了品牌色彩、字體、間距、按鈕樣式等核心設計元素。

直接在頁面編輯器中「手動」調整每一個標題、每一顆按鈕的顏色與字體，不僅效率低落，更容易造成視覺風格不統一。專業的作法，是將 UI Kit 的規範「導入」到 WordPress 的核心設定中，讓全站的元素都能**自動繼承**這些設計規範。

這堂課的目標，就是帶你完成這個至關重要的 foundational step。我們將以 **Moksa 技術棧**中最推薦的 **Kadence Theme (佈景主題)** 為範例，將你的 UI Kit 導入 WordPress，建立一個**全域 (Global)** 的色彩與字體系統。

---

### 課前準備

在開始之前，請確認你手邊已經有：

1.  **一份明確的 UI Kit**：你不需要是設計師，但你必須清楚知道網站的設計規範。至少需要包含：
    *   **主要顏色 (Primary Color)**：例如品牌主色、按鈕顏色。
    *   **輔助顏色 (Secondary Color)**：例如次要按鈕、點綴顏色。
    *   **文字顏色 (Text Color)**：內文的標準顏色。
    *   **背景顏色 (Background Color)**：網站主要的底色。
    *   **字體規範**：標題 (H1, H2, H3...) 與內文 (Paragraph) 各自使用的字體家族 (Font Family)、字重 (Font Weight) 與字體大小 (Font Size)。
2.  **已安裝並啟用 Kadence Theme**：MoksaWP 環境預設使用 Kadence 作為佈景主題，它強大的全域設定功能是我們這次實作的核心。

準備好了嗎？我們開始吧！

---

### 步驟 1：設定全域色彩 (Global Color Palette)

首先，我們要將 UI Kit 中的品牌色票，建立成一個全站共用的「調色盤」。

1.  登入 WordPress 後台，從左側選單前往「**外觀**」>「**自訂**」。
    ``

2.  進入自訂器後，點擊左側面板的「**色彩與字型**」>「**色彩**」。
    ``

3.  你會看到 Kadence 提供的「**全域色彩**」。這裡有三組預設的調色盤，但我們的目標是建立屬於我們自己的。點擊你想要編輯的調色盤（通常是第一個）。
    ``

4.  現在，我們要將 UI Kit 的色碼（例如 `#3A86FF`）填入對應的格子中。
    *   **Moksa 技巧：** Kadence 的色票有其邏輯性，建議這樣對應：
        *   **前兩個（Accent Colors）**：設定為你的「**品牌主色**」與「**輔助色**」。這兩個顏色會被按鈕、連結等互動元素預設使用。
        *   **中間四個（Text Colors）**：設定為你的「**標題顏色**」、「**內文顏色**」以及一些深淺不同的灰色。
        *   **最後三個（Background Colors）**：設定為你的「**網站底色**」、區塊背景色等。通常會是白色、淺灰色。

5.  點擊色票格子，然後在色碼輸入框中，貼上你 UI Kit 中定義的 HEX 色碼。重複這個動作，直到所有顏色都設定完成。
    ``

6.  完成後，記得點擊右上角的藍色按鈕「**發佈**」，儲存你的設定。

恭喜！你已經完成了全域色彩的設定。現在，當你在 Kadence Blocks 或其他地方要選擇顏色時，這個調色盤會優先出現，確保你使用的都是品牌標準色。

---

### 步驟 2：設定全域字體 (Global Typography)

接下來，我們要設定全站的字體。這能確保你的 H1 標題、H2 標題和內文，在任何頁面都長得一模一樣。

1.  同樣在「**外觀**」>「**自訂**」的面板中，前往「**色彩與字型**」>「**字型**」。
    ``

2.  **設定基礎字型 (Base Font)**：
    *   第一個選項是「**基礎字型**」。這裡的設定會被網站上**所有沒有**特別指定的文字（例如內文、段落）所繼承。
    *   點開它，根據你的 UI Kit 設定「**字型家族 (Font Family)**」。你可以選擇 Google Fonts 提供的字體。
    *   設定「**字型變化 (Variant)**」，也就是字重 (Font Weight)。一般內文建議設定為 `normal` 或 `400`。
    *   設定「**字體大小 (Font Size)**」與「**行高 (Line Height)**」。**Moksa 技巧：** 記得點擊旁邊的裝置圖示（電腦、平板、手機），分別為不同裝置設定最適當的字體大小，這對 RWD (響應式設計) 至關重要！
    ``

3.  **設定標題字型 (Headings)**：
    *   回到上一層，你會看到 `H1`, `H2`, `H3`, `H4`, `H5`, `H6` 的設定選項。
    *   點開「**H1**」，如同設定基礎字型一樣，根據你的 UI Kit 規範，設定 H1 標題的「**字型家族**」、「**字型變化**」（標題通常會用比較粗的字重，如 `bold` 或 `700`）、以及在不同裝置上的「**字體大小**」與「**行高**」。
    ``
    *   依序完成 H2, H3... 等其他標題的設定。雖然有點繁瑣，但這是一勞永逸的工作。

4.  設定完成後，再次點擊右上角的「**發佈**」儲存。

---

### 步驟 3：驗收成果

設定完畢後，我們該如何確認設定是否生效？

1.  新增一個頁面，命名為「樣式測試」或「Style Guide」。
2.  使用 Kadence Blocks 編輯器，在頁面上依序插入以下區塊：
    *   標題區塊 (Heading)，並分別設為 H1, H2, H3。
    *   段落區塊 (Paragraph)，輸入一些測試文字。
    *   按鈕區塊 (Advanced Button)。
3.  **不要**對這些區塊做任何**個別的**樣式調整。
4.  儲存草稿並預覽頁面。

你會發現，頁面上的 H1, H2, H3、內文、甚至按鈕的預設顏色，都已經**自動套用**了我們剛剛在全域設定中定義的樣式！這就是全域設定的威力。

### 結論

**先建立規則，再開始設計。**

將 UI Kit 導入 WordPress 的全域色彩與字體系統，是建立一個專業、高品質網站的基石。這個步驟雖然看似基礎，卻能大幅提升你後續建立頁面的效率，並確保整個網站的視覺風格高度一致。

從現在開始，當你要設計任何頁面時，請務必優先使用你在這個章節設定好的「**全域色彩**」與「**預設字體樣式**」，避免手動去覆蓋它們。養成這個好習慣，你的網站維護之路將會輕鬆許多。
