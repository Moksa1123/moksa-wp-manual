# 04.05 (實作) 打造「全站頁尾 (Footer)」

嗨，歡迎來到這個實作單元！我是 Moksa 的首席課程導師。

網站的「頁尾 (Footer)」雖然位於頁面的最底部，但它扮演著不可或缺的角色。一個好的頁尾不僅能提供重要的網站資訊、導覽連結，更能建立訪客的信任感，是專業網站的標誌。

在這個章節，我們將一步步帶你使用 **Moksa 技術棧**中推薦的 **Kadence Blocks**，透過其強大的 **Elements (元素)** 功能，來設計一個專業、美觀且適用於全站的頁尾。

---

### 課前準備：頁尾的規劃與設計

在我們動手之前，先來思考一下，一個好的電商網站頁尾通常會包含哪些元素？

*   **公司資訊：** Logo、品牌簡介、聯絡方式 (地址、電話、Email)。
*   **顧客服務：** 常見問題 (FAQ)、運送政策、退換貨政策、隱私權條款、服務條款。
*   **網站導覽：** 關於我們、商店頁面、最新消息、會員專區。
*   **社群連結：** Facebook、Instagram、LINE 官方帳號等圖示連結。
*   **信任標章：** 金流服務的 Logo (例如：**綠界 ECPay**、**LINE Pay**、Visa、Mastercard)，讓顧客安心付款。
*   **版權宣告：** 例如 `Copyright © 2024 MoksaWeb. All Rights Reserved.`。

**Moksa 技巧：** 先用紙筆或任何你習慣的設計工具，簡單畫出頁尾的草圖，規劃好要放幾個欄位、每個欄位要放什麼內容。這樣在實作時會更有效率，思路也會更清晰。

---

### 實作開始：使用 Kadence Elements 打造全站頁尾

我們將使用 Kadence 主題內建的 `Elements` 功能，這個功能可以讓我們建立一個區塊，並指定它出現在網站的特定位置，例如頁首、頁尾，或是安插在文章內容中，非常強大。

#### 步驟 1：啟用並進入 Kadence Elements 功能

1.  從 WordPress 後台，前往左側選單的「**外觀**」>「**Kadence**」。
2.  確認 `Hooked Elements` 這個功能是**啟用 (Enabled)** 的狀態。
3.  接著，點擊「**自訂**」下方的「**Elements**」連結，進入 Elements 的管理介面。
4.  點擊畫面右上角的「**Add New**」按鈕，準備新增一個頁尾元素。

` `

#### 步驟 2：建立一個新的「頁尾 (Footer)」元素

1.  點擊「Add New」後，系統會跳出一個視窗，詢問你要建立哪種類型的 Element。
2.  請選擇「**Footer**」這個選項。
3.  系統會自動進入區塊編輯器畫面。首先，在最上方的「**新增標題**」欄位，為你的頁尾取一個好辨識的名稱，例如「**全站通用頁尾**」。這個標題只會顯示在後台，方便你管理。

` `

#### 步驟 3：設定頁尾的「顯示規則 (Display Settings)」

這是最關鍵的一步，我們要告訴系統這個頁尾要顯示在網站的哪些地方。

1.  點擊編輯器右上角的 **Kadence 圖示 (Element Settings)**。
2.  在右側邊欄找到「**Display Settings (顯示設定)**」區塊。
3.  在 `Show On` (顯示於) 的下拉選單中，選擇「**Entire Site (整個網站)**」。
4.  在 `User Settings` (使用者設定) 區塊，確認 `Visible to` (對誰可見) 是設定為「**All Users (所有使用者)**」。
5.  **重要觀念：** `Display Settings` 是 Kadence Elements 的精髓，你可以非常精準地控制元素顯示的位置，例如只顯示在「單一商品頁面」或只對「已登入會員」顯示。但對於頁尾，我們通常會選擇「整個網站」。

` `

#### 步驟 4：使用 Kadence Blocks 設計頁尾版面

現在，我們可以像編輯一般頁面一樣，使用區塊編輯器來設計頁尾的內容與排版了。這裡我們以一個常見的「四欄式」頁尾為例。

1.  點擊 `+` 新增區塊，搜尋並選擇「**Row Layout (行列版面)**」。
2.  選擇「**四欄 (Four Columns)**」的版面配置。
3.  **第一欄 (品牌資訊)：**
    *   新增「**Image (圖片)**」區塊，上傳你的 Logo。
    *   在 Logo 下方新增「**Advanced Text (進階文字)**」區塊，輸入簡短的品牌介紹。
4.  **第二欄 (顧客服務)：**
    *   新增「**Advanced Text (進階文字)**」區塊，將其設定為 `H4` 標題，輸入「顧客服務」。
    *   在標題下方，再次新增「**Advanced Text (進階文字)**」區塊，輸入「常見問題」、「退換貨政策」、「隱私權條款」等文字，並分別為它們加上對應頁面的連結。
5.  **第三欄 (社群連結)：**
    *   新增「**Advanced Text (進階文字)**」區塊，設定為 `H4` 標題，輸入「追蹤我們」。
    *   在標題下方，新增「**Icons (圖示)**」區塊，點擊「Add Icon」並選擇 Facebook、Instagram 等社群圖示，並為每個圖示設定你的社群連結。
6.  **第四欄 (金流支援)：**
    *   新增「**Advanced Text (進階文字)**」區塊，設定為 `H4` 標題，輸入「付款方式」。
    *   在標題下方，新增「**Section (區塊)**」或「**Row Layout**」區塊，並在裡面放入多個「**Image (圖片)**」區塊，分別上傳 **綠界 ECPay**、**LINE Pay**、Visa、Mastercard 的 Logo 圖片，增加顧客的信任感。

` `

#### 步驟 5：設計頁尾底部的版權宣告 (Copyright Bar)

通常頁尾最下方還會有一條深色的版權宣告列。

1.  在剛剛建立的四欄式版面下方，再新增一個「**Row Layout (行列版面)**」，這次選擇「**單欄**」。
2.  點選這個新的 Row Layout，在右側的「**Style (樣式)**」設定中，為它設定一個**深色的背景** (例如 `#232323`)。
3.  在版面中新增「**Advanced Text (進階文字)**」區塊。
4.  **Moksa 技巧：** 為了讓年份自動更新，你可以輸入 `Copyright © {current_year} {site_title}`。Kadence 的進階文字區塊支援動態內容，它會自動將 `{current_year}` 轉換為當前年份，並將 `{site_title}` 轉換為你的網站標題。
5.  將這段文字設定為**白色**，並**置中對齊**。

#### 步驟 6：發佈與最終檢查

1.  完成所有設計後，點擊右上角的藍色按鈕「**發佈 (Publish)**」或「**更新 (Update)**」。
2.  打開你的網站前台 (建議使用瀏覽器的「無痕模式」來避免快取影響)，檢查首頁、商店頁面、商品頁面等不同頁面，確認新的全站頁尾是否都已正確顯示。

### 補充說明：如果我使用 Elementor Pro 呢？

對於使用 Elementor Pro 的學員，其概念是完全一樣的，只是工具不同。

1.  請至 WordPress 後台 >「**範本 (Templates)**」>「**Theme Builder**」。
2.  在 Theme Builder 介面中，選擇「**Footer**」，並點擊「**Add New**」。
3.  使用 Elementor 的編輯器與各種 Widget (小工具) 來設計你的頁尾版面。
4.  完成設計後，點擊「**發佈 (Publish)**」，Elementor 會請你設定「**顯示條件 (Display Conditions)**」。
5.  點擊「**ADD CONDITION**」，並將條件設定為「**Entire Site (整個網站)**」，然後儲存。
