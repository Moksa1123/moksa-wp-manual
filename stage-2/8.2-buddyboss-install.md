# 08.02 (實作) 導入「BuddyBoss」平台

嗨，歡迎來到這個實作單元！在上一個章節中，我們討論了建立線上社群的各種可能性。現在，我們要動手將這個想法化為現實。在這個單元，我們將一步步帶你安裝、設定並啟用 WordPress 生態系中最強大、也最受歡迎的社群解決方案 —— **BuddyBoss**。

BuddyBoss 不僅僅是一個外掛，它是一個完整的「**平台 (Platform)**」加上一個高度整合的「**主題 (Theme)**」。透過它，你可以將你的 WordPress 網站，從一個單純的內容網站或電商網站，轉變為一個功能齊全、互動性極高的線上社群。無論是建立課程討論區、付費會員專屬社團，還是打造一個類似 Facebook 的社交網絡，BuddyBoss 都是 Moksa 團隊的首選方案。

讓我們開始吧！

---

### 前置準備：你需要什麼？

在開始安裝之前，請確保你已經準備好以下項目：

1.  **一個乾淨的 WordPress 網站：** 雖然 BuddyBoss 可以與現有網站整合，但為了教學單純化，我們建議在一個相對乾淨的環境下開始。
2.  **BuddyBoss 官方授權檔案：** 你需要從 BuddyBoss 官網購買並下載最新的 `buddyboss-platform` (平台外掛) 與 `buddyboss-theme` (主題) 的 .zip 檔案。
3.  **伺服器環境檢查：** **Moksa 技巧**：BuddyBoss 是一個功能豐富的平台，相對也比較消耗主機資源。請確保你的主機環境符合 BuddyBoss 的[官方建議規格](https://www.buddyboss.com/resources/docs/getting-started/server-requirements/)，特別是 **PHP 版本**、**記憶體限制 (Memory Limit)** 與 **執行時間 (Max Execution Time)**。一個好的主機是社群網站成功的基石。

---

### 步驟 1：安裝與啟用 BuddyBoss Platform (平台外掛)

BuddyBoss 的核心功能都建立在「平台外掛」之上。所以我們的第一步是安裝它。

1.  登入你的 WordPress 後台。
2.  前往「**外掛**」>「**安裝外掛**」>「**上傳外掛**」。
3.  點擊「**選擇檔案**」，並選擇你從官網下載的 `buddyboss-platform.zip` 檔案。
4.  點擊「**立即安裝**」，並在安裝完成後點擊「**啟用外掛**」。

``
*（截圖：WordPress 後台啟用 BuddyBoss Platform 外掛的畫面）*

啟用後，你會看到 WordPress 後台左側選單多出了一個「**BuddyBoss**」的主要選項，這代表平台已經成功安裝。

---

### 步驟 2：安裝與啟用 BuddyBoss Theme (主題)

雖然 BuddyBoss 平台可以與多數主題相容，但要達到最佳的視覺效果與功能整合，**強烈建議**使用官方的 BuddyBoss Theme。

1.  前往「**外觀**」>「**佈景主題**」。
2.  點擊上方的「**安裝佈景主題**」>「**上傳佈景主題**」。
3.  點擊「**選擇檔案**」，並選擇你下載的 `buddyboss-theme.zip` 檔案。
4.  點擊「**立即安裝**」。
5.  安裝完成後，點擊「**啟用**」來啟用 BuddyBoss 主題。

``
*（截圖：啟用 BuddyBoss Theme 的畫面）*

啟用主題後，你的網站前台外觀會立刻改變。別擔心，這只是個開始，我們稍後會進行詳細的客製化設定。

---

### 步驟 3：執行 BuddyBoss 設定精靈 (Setup Wizard)

當你同時啟用 Platform 與 Theme 後，BuddyBoss 會引導你進入一個非常友善的「**設定精靈**」。這個精靈能幫助我們快速完成基礎設定。

1.  在畫面上方，你會看到一個提示，點擊「**Run the Setup Wizard**」按鈕開始。
2.  **License Key (授權碼)：** 輸入你在 BuddyBoss 官網購買後取得的授權碼，以確保你能收到後續的更新與技術支援。
3.  **Components (功能元件)：** 這是最重要的一步。BuddyBoss 採用模組化設計，你可以選擇要啟用哪些社群功能。
    *   **Moksa 技巧**：對於一個標準的線上課程或會員社群，我們建議**至少啟用**以下元件：
        *   `Community Features`
            *   **Extended Profiles:** 必選，讓會員有更豐富的個人資料頁。
            *   **Member Connections:** 類似 Facebook 的「好友」功能。
            *   **Private & Group Messaging:** 讓會員間可以私訊或群聊。
            *   **Social Groups:** 建立公開、私密或隱藏社團的核心功能。
        *   `Forum Features`
            *   **Group Forums:** 讓每個社團都能有自己的討論區。
        *   `Media Features`
            *   **Media Uploading:** 允許會員上傳照片與影片。

    ``
    *（截圖：BuddyBoss 設定精靈選擇 Components 的畫面）*

4.  **Settings (設定)：**
    *   **Branding:** 上傳你的 Logo，設定網站主色調。這些之後都還能修改。
    *   **Integrations:** 如果你的網站已經安裝了 **LearnDash** 或 **WooCommerce**，BuddyBoss 會自動偵測到並詢問是否要啟用整合功能。**請務必勾選**，這是實現社群學習與電商結合的關鍵。

5.  **Finish (完成)：** 完成精靈後，BuddyBoss 會自動建立一些必要頁面（例如：動態牆、會員列表、社團列表等），並將它們指派到對應的功能。

---

### 步驟 4：核心設定校調

設定精靈幫我們完成了 80% 的工作，但還有一些細節需要我們手動調整，以符合台灣市場與 Moksa 技術棧的需求。

1.  **頁面設定檢查：**
    *   前往「**BuddyBoss**」>「**Pages**」。
    *   這裡會列出所有由 BuddyBoss 管理的核心頁面。請確認「**註冊 (Register)**」與「**啟用 (Activate)**」頁面都已正確指派。
    *   **Moksa 技巧**：如果你希望完全自訂註冊流程（例如透過 FluentCRM 或 WooCommerce），可以考慮在此處停用 BuddyBoss 的預設註冊頁面。

2.  **整合 LearnDash/LifterLMS：**
    *   前往「**BuddyBoss**」>「**Integrations**」。
    *   確認 LearnDash (或 LifterLMS) 的整合是啟用的。
    *   啟用後，當學員進入一個課程時，課程頁面會自動增加「**社團 (Group)**」與「**論壇 (Forum)**」的頁籤，讓你可以將課程與特定的討論社團綁定，創造絕佳的學習體驗。

3.  **整合 WooCommerce：**
    *   前往「**BuddyBoss**」>「**Integrations**」。
    *   啟用 WooCommerce 整合後，會員的個人資料頁會多出「**商店**」、「**購物車**」、「**訂單**」等分頁，讓會員可以在自己的空間內管理購物相關的所有資訊。

4.  **設定 Email 通知：**
    *   前往「**BuddyBoss**」>「**Emails**」。
    *   BuddyBoss 會在各種社群互動時（如：有人加你好友、邀請你加入社團）寄送大量 Email 通知。
    *   **Moksa 技巧**：為了確保這些重要的通知信能順利寄達，並統一管理範本，我們強烈建議使用 **FluentCRM** 搭配第三方 SMTP 服務（如 SendGrid, Mailgun）來取代 WordPress 預設的郵件功能。你只需要在 FluentCRM 中設定好 SMTP，BuddyBoss 的信件就會自動透過它來發送。

---

### 步驟 5：客製化外觀與品牌識別

最後，讓我們來調整網站的門面，讓它看起來更符合你的品牌風格。

1.  **佈景主題選項：**
    *   前往「**BuddyBoss**」>「**Theme Options**」。
    *   這裡就是 BuddyBoss Theme 的控制中心，你可以在這裡調整幾乎所有的視覺元素。
    *   **Logo & Favicon:** 上傳你的品牌 Logo，包括給桌面、手機版以及 Email 用的不同版本。
    *   **Typography:** 設定全站的字體。**Moksa 技巧**：為了在台灣有最佳的瀏覽體驗，建議選擇支援繁體中文的字體，例如「**思源黑體 (Noto Sans TC)**」。
    *   **Colors:** 調整網站的主色、連結顏色、按鈕顏色等。
    *   **Header:** 設定網站頂部的導覽列樣式，包含選單、登入/註冊按鈕、個人頭像下拉選單等。

2.  **使用頁面編輯器自訂頁面：**
    *   BuddyBoss Theme 與 **Kadence Blocks** 及 **Elementor** 有良好的相容性。
    *   對於非社群核心的頁面（例如：首頁、關於我們、聯絡我們），你可以像平常一樣，使用你熟悉的頁面編輯器來拖拉設計，打造獨一無二的頁面。
