# 04.04 (實作) 打造「全站頁首 (Header)」

大家好！我是 Moksa 的首席課程導師。頁首 (Header) 是使用者對你網站的第一印象，它承載了品牌 Logo、主要導覽、以及最重要的行動呼籲 (Call-to-Action)。一個設計良好、反應快速的頁首，能大幅提升使用者體驗與網站專業度。

在這堂實作課中，我們將會學習如何使用 **Moksa 技術棧**中推薦的 **Kadence 主題**內建的「**頁首建置器 (Header Builder)**」來打造一個功能完整、專業且在電腦、平板、手機上都完美呈現的「全站頁首」。

**為什麼使用 Kadence 頁首建置器？**

*   **視覺化操作：** 直覺的拖放式介面，所見即所得。
*   **高度整合：** 與主題完美結合，不需安裝額外外掛，效能最佳。
*   **功能強大：** 內建多種常用元件 (如按鈕、社群圖示)，並提供豐富的設計選項。
*   **完整響應式設計：** 可針對不同裝置獨立設計不同的頁首排版。

---

### 步驟 1：進入 Kadence 頁首建置器

首先，我們需要找到這個強大的工具。

1.  從 WordPress 後台，點擊左側選單的「**外觀 (Appearance)**」。
2.  在下拉選單中選擇「**自訂 (Customize)**」。

`[截圖：WordPress 後台，滑鼠停在「外觀」->「自訂」的選項上]`

3.  進入自訂畫面後，在左側的設定面板中，點擊「**頁首 (Header)**」。

`[截圖：WordPress 自訂器畫面，左側面板中「頁首 (Header)」的選項被框起來]`

太棒了！現在你應該可以看到 Kadence 頁首建置器的主要操作介面。它位於畫面下方，由三列橫條與許多可用的元件組成。

### 步驟 2：認識頁首建置器介面

在我們開始動手之前，先花一分鐘了解這個介面。

`[截圖：Kadence 頁首建置器的完整介面，包含上方預覽區、下方三列式編輯區、左側元件區，並用數字標示各區塊]`

*   **預覽區 (Preview Area):** 畫面的主要區域，即時顯示你的設計成果。
*   **可用項目 (Available Items):** 左側存放所有可以被拖曳到頁首的元件，例如 Logo、導覽選單、按鈕、搜尋框等。
*   **三列式編輯區 (Three-Row Builder):** 這是核心區域，由上到下分別是「**頂部列 (Top Row)**」、「**主要列 (Main Row)**」、「**底部列 (Bottom Row)**」。你可以將「可用項目」拖曳到這三列的任何位置（左、中、右）。
*   **裝置切換 (Device Switcher):** 在編輯區的左下角，有「**電腦**」、「**平板**」、「**手機**」三個圖示，點擊後可以分別預覽和設定不同裝置的頁首樣式。
*   **Moksa 技巧：** 一般來說，**主要列**會放置最重要的項目，如 **Logo** 和**主要導覽**；**頂部列**則放次要資訊，如聯絡電話、社群連結；**底部列**較少使用，可用於公告或次要選單。

### 步驟 3：實作「電腦版 (Desktop)」頁首

現在，我們來打造一個最常見的專業頁首佈局。

1.  **放置 Logo：**
    *   從「可用項目」中，將「**Logo**」元件拖曳到「**主要列**」的**最左側**。
    *   點擊剛剛放置的 Logo 元件上的「**鉛筆**」圖示，或左側面板對應的 Logo 設定。
    *   上傳你的 Logo 圖片，並調整「**Logo 最大寬度 (Logo Max Width)**」到一個合適的尺寸（例如：150px）。

    `[截圖：Logo 設定畫面，顯示如何上傳圖片以及調整寬度]`

2.  **放置主要導覽選單：**
    *   將「**主要導覽 (Primary Navigation)**」元件拖曳到「**主要列**」的**中間**或**右側**區域。
    *   **重要觀念：** 這裡顯示的選單內容，是在「外觀」>「選單 (Menus)」中設定的。建置器只負責「放置」和「設計樣式」。
    *   點擊主要導覽元件的設定圖示，你可以進入「**設計 (Design)**」分頁，調整選單的字體顏色、滑鼠懸停 (Hover) 效果等。

    `[截圖：主要導覽的設計分頁，顯示顏色和字型設定選項]`

3.  **加入行動呼籲 (CTA) 按鈕：**
    *   一個好的頁首絕對不能少了 CTA 按鈕！
    *   將「**按鈕 (Button)**」元件拖曳到「**主要列**」的**最右側**。
    *   點擊按鈕的設定圖示，修改「**按鈕文字 (Button Text)**」（例如：立即諮詢、免費試用）。
    *   在「**連結 (Link)**」欄位中，填入目標頁面的網址。
    *   切換到「**設計 (Design)**」分頁，設定按鈕的背景顏色、文字顏色、邊框圓角等，讓它看起來更醒目。
    *   **Moksa 技巧：** CTA 按鈕的顏色最好使用與網站主色系**形成對比**的顏色，才能在第一時間抓住使用者的目光。

    `[截圖：按鈕的設計畫面，特別標示出背景顏色與圓角設定]`

4.  **發佈儲存！**
    *   完成電腦版設計後，點擊自訂器左上角的藍色「**發佈 (Publish)**」按鈕，儲存你的設定。

### 步驟 4：實作「手機版 (Mobile)」頁首

手機版的體驗至關重要。Kadence 讓這件事變得非常簡單。

1.  **切換到手機預覽：**
    *   點擊頁首建置器左下角的「**手機**」圖示。
    *   你會發現佈局變了，出現了一個名為「**觸發器 (Trigger)**」的元件，也就是我們俗稱的「漢堡選單圖示」。

    `[截圖：手機版頁首建置器畫面，顯示 Logo 在左邊，觸發器在右邊]`

2.  **設定行動版頁首：**
    *   在手機版模式下，你可以重新拖曳元件。通常的作法是：將「**Logo**」放在**左側**，「**觸發器**」放在**右側**。其他電腦版的元件（如主要導覽、按鈕）會自動隱藏。

3.  **設計彈出式選單 (Off-Canvas Menu)：**
    *   點擊「**觸發器**」元件的設定圖示。
    *   在左側的設定面板中，你會看到「**彈出式選單設計 (Off-Canvas Menu Design)**」的選項。
    *   在這裡，你可以設定使用者點擊漢堡圖示後，彈出的選單樣式。
    *   **Moksa 技巧：** 你可以將「**行動版導覽 (Mobile Navigation)**」、「**搜尋框**」、「**社群圖示**」甚至「**按鈕**」等元件拖曳到彈出式選單的專屬編輯區塊中，打造一個功能強大的行動版選單。
    *   記得在「**設計 (Design)**」分頁調整彈出選單的背景顏色、連結顏色等，確保它清晰易讀。

    `[截圖：彈出式選單的設定畫面，顯示可以拖曳元件的區塊以及設計選項]`

4.  **再次發佈儲存：**
    *   調整完畢後，再次點擊「**發佈 (Publish)**」。

### 總結與進階技巧

恭喜你！你已經成功使用 Kadence 頁首建置器，打造了一個在所有裝置上都表現優異的全站頁首。

**幾個可以讓你更上一層樓的進階技巧：**

*   **黏性頁首 (Sticky Header):** 在頁首的「**通用 (General)**」設定中，可以找到「**黏性頁首**」選項。啟用後，當使用者向下滾動頁面時，頁首會固定在畫面頂端，方便隨時點擊導覽。你可以選擇只固定「主要列」或整個頁首。
*   **透明頁首 (Transparent Header):** 如果你的首頁有很漂亮的全螢幕 Banner，可以啟用「**透明頁首**」功能（此為 Kadence Pro 功能），讓頁首疊加在圖片上方，創造出現代化的視覺效果。
