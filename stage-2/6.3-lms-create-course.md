# 06.03 (實作) 建立「課程」、「單元」、「測驗」

嗨，歡迎來到 MoksaWP 網站設計與進階客製化課程！

在之前的章節中，我們已經成功安裝並設定了線上課程系統 (LMS) 的核心外掛 LearnDash。現在，我們要開始動手建立線上課程的骨架了。一門結構清晰的課程，能大幅提升學員的學習體驗與完成率。

這個章節，我們將一步步帶你完成三個最重要的核心元素：**課程 (Courses)**、**單元 (Lessons)** 與 **測驗 (Quizzes)**。我們會學習如何建立它們，並將它們有邏輯地組織在一起。

---

### 核心觀念：LearnDash 的內容階層

在開始實作前，請務必先理解 LearnDash 的內容階層，這會讓你在後續操作時更有方向感：

1.  **課程 (Course):** 這是最上層的容器，代表一整門完整的課程。例如：「MoksaWP - 網站設計與進階客製化」。
2.  **單元 (Lesson):** 課程底下的主要章節或模組。例如：「第一章：WordPress 基礎概念」、「第二章：網站設計實戰」。一個課程可以包含數個單元。
3.  **主題 (Topic):** (選用) 單元底下更細分的子章節。例如：「1.1 什麼是 WordPress」、「1.2 安裝 WordPress」。一個單元可以包含數個主題。
4.  **測驗 (Quiz):** 用於評量學習成效的工具，可以被關聯到「課程」、「單元」或「主題」的結尾。

理解了嗎？我們的目標就是像蓋房子一樣，先蓋好「課程」這個主體結構，再放入「單元」這個房間，最後在房間裡放上「測驗」這個家具。

### Part 1：建立你的第一門「課程」

「課程」是所有學習內容的起點，讓我們開始建立它。

#### 步驟 1：新增課程

1.  從 WordPress 後台，前往左側選單的 **LearnDash LMS** > **課程 (Courses)**。
2.  點擊頁面上方的「**新增**」按鈕。
3.  在標題欄位，輸入你的「**課程標題**」。例如：「MoksaWP - 網站設計與進階客製化」。
4.  在下方的內容編輯區，你可以輸入這門課程的詳細介紹，包含課程目標、適合對象、講師介紹等。這裡的內容會顯示在課程主頁上，你可以使用 **Kadence Blocks** 來進行排版，讓頁面更吸引人。

`[SCREENSHOT]`
*示意圖：顯示正在編輯一門新課程，標題和內容編輯區已被填寫。*

#### 步驟 2：設定課程細節

在課程編輯頁面，你會看到下方有幾個重要的設定頁籤。

1.  點擊「**課程內容 (Builder)**」頁籤。這是 LearnDash 最強大的功能之一，它允許你用拖拉的方式來組織課程架構。現在這裡應該是空的，但請記住這個地方，我們稍後會回來使用它。

2.  點擊「**設定 (Settings)**」頁籤，這裡有幾個**必填**的關鍵設定：
    *   **課程存取設定 (Course Access Settings)** > **存取模式 (Access Mode)**：
        *   **開放 (Open):** 任何人都可以存取，無需登入。
        *   **免費 (Free):** 需要註冊並登入網站，但無需付費。
        *   **立即購買 (Buy Now):** 使用 LearnDash 內建的金流 (通常是 PayPal)。
        *   **定期扣款 (Recurring):** 同上，用於訂閱制課程。
        *   **關閉 (Closed):** **這是我們的首選**。課程無法直接被購買，存取權限必須透過手動新增、會員資格或與其他外掛（如 WooCommerce）整合來授予。

    *   **Moksa 技巧：** 為什麼我們要選擇「**關閉 (Closed)**」？因為在我們的技術棧中，**所有銷售和金流都由 WooCommerce 統一管理**。這麼做的好處是，你可以使用台灣主流的綠界金流、LINE Pay，還能搭配 Advanced Coupons 建立複雜的優惠活動，或整合 FluentCRM 進行再行銷。我們會將這門 LearnDash 課程與一個 WooCommerce 商品綁定，當學員購買該商品後，系統會自動將他們註冊進這門課程。

    *   **課程先決條件 (Course Prerequisites):** 如果你希望學員必須先完成另一門課才能學習這門課，可以在這裡設定。
    *   **課程證書 (Course Certificate):** 如果你有設計證書，可以在這裡指定學員完成課程後會獲得哪一張證書。

`[SCREENSHOT]`
*示意圖：顯示在「設定」頁籤中，將「存取模式」設定為「關閉 (Closed)」。*

#### 步驟 3：發布課程

完成設定後，點擊右上角的「**發布**」按鈕。

恭喜！你已經成功建立了課程的主體。雖然它現在還是個空殼，但我們已經打好了地基。

---

### Part 2：新增課程「單元」

接下來，我們要為課程添加實際的學習內容，也就是「單元」。

#### 步驟 1：建立新單元

1.  從 WordPress 後台，前往 **LearnDash LMS** > **單元 (Lessons)**。
2.  點擊「**新增**」按鈕。
3.  輸入你的「**單元標題**」，例如：「第一章：WordPress 基礎概念」。
4.  在內容編輯區，放入這個單元的主要教學內容。可以是文字、圖片，或是嵌入 YouTube/Vimeo 影片。

#### 步驟 2：設定單元與課程的關聯

這是**最重要的一個步驟**，如果沒做，你的單元將會是「孤兒」，學員也看不到它。

1.  在單元編輯頁面的右側邊欄，或下方的「**設定 (Settings)**」頁籤中，找到「**關聯課程 (Associated Course)**」這個選項。
2.  從下拉選單中，選擇我們剛剛建立的課程：「MoksaWP - 網站設計與進階客製化」。
3.  **單元存取設定 (Lesson Access Settings)** > **釋出排程 (Lesson Release Schedule)**：
    *   **立即 (Immediately):** 預設選項，學員一註冊即可看到此單元。
    *   **註冊後 X 天 (Enrollment-based):** 可設定在學員註冊課程後的第 N 天才能看到此單元。這就是所謂的「**滴灌式內容 (Drip-Feed Content)**」，可以引導學員按照你的步調學習。
    *   **特定日期 (Specific date):** 在某個特定日期之後，所有學員才能看到此單元。

`[SCREENSHOT]`
*示意圖：顯示在單元設定中，將「關聯課程」指向正確的課程。*

#### 步驟 3：使用「課程內容 (Builder)」來組織

雖然我們已經將單元關聯到課程，但**Moksa 技巧**是，我們更推薦直接使用「課程內容」編輯器來管理，這樣更直觀。

1.  回到 **LearnDash LMS** > **課程 (Courses)**，編輯我們的主課程。
2.  切換到「**課程內容 (Builder)**」頁籤。
3.  在左側的方框中，你會看到剛剛建立的「第一章：WordPress 基礎概念」單元。
4.  直接用滑鼠**拖曳**它到右邊的課程結構區域。
5.  你也可以直接在這裡點擊「**新增單元**」來快速建立新的單元，無需跳轉頁面。

`[SCREENSHOT]`
*示意圖：展示如何從左側將一個單元拖曳到右側的課程結構中。*

### Part 3：建立學習「測驗」

測驗是驗收學習成果、增加互動性的好方法。

#### 步驟 1：新增測驗

1.  從 WordPress 後台，前往 **LearnDash LMS** > **測驗 (Quizzes)**。
2.  點擊「**新增**」按鈕。
3.  輸入「**測驗標題**」，例如：「第一章隨堂測驗」。

#### 步驟 2：建立測驗問題

1.  在測驗編輯頁面，切換到「**課程內容 (Builder)**」頁籤。
2.  點擊「**新問題**」。
3.  在彈出的視窗中，輸入「**問題標題 (或題目)**」，例如：「WordPress 是一個什麼系統？」。
4.  點擊「**新增問題**」。
5.  你會看到問題已經被新增，點擊它右邊的下拉箭頭展開設定。
6.  在「**答案類型 (Answer Type)**」中，選擇題型。最常用的是「**單選 (Single Choice)**」和「**多選 (Multiple Choice)**」。
7.  在下方的「**答案 (Answers)**」區塊，點擊「**新增答案**」來建立選項。
8.  填寫選項內容，並記得勾選哪一個是「**正確 (Correct)**」答案。
9.  重複以上步驟，建立所有測驗題目。

`[SCREENSHOT]`
*示意圖：展示在測驗的 Builder 中新增一個單選題，並設定正確答案。*

#### 步驟 3：設定測驗細節與關聯

1.  切換到「**設定 (Settings)**」頁籤。
2.  **最重要的步驟來了**：找到「**關聯課程 (Associated Course)**」和「**關聯單元 (Associated Lesson)**」，將這個測驗指定給正確的課程與單元。這能確保測驗會出現在第一章單元的結尾。
3.  **測驗存取設定 (Quiz Access Settings)**：
    *   **及格分數 (Passing Score):** 設定學員需要答對多少百分比才能通過測驗，例如輸入 `80` 代表 80%。
    *   **重考次數限制 (Retake Restrictions):** 限制學員可以重考的次數。
    *   **時間限制 (Time Limit):** 設定測驗的作答時間。

`[SCREENSHOT]`
*示意圖：顯示在測驗設定中，將其關聯到指定的課程和單元。*

#### 步驟 4：將測驗放入課程結構

最後，讓我們回到課程的「**課程內容 (Builder)**」。

1.  前往 **LearnDash LMS** > **課程 (Courses)**，編輯主課程。
2.  切換到「**課程內容 (Builder)**」頁籤。
3.  從左邊的「測驗」列表中，找到我們剛剛建立的「第一章隨堂測驗」。
4.  將它拖曳到「第一章：WordPress 基礎概念」單元的下方，形成巢狀結構。

`[SCREENSHOT]`
*示意圖：展示課程 Builder 的最終樣貌，一個課程底下有一個單元，單元底下有一個測驗。*

### 總結

做得好！你已經完成了建立一門線上課程最核心的三個步驟。

我們來回顧一下今天的重點：
*   **課程 (Course):** 是所有內容的家，主要設定是存取模式，**Moksa 技巧**是使用「**關閉 (Closed)**」模式，以便與 WooCommerce 整合。
*   **單元 (Lesson):** 是課程的主要章節，**最重要的設定是將它「關聯」到正確的課程**。
*   **測驗 (Quiz):** 是評量工具，**最重要的設定是將它「關聯」到正確的課程與單元**。
*   **課程內容 (Builder):** 是你最好的朋友，所有課程架構的組織與排序，都應該在這裡完成。

現在，你的課程已經有了基本的架構。接下來，你可以繼續重複 Part 2 和 Part 3 的步驟，建立更多的單元與測驗，逐步豐富你的課程內容。在下一個章節，我們將會探討如何設定 WooCommerce 商品並將它與這門課程綁定，實現自動化的銷售與註冊流程。
