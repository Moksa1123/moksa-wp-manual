# 04.02 (實作) 打造「可重複使用」的區塊 (Reusable Blocks)

嗨，歡迎來到這個章節！我是 Moksa 的首席課程導師。

在經營網站的過程中，你一定會發現有些內容區塊是「一直重複出現」的。例如：每個頁面頁尾的聯絡資訊、文章結尾的「作者介紹」、或是吸引客戶報名課程的「行動呼籲 (Call-to-Action)」區塊。

如果每次都要手動複製貼上，不僅浪費時間，未來要修改時更是天大的災難——你必須一個一個頁面找出來修改。

為了解決這個問題，WordPress 內建了一個極其強大的功能，早期稱為「**可重複使用區塊 (Reusable Blocks)**」，在 WordPress 6.3 版本後，這個功能被整合並升級為「**同步版面配置 (Synced Patterns)**」。

它們的核心概念完全相同：**「一處修改，處處同步更新」**。

今天，我們就要來學習如何 masterful 地運用這個功能，大幅提升你的網站建置與管理效率！

---

### 什麼是「同步版面配置」(可重複使用區塊)？

你可以把它想像成一個「雲端樣板」。你先設計好一個區塊（或一組區塊），並將它存為「同步版面配置」。接著，你可以在網站的任何頁面、任何文章中，像插入圖片或標題一樣，輕鬆地把它插入進來。

最神奇的是，當你需要修改這個樣板的內容時（例如更新電話號碼），你只需要在**任何一個**地方編輯並儲存這個「同步版面配置」，那麼整個網站所有用到它的地方，都會**瞬間自動更新**！

**常見應用場景：**
*   **行動呼籲 (CTA)：** 放在文章或課程頁面結尾，引導使用者點擊報名或購買。
*   **聯絡資訊卡：** 包含公司地址、電話、Email、營業時間等。
*   **優惠活動橫幅：** 在特定期間，可以快速在多個頁面頂部或底部插入促銷訊息。
*   **作者簡介：** 放在所有部落格文章的結尾。
*   **社群媒體追蹤按鈕：** 一組設計好的 Facebook, Instagram, YouTube 圖示連結。

---

### (一) 建立你的第一個「同步版面配置」

讓我們以一個最常見的「課程諮詢 CTA」區塊為例，來實際操作一次。這個區塊將包含一個標題、一段文字和一個按鈕。

**步驟 1：設計你的區塊內容**

首先，在任何一個頁面或文章的編輯器中，開始設計你想要的內容。這裡我們將大量使用 **Moksa 技巧**，也就是活用 **Kadence Blocks** 來打造更精美的版面。

1.  新增一個 **Kadence** 的「**Row Layout**」區塊，並選擇單欄。
2.  在「Row Layout」中，新增一個 **Kadence** 的「**Advanced Heading**」區塊，輸入標題，例如：「對課程有疑問嗎？」
3.  接著新增一個「**Paragraph**」區塊，輸入說明文字，例如：「立即填寫表單，Moksa 專業顧問將親自與您聯繫，解開所有疑惑！」
4.  最後，新增一個 **Kadence** 的「**Advanced Button**」區塊，設定按鈕文字為「**立即預約諮詢**」，並填上你的諮詢頁面連結。
5.  你可以利用右側的設定面板，調整顏色、背景、間距，讓它看起來更吸引人。

``
*（上圖：使用 Kadence Blocks 設計完成的一個精美 CTA 區塊，包含標題、內文與按鈕，並已設定好背景顏色與邊框。）*

**步驟 2：選取並儲存為同步版面配置**

1.  設計完成後，點擊最外層的「**Row Layout**」區塊，確保整個區塊都被選取。（**Moksa 技巧：** 你可以點擊左上角的「清單檢視」來精準選取整個父層區塊）。
2.  點擊區塊工具列上的「**三個點**」選項圖示。
3.  在下拉選單中，點選「**建立版面配置/可重複使用區塊 (Create pattern/reusable block)**」。

``
*（上圖：在選取整個區塊後，點擊三個點的選單，並選擇「建立版面配置/可重複使用區塊」的選項。）*

**步驟 3：命名與設定**

1.  系統會跳出一個視窗，要求你為這個版面配置命名。請取一個**有意義、好辨識**的名稱，例如：「**通用課程諮詢 CTA**」。
2.  下方會有一個「**同步 (Synced)**」的開關，**請務必保持開啟**。這就是「一處修改，處處同步」的關鍵設定。
3.  你可以選擇性地將它歸類在特定的「**分類**」中，方便未來管理。
4.  點擊「**建立 (Create)**」。

``
*（上圖：為版面配置命名的彈出視窗，已填寫名稱並確保「同步」選項為開啟狀態。）*

恭喜你！你的第一個「同步版面配置」已經成功建立並儲存到網站的資料庫中了。

---

### (二) 如何插入與使用「同步版面配置」

建立完成後，要在其他頁面使用就非常簡單了。

**步驟 1：打開區塊插入器**

在你想要插入內容的地方，點擊「**+**」號按鈕，打開區塊插入器。

**步驟 2：切換至「版面配置」分頁**

在區塊插入器的頂部，點擊「**版面配置 (Patterns)**」分頁。

**步驟 3：找到並插入你的自訂版面配置**

1.  在「版面配置」分頁中，你可能會看到許多預設的版面，點擊「**我的版面配置 (My patterns)**」分類，就可以看到所有你自己建立的版面。
2.  找到我們剛剛建立的「**通用課程諮詢 CTA**」，直接點擊它。

``
*（上圖：在區塊插入器的「版面配置」分頁中，找到了「我的版面配置」分類下的「通用課程諮詢 CTA」並準備點擊插入。）*

**步驟 4：完成插入**

點擊後，整個設計好的區塊就會原封不動地出現在你的頁面中。你會發現它的外框是紫色的，這代表它是一個「同步版面配置」。

### (三) 管理與編輯「同步版面配置」

這才是這個功能最強大的地方。假設今天你的諮詢頁面連結換了，你需要更新所有 CTA 按鈕的連結。

**方法一：就地編輯 (最推薦)**

1.  在任何一個有插入這個 CTA 區塊的頁面，直接點擊該區塊。
2.  你會看到區塊上方出現一個提示，點擊「**編輯 (Edit)**」按鈕。
3.  接著，你就可以像編輯一般區塊一樣，去修改裡面的文字、連結或樣式。
4.  修改完成後，看一下編輯器右上角的藍色按鈕，它會變成「**儲存 (Save)**」。
5.  點擊「**儲存**」，系統會再次跟你確認是否要將變動儲存到這個同步版面配置中。確認後，網站上**所有**用到這個版面配置的地方就全部更新了！

``
*（上圖：點擊同步版面配置後，按下「編輯」按鈕，並在修改後，點擊右上角的「儲存」按鈕。）*

**方法二：從後台統一管理**

你可以到網站後台，點擊左側選單的「**外觀 (Appearance)**」>「**編輯器 (Editor)**」。
進入網站編輯器後，再點擊左側的「**版面配置 (Patterns)**」。

``
*（上圖：WordPress 後台的網站編輯器介面，左側選單已展開，並高亮顯示「版面配置」選項。）*

在這裡，你可以看到所有你建立的同步版面配置。點擊進入後，就可以進行編輯、重新命名、匯出或刪除等管理操作。

### (四) 進階 Moksa 技巧

**技巧 1：解除同步，變為一般區塊**

有時候，你可能只是想「借用」同步版面配置的設計當作範本，但希望這次的修改**不要**影響到其他地方。

1.  插入你的「同步版面配置」。
2.  選取該區塊，點擊工具列上的「**三個點**」圖示。
3.  選擇「**中斷連結版面配置 (Detach pattern)**」。
4.  這個區塊就會立刻變回一般的、獨立的區塊（紫色外框會消失），你對它的任何修改都只會影響當前頁面。

**技巧 2：跨站點匯出與匯入**

如果你管理多個網站，或是在本機開發環境 (例如 LocalWP) 設計好版面後要上傳到正式站，這個功能就非常實用。

1.  到「外觀 > 編輯器 > 版面配置」的管理頁面。
2.  找到你要匯出的版面配置，點擊它右邊的「**三個點**」。
3.  選擇「**匯出為 JSON (Export as JSON)**」，一個 .json 檔案就會被下載。
4.  到你的目標網站，同樣進入版面配置管理頁面，點擊上方的「**匯入**」按鈕，並上傳剛剛下載的 .json 檔即可。

這個技巧對於團隊協作和標準化設計流程非常有幫助！

---

### 總結

「同步版面配置」(可重複使用區塊) 是 WordPress 編輯器中一個革命性的功能。它能幫你節省大量的時間、確保網站內容風格的一致性，並讓未來的維護工作變得異常輕鬆。

現在，花點時間思考一下，你的網站上有哪些元素是重複出現的？試著將它們製作成你的第一個同步版面配置吧！這個小小的改變，將會為你的工作流程帶來巨大的提升。
