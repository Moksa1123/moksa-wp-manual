# 03.05 (實作) CSS 偽類 (Hover 效果)

嗨，歡迎來到這個章節！今天我們要來學習一個前端設計中非常有趣且實用的技巧：**CSS 偽類 (Pseudo-classes)**。

簡單來說，偽類可以讓你根據**使用者的特定行為或元素狀態**來改變 CSS 樣式。最常見的例子就是當滑鼠**懸停 (Hover)** 在一個按鈕或連結上時，改變它的顏色或大小，提供視覺回饋，讓網站感覺更「活」、更有互動性。

這堂課的目標，就是讓你學會如何使用 `:hover` 這個最核心的偽類，並將它應用在我們常用的頁面編輯器 (Kadence Blocks 與 Elementor) 中，為你的網站增添專業的動態效果。

---

### 什麼是 CSS 偽類？

在我們開始實作之前，先建立一個重要觀念。

一般的 CSS 選擇器是選擇「元素本身」，例如 `p` (選擇所有段落)、`.my-button` (選擇 class 為 my-button 的元素)。

而**偽類**則是在選擇器的基礎上，加上一個「**狀態**」，語法通常是 `選擇器:偽類`。

**基本語法結構：**

```css
選擇器:偽類 {
  屬性: 值;
}
```

**最常見的例子：**
這段程式碼的意思是「當滑鼠懸停在任何 `<a>` 標籤 (連結) 上時，將它的文字顏色變成橘色」。

```css
a:hover {
  color: #ff6600;
}
```

### ✨ Moksa 技巧：讓動態效果更流暢的 `transition`

如果只有 `:hover` 效果，樣式的變化會非常生硬、瞬間完成。要做出專業、滑順的動畫感，我們必須搭配 `transition` (轉場) 屬性。

`transition` 屬性要加在**元素原本的狀態**上，而不是 `:hover` 狀態上。它告訴瀏覽器：「當這個元素的任何屬性要發生變化時，請花 0.3 秒的時間，用 'ease' 的速度曲線來完成動畫。」

```css
/* 設定 a 標籤的原始狀態 */
a {
  color: #0073aa; /* 原始顏色是藍色 */
  transition: all 0.3s ease; /* 關鍵！告訴瀏覽器所有變動都套用 0.3 秒的轉場動畫 */
}

/* 設定 a 標籤在 hover 狀態時的樣式 */
a:hover {
  color: #ff6600; /* 滑鼠移上去時變成橘色 */
}
```

加上 `transition` 後，顏色變化就會從藍色平滑地過渡到橘色，而不是瞬間跳變。這是一個小細節，但對使用者體驗有巨大的提升。

### (實作一) 在 Kadence Blocks 中應用 Hover 效果

讓我們來實作一個常見的需求：當滑鼠移到 Kadence 的「進階按鈕 (Advanced Button)」上時，讓按鈕背景變色，並稍微放大。

#### 步驟 1：新增一個「進階按鈕」區塊

首先，在你的頁面中新增一個 Kadence 的「進階按鈕」區塊，並完成基本設定，例如按鈕文字、連結、以及**原始的**背景與文字顏色。

``

#### 步驟 2：找到自訂 CSS 的位置

1.  點選你剛剛新增的按鈕區塊。
2.  在右側的區塊設定欄中，切換到「**進階 (Advanced)**」分頁。
3.  往下捲動，找到「**自訂 CSS (Custom CSS)**」的欄位。

``

#### 步驟 3：撰寫 Hover 的 CSS 程式碼

Kadence Blocks 很貼心地提供了一個特殊的選擇器 `selector`，它會自動代表這個區塊本身。我們可以在這個基礎上撰寫我們的 CSS。

請在「自訂 CSS」欄位中，貼上以下程式碼：

```css
/* 設定按鈕的原始狀態，加入轉場效果 */
selector .kt-button {
  transition: all 0.3s ease-in-out;
}

/* 設定滑鼠懸停時的狀態 */
selector .kt-button:hover {
  background-color: #333333; /* 變成深灰色，你可以換成自己喜歡的顏色 */
  color: #ffffff; /* 文字變成白色 */
  transform: scale(1.05); /* 放大 5% */
}
```

*   **`selector .kt-button`**: `selector` 指的是這個「進階按鈕」區塊的外層容器，而 `.kt-button` 才是按鈕本身，所以我們這樣寫可以更精準地選到目標。
*   **`transform: scale(1.05);`**: 這是一個 CSS 的變形屬性，`scale(1.05)` 代表將元素放大到原本的 105%。

#### 步驟 4：預覽效果

完成後，直接在前台預覽你的頁面。現在當你的滑鼠移到按鈕上時，應該就能看到流暢的變色與放大效果了！

### (實作二) 在 Elementor 中應用 Hover 效果

Elementor 對於 Hover 效果提供了更視覺化的設定，但如果想做一些進階效果 (例如放大、旋轉)，還是需要用到自訂 CSS。

#### 方法一：使用內建的 Hover 設定 (最簡單)

對於單純的顏色變化，Elementor 的內建功能就非常足夠。

##### 步驟 1：新增一個「按鈕 (Button)」小工具

從左側小工具面板中，拖曳一個「按鈕」到你的頁面編輯區。

##### 步驟 2：切換到「樣式 (Style)」分頁

點選按鈕小工具，在左側設定面板中，切換到「**樣式**」分頁。

##### 步驟 3：設定 Hover 樣式

在樣式設定中，你會看到「**正常 (NORMAL)**」與「**滑鼠懸停 (HOVER)**」兩個選項。

1.  先在「正常」狀態下設定好你按鈕的原始樣式。
2.  點擊「**滑鼠懸停**」，這時候下方的設定項就會變成用來設定 Hover 狀態的樣式。你可以直接在這裡設定不同的**文字顏色**與**背景顏色**。
3.  你還可以在「**滑鼠懸停動畫 (Hover Animation)**」中選擇一個預設的動畫，例如 `Grow` (放大)。

``

#### 方法二：使用自訂 CSS (更彈性)

如果你使用的是 **Elementor Pro**，並且想要做出比內建動畫更細緻的效果 (例如同時變色、放大、並輕微旋轉)，就可以使用自訂 CSS。

##### 步驟 1：找到自訂 CSS 的位置

1.  點選你的按鈕小工具。
2.  在左側設定面板中，切換到「**進階 (Advanced)**」分頁。
3.  展開「**自訂 CSS (Custom CSS)**」區塊。

##### 步驟 2：撰寫 CSS 程式碼

與 Kadence 類似，Elementor 也提供 `selector` 這個關鍵字來代表目前的小工具。

```css
/* 在按鈕的「樣式」>「滑鼠懸停」>「轉場持續時間」中設定 0.3 秒 */
/* 這樣可以讓 Elementor 自動處理 transition 屬性 */

/* 接著在這裡只寫 hover 的特殊效果 */
selector .elementor-button:hover {
  transform: scale(1.05) rotate(-2deg); /* 放大 5% 並逆時針旋轉 2 度 */
}
```

**Moksa 技巧：** 在 Elementor 中，我們可以直接利用它內建的「**轉場持續時間 (Transition Duration)**」欄位（位於「樣式」>「滑鼠懸停」）來設定動畫時間，這樣就不需要在 Custom CSS 裡重複撰寫 `transition` 屬性了，讓程式碼更乾淨。

``

### 補充：其他實用的偽類

`:hover` 只是眾多偽類中的一種，以下再簡單介紹幾個你未來可能會用到的：

*   **`:active`**: 元素被「啟用」或「點擊」的瞬間。常用來做按鈕按下去時的瞬間下沉或變色效果。
*   **`:focus`**: 當元素取得焦點時。最常用在表單的 `input` 欄位，當使用者點擊輸入框準備打字時，可以讓輸入框的邊框變色，提示使用者目前正在編輯此欄位。
*   **`:visited`**: 只適用於連結 (`<a>` 標籤)，代表這個連結已經被使用者點擊瀏覽過了。瀏覽器預設會將點過的連結變成紫色。
*   **`:nth-child(n)`**: 一個更進階的用法，可以用來選擇「第 n 個」子元素。例如 `li:nth-child(odd)` 可以選到所有奇數項的 `<li>` 列表項目，常用來製作斑馬紋表格或列表。
