# 03.01 CSS 核心觀念 (Selector, Box Model, Flexbox)

嗨，歡迎來到網站設計與進階客製化的第一個核心技術章節。我是 Moksa 的首席導師。

在前面的課程中，我們已經學會了如何使用 Kadence Blocks 或 Elementor 等頁面編輯器來「拖拉」出一個網站的樣貌。但你可能會發現，有時候編輯器提供的選項就是差那麼一點點，無法達到你心目中 100% 的設計。

這時候，我們就需要學會網站的「化妝術」—— **CSS (Cascading Style Sheets)**。

把網站想像成一個人：
*   **HTML** 是骨架，決定了有哪些部位 (標題、段落、圖片)。
*   **CSS** 則是外觀與穿搭，決定了骨架的樣貌 (顏色、大小、位置、排版)。

學會 CSS，就像是獲得了微調網站外觀的超能力。你將不再受限於編輯器的設定，可以隨心所欲地打造獨一無二的視覺風格。這堂課，我們將專注於 CSS 中最重要、最基礎的三大核心觀念，它們是你未來所有進階客製化的基石。

### 觀念一：選取器 (Selector) - 如何精準地選中你想改變的對象？

CSS 的運作原理很簡單：「**選中一個東西，然後告訴瀏覽器要改變它的什麼樣式**」。而「選取器 (Selector)」就是用來執行「選中一個東西」這個動作的語法。

想像一下，你要告訴設計師：「請把那張海報的標題變成紅色。」在這個指令中，「那張海報的標題」就是選取器。在 CSS 中，我們有幾種常見的選取方式：

1.  **標籤選取器 (Tag Selector)**
    *   直接使用 HTML 標籤的名稱來選取。這個方法會選取到頁面上「所有」同類型的標籤，適合用來做全站的基礎設定。
    *   **語法**：`標籤名 { 屬性: 值; }`
    *   **範例**：讓網站上**所有**的段落 `p` 文字都變成灰色。
        ```css
        p {
          color: #555555;
        }
        ```

2.  **Class 選取器 (Class Selector)**
    *   這是**最常用、最靈活**的選取器。你可以為 HTML 元素指定一個或多個「類別 (class)」，然後針對這個類別來設定樣式。
    *   **Moksa 技巧：** Class 的概念就像是幫人貼標籤，例如「VIP客戶」、「潛在客戶」。你可以對所有貼上「VIP客戶」標籤的人提供特殊優惠，這就是 Class 的精神。
    *   **語法**：`.你的class名稱 { 屬性: 值; }` (注意，class 名稱前面要有一個點 `.`)
    *   **範例**：假設你在 Kadence Blocks 中為一個按鈕新增了一個額外的 CSS Class 叫做 `highlight-button`，現在你想讓所有帶有這個 class 的按鈕都有橘色背景。
        ```css
        .highlight-button {
          background-color: orange;
          color: white; /* 同時也讓文字變白色，比較好看 */
        }
        ```

3.  **ID 選取器 (ID Selector)**
    *   ID 就像是人的「身分證字號」，在**一個頁面中，每一個 ID 都必須是獨一無二的**。它用來選取頁面上某個特定的、唯一的元素。
    *   **語法**：`#你的ID名稱 { 屬性: 值; }` (注意，ID 名稱前面要有一個井號 `#`)
    *   **範例**：你想特別針對網站頁尾的聯絡我們區塊 `#contact-us` 進行樣式調整。
        ```css
        #contact-us {
          border-top: 2px solid #eeeeee;
          padding-top: 40px;
        }
        ```

**Moksa 技巧：如何找到我想要的選取器？**

這是實務上最重要的技能。我們必須使用瀏覽器的「**開發者工具 (Developer Tools)**」來檢查元素。

*   **步驟 1：** 在你想修改的網頁元素上，點擊滑鼠右鍵，選擇「**檢查 (Inspect)**」。
    ``
*   **步驟 2：** 畫面會分割，出現一個充滿程式碼的視窗。這就是開發者工具。你可以看到 HTML 結構。
    ``
*   **步驟 3：** 在開發者工具的左上角，點擊「**選取元素圖示**」(通常是一個箭頭指向方框的圖示)。
*   **步驟 4：** 將滑鼠移回網頁上，點擊你想修改的元素。此時，右方的 HTML 結構會自動跳到該元素的程式碼位置。
*   **步驟 5：** 觀察該行程式碼，你就能清楚地看到它的**標籤**、**class** 或 **id** 是什麼，然後你就可以用這些資訊來撰寫你的 CSS 選取器了！
    ``

### 觀念二：盒子模型 (Box Model) - 每個網頁元素都是一個盒子

在 CSS 的世界裡，**所有在網頁上你看得到的東西，都是一個長方形的盒子**。 चाहे वो是一段文字、一張圖片，或是一個按鈕。理解這個「盒子」是如何構成的，對於調整間距、邊框至關重要。

這個盒子由內而外，包含四個部分：

``

1.  **內容 (Content)**
    *   盒子的最內層，也就是你實際看到的文字或圖片。它的尺寸由 `width` 和 `height` 屬性決定。

2.  **內距 (Padding)**
    *   **內容**與**邊框**之間的距離。想像成是盒子裡的填充物，用來保護內容，讓內容不要太貼近邊框。Padding 的區域會顯示元素的背景顏色。

3.  **邊框 (Border)**
    *   包圍在**內距**外層的線條。你可以設定它的粗細、樣式（實線、虛線等）和顏色。

4.  **外距 (Margin)**
    *   **邊框**以外的空間，完全透明。它用來推開其他的盒子，創造元素與元素之間的距離。

**範例：**
假設我們有一個 `div` 元素，我們想讓它有更清楚的邊界和內部空間。

```css
.my-box {
  background-color: #f0f8ff; /* 淺藍色背景，方便觀察 */
  width: 300px;
  
  padding: 20px; /* 內容上下左右各有 20px 的內距 */
  border: 1px solid #4682b4; /* 1px 粗的實心藍色邊框 */
  margin: 30px; /* 盒子上下左右會推開其他元素 30px 的距離 */
}
```
**Moksa 技巧：** 當你發現兩個元素黏得太近，你應該思考：我是要在盒子「內部」增加空間 (**Padding**)，還是要讓兩個盒子「互相推開」(**Margin**)? 這是初學者最常搞混的地方。一個簡單的判斷法：如果改變的間距會影響到背景色的範圍，那通常是 Padding；如果只是元素間的空白，那通常是 Margin。

### 觀念三：Flexbox - 現代網頁排版的瑞士刀

過去，要讓元素水平排列、垂直置中，是一件非常痛苦的事情。但現在，我們有了 **Flexbox (彈性盒子佈局)**，它徹底改變了網頁排版的方式。

**Moksa 技巧：** 你在 Kadence Blocks 或 Elementor 裡面使用的「橫向佈局 (Row Layout)」、「欄 (Column)」、「容器 (Container)」等功能，其底層技術幾乎都是基於 Flexbox 實現的。學會 Flexbox 的核心觀念，能幫助你在編輯器卡關時，用幾行 CSS 就解決排版問題。

Flexbox 的核心觀念只有兩個：

1.  **Flex 容器 (Flex Container)**: 一個外層的盒子。
2.  **Flex 項目 (Flex Items)**: 容器裡面的所有子層盒子。

你只需要在「外層容器」身上下一道指令 `display: flex;`，它就會瞬間變成一個 Flex 容器，而它裡面的所有項目就會開始聽從 Flexbox 的排版規則。

``

以下是幾個最常用的 Flexbox 屬性 (都是加在**容器**上)：

*   `display: flex;`
    *   **必備起手式**。啟動 Flexbox 佈局。

*   `flex-direction:`
    *   決定項目要「**水平排列**」還是「**垂直排列**」。
    *   `row`: 水平排列 (預設值)。
    *   `column`: 垂直排列。

*   `justify-content:`
    *   決定項目在「**主軸**」上的對齊方式 (如果是 `flex-direction: row`，主軸就是水平方向)。
    *   `flex-start`: 靠左對齊 (預設值)。
    *   `flex-end`: 靠右對齊。
    *   `center`: 水平置中。
    *   `space-between`: 左右兩邊貼齊，中間的項目平均分配剩餘空間。
    *   `space-around`: 每個項目的左右都環繞著均等的空間。

*   `align-items:`
    *   決定項目在「**交錯軸**」上的對齊方式 (如果是 `flex-direction: row`，交錯軸就是垂直方向)。
    *   `flex-start`: 靠上對齊。
    *   `flex-end`: 靠下對齊。
    *   `center`: **這就是傳說中的垂直置中！**
    *   `stretch`: 將項目拉伸，填滿整個容器高度 (預設值)。

**範例：**
想像我們有三個盒子，放在一個名為 `.container` 的大盒子裡。我們希望這三個盒子能夠水平排列、垂直置中、並且均分所有空間。

```html
<!-- HTML 結構 -->
<div class="container">
  <div class="item">盒子1</div>
  <div class="item">盒子2</div>
  <div class="item">盒子3</div>
</div>
```

```css
/* CSS 樣式 */
.container {
  display: flex; /* 步驟 1: 啟動 Flexbox */
  justify-content: space-between; /* 步驟 2: 水平均分 */
  align-items: center; /* 步驟 3: 垂直置中 */
  
  height: 200px; /* 為了看出垂直置中的效果，給容器一個高度 */
  background-color: #eee; /* 給容器一個背景色 */
}
```
只需要這幾行簡單的 CSS，就能實現過去需要複雜技巧才能達成的排版效果。

### 總結

恭喜你！你已經掌握了 CSS 中最關鍵的三塊基石：

1.  **Selector (選取器):** 讓你學會如何精準地「瞄準」目標。
2.  **Box Model (盒子模型):** 讓你理解每個元素的「佔位」和「間距」是如何計算的。
3.  **Flexbox (彈性盒子):** 讓你擁有強大而靈活的「排版」能力。
