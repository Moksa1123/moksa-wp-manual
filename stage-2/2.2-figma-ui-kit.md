# 02.02 (實務) 如何在 Figma 建立「品牌 UI Kit」

嗨，歡迎來到 Moksa 的網站設計與進階客製化課程！我是你的導師。

在我們開始用 Kadence 或 Elementor 打造漂亮的頁面之前，有一個至關重要的前置步驟，許多新手會忽略，但所有專業團隊都會執行，那就是——**建立一個「品牌 UI Kit」**。

把它想像成是你的網站專屬的樂高積木盒。當你需要蓋房子時，你不會從零開始製作每一塊磚頭，而是會從盒子裡拿出預先做好的標準積木。UI Kit 就是這個概念，它包含了你網站所有的基礎設計元素：顏色、字體、按鈕、表單等等。

這堂課，我們將一步步教你如何在免費又強大的設計工具 Figma 中，為你的品牌建立一個專業的 UI Kit。這不僅能讓你的網站視覺風格高度統一，更能在未來的開發與維護中，省下數十甚至數百個小時。

---

### 為什麼「品牌 UI Kit」如此重要？

*   **一致性 (Consistency):** 確保網站上每一個按鈕、每一段文字，都遵循相同的視覺規範，提升品牌專業感。
*   **高效率 (Efficiency):** 設計師與工程師（在這裡，可能都是你！）可以直接取用定義好的元件，無需重複設計或猜測樣式。**Moksa 技巧：** 這將會是你設定 Kadence Blocks 全域樣式的「**唯一參考標準**」。
*   **易於維護 (Maintainability):** 當品牌需要更新主色調時，你只需要在 UI Kit 中修改一個顏色，所有相關的元件就會自動更新，一勞永逸。

---

### UI Kit 的核心組成元素

一個基礎但完整的網站 UI Kit 至少需要包含以下幾個部分：

1.  **色彩 (Colors):** 品牌主色、輔色、中性色（灰階）、系統狀態色（成功、警告、錯誤）。
2.  **字體排版 (Typography):** 從大標題 (H1) 到小字備註 (Caption) 的所有文字樣式。
3.  **間距 (Spacing):** 定義元件與元件之間的標準距離，建立視覺上的和諧感。
4.  **網格系統 (Grid System):** 網站內容佈局的基礎框架。
5.  **元件 (Components):**
    *   按鈕 (Buttons)
    *   表單輸入框 (Input Fields)
    *   下拉選單 (Dropdowns)
    *   核取方塊 (Checkboxes) & 單選按鈕 (Radio Buttons)

---

### (實作) 開始建立你的 UI Kit

首先，請確保你已經註冊並登入 [Figma](https://www.figma.com/)。

#### 步驟 1：建立新檔案與頁面 (Pages)

1.  在 Figma 中，點擊右上角的 `+ Design file` 來建立一個新的設計稿。
2.  將檔案重新命名，例如：「Moksa 品牌 UI Kit」。
3.  在左側的圖層面板，你會看到一個預設的 `Page 1`。一個好的習慣是將不同類型的設計元素分門別類。點擊 `+` 號新增頁面，建立以下結構：
    *   `_Cover` (用於放置檔案封面)
    *   `1. Foundations` (用於放置顏色、字體、網格等基礎)
    *   `2. Components` (用於放置按鈕、表單等元件)

``

#### 步驟 2：定義色彩系統 (Color System)

我們將在 `1. Foundations` 這個頁面進行操作。

1.  使用矩形工具 (快捷鍵 `R`) 在畫布上畫出幾個方塊。
2.  選取第一個方塊，在右側的屬性面板中，找到 **Fill** (填色) 區塊，點擊顏色方塊，然後輸入你的**品牌主色**的 HEX 色碼 (例如 `#2563EB`)。
3.  **Moksa 技巧：** 建立「顏色樣式 (Color Styles)」是整個 UI Kit 的核心。點擊 **Fill** 區塊右邊的四個點圖示 (Style)，然後點擊 `+` 號。
4.  在 **Create new color style** 視窗中，為你的顏色命名。建議使用「分類 / 名稱」的結構，例如：`Primary / Blue-500`。
5.  重複以上步驟，建立你的完整調色盤。一個專業的調色盤通常包含：
    *   **Primary:** 品牌主色 (e.g., `Primary / Blue-500`)
    *   **Secondary:** 品牌輔色 (e.g., `Secondary / Green-500`)
    *   **Neutral:** 灰階，用於文字、背景、邊框 (e.g., `Neutral / Gray-900`, `Neutral / Gray-500`, `Neutral / Gray-100`)
    *   **System:** 系統回饋色
        *   `Success / Green-500` (成功)
        *   `Warning / Yellow-500` (警告)
        *   `Error / Red-500` (錯誤)

建立完成後，你的右側面板應該會顯示所有已儲存的顏色樣式。未來，你只需要點擊物件，然後從這個樣式庫選取顏色即可，無需再手動輸入色碼。

``

#### 步驟 3：定義字體排版 (Typography)

1.  使用文字工具 (快捷鍵 `T`) 在畫布上打出一些範例文字，例如 "Heading 1", "Body Text"。
2.  選取 "Heading 1" 文字，在右側的 **Text** 屬性面板中，設定好它的**字體 (Font Family)**、**字重 (Font Weight)**、**大小 (Font Size)**、**行高 (Line Height)**。
3.  與顏色一樣，點擊 **Text** 區塊右邊的四個點圖示 (Style)，點擊 `+` 號。
4.  命名你的文字樣式，例如 `Heading / H1`。
5.  重複以上步驟，為你的網站建立完整的文字階層。**Moksa 技巧：** 你的文字樣式應該要能直接對應到 WordPress 中的元素：
    *   `Heading / H1` (對應 `<h1>`)
    *   `Heading / H2` (對應 `<h2>`)
    *   `Heading / H3` (對應 `<h3>`)
    *   `Heading / H4` (對應 `<h4>`)
    *   `Body / Large`
    *   `Body / Regular` (對應 `<p>`)
    *   `Link / Regular` (對應 `<a>`)
    *   `Caption` (用於註釋或小字)

``

#### 步驟 4：建立第一個元件：按鈕 (Button)

現在，我們前往 `2. Components` 頁面，來製作最常用的元件——按鈕。

1.  使用文字工具 (T) 打出 "Button Text"。
2.  選取文字，在右側 **Text** 屬性面板，套用你剛剛建立的文字樣式 (例如 `Body / Large`)。
3.  在選取文字的狀態下，按下快捷鍵 `Shift + A`，Figma 會自動為它包上一層 **Auto Layout** 框架。這是製作響應式元件的關鍵。
4.  在右側的 **Auto Layout** 面板中，設定按鈕的 `padding` (上下、左右內距)，例如上下 `12px`，左右 `24px`。
5.  在右側的 **Fill** 面板中，點擊 `+` 號，並套用你的品牌主色樣式 (例如 `Primary / Blue-500`)。
6.  將文字顏色也從樣式庫中改為白色 (例如 `Neutral / White`)。
7.  在 **Stroke** (邊框) 面板，設定圓角 (Corner Radius)，例如 `8px`。
8.  **Moksa 技巧：** 為了方便管理，我們要將這個按鈕轉化為「**元件 (Component)**」。選取你做好的按鈕，點擊畫面頂部中央的菱形圖示 (Create Component)，或使用快捷鍵 `Ctrl + Alt + K` (Windows) / `Cmd + Option + K` (Mac)。

現在，你已經有了一個可重複使用的「主元件」。你可以從左側的 `Assets` 面板把它拖曳出來使用，任何對主元件的修改，都會同步到所有分身 (Instance) 上。

``

### Moksa 技巧：如何將 UI Kit 對應到 Kadence Blocks

這就是建立 UI Kit 最有價值的地方！當你開始用 Kadence Blocks 建站時，可以直接將 Figma 中的定義無縫轉移。

1.  **設定全域色彩 (Global Palette):**
    *   在 WordPress 後台，進入 `外觀` > `自訂` > `全域` > `色彩`。
    *   你會看到 Kadence 的調色盤。點擊每一個顏色。
    *   回到你的 Figma UI Kit，選取對應的顏色樣式，複製它的 **HEX 色碼**。
    *   將色碼貼到 Kadence 的調色盤中。
    *   **務必**將你在 Figma 中定義的 `Primary`, `Secondary`, `Neutral` 等顏色，一一對應到 Kadence 的調色盤中。這樣一來，Kadence 全站的按鈕、連結顏色都會自動統一。

``

2.  **設定全域字體 (Typography):**
    *   在 WordPress 後台，進入 `外觀` > `自訂` > `全域` > `字體排版`。
    *   你會看到 `標題字型 (H1, H2...)` 和 `內文字型` 的設定。
    *   回到你的 Figma UI Kit，查看你為 `Heading / H1` 設定的字體、字重、大小等數值。
    *   將這些數值，一模一樣地填入 Kadence 的 H1 設定中。
    *   重複這個過程，完成 H1 到 H6 以及內文 (Body) 的所有設定。

完成以上兩個步驟後，你的 WordPress 網站的基礎視覺就已經和你的 Figma 設計稿 100% 同步了。未來你在使用任何 Kadence Block 時，它都會預設繼承這些全域樣式，達到真正的一致與高效。
