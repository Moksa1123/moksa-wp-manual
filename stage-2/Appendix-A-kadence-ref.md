# 附錄 A：Kadence Blocks 完整區塊參數表

你好，歡迎來到 MoksaWP 網站設計與進階客製化課程的附錄章節。

本附錄的目標是為你提供一份常用 **Kadence Blocks** 區塊的「**速查表**」。當你在使用 Kadence Blocks 進行頁面排版與設計時，時常會需要調整區塊右側邊欄的各種參數。這份文件整理了我們在實務上最常用到的核心區塊，並解釋了其重要參數的功能與用途。

請將本附錄當作你的字典或參考手冊，在需要時隨時查閱，這將大幅加速你的開發與設計流程。我們不會條列出「所有」的參數，而是專注在那些**最關鍵、最常用**的設定上。

---

## 1. Row Layout (列佈局)

**Row Layout** 是 Kadence Blocks 的**基石**，幾乎所有複雜的頁面佈局都是從它開始的。它讓你能夠建立響應式的欄 (Column) 結構。

``

*   **用途：** 建立頁面的主要水平結構，並在其中劃分出多個垂直的欄位來放置內容。例如：兩欄式圖文介紹、三欄式特色列表等。

*   **重要參數 (右側邊欄設定):**
    *   **一般 (General) 頁籤：**
        *   **Columns Layout:** 預設的欄位佈局，可以快速選擇 1-6 欄以及它們的寬度比例。
        *   **Column Gutter:** 設定**欄與欄之間的間距**。
        *   **Vertical Align:** 設定欄內所有內容的**垂直對齊方式** (靠上、置中、靠下)。
        *   **Inherit Max Width from Theme:** **Moksa 技巧：** 建議**保持啟用**。這會讓 Row Layout 的最大寬度與佈景主題設定 (外觀 > 自訂 > 一般 > 佈局) 保持一致，確保全站視覺風格統一。
        *   **Content Max Width:** 若取消繼承佈景主題寬度，可在此自訂該 Row Layout 的內容最大寬度。

    *   **樣式 (Style) 頁籤：**
        *   **Background:** 設定整個 Row Layout 的背景，可以是**單色、漸層色、圖片或影片**。圖片背景有視差 (Parallax) 效果可選。
        *   **Background Overlay:** 在背景之上再疊加一層顏色，常用於讓背景圖片變暗，以凸顯前景的文字。
        *   **Border:** 設定外框線的樣式、寬度、顏色與圓角 (Border Radius)。
        *   **Shape Dividers:** 在 Row Layout 的**頂部或底部**加入有趣的SVG形狀分隔線，增加設計感。

    *   **進階 (Advanced) 頁籤：**
        *   **Padding/Margin:** **核心觀念**。**Padding (內距)** 是區塊**內部邊緣與內容之間的距離**；**Margin (外距)** 是區塊**外部與其他區塊之間的距離**。Kadence 強大的地方在於可以分別設定電腦、平板、手機的內外距，是實現 RWD 響應式設計的關鍵。
        *   **Visibility Settings:** 控制此區塊在**電腦、平板、手機上是否顯示**。常用於針對不同裝置製作專屬的排版。
        *   **HTML Anchor:** 設定一個錨點 ID，可以讓其他地方的連結直接跳轉到這個區塊。

## 2. Advanced Heading (進階標題)

比 WordPress 內建標題更強大的標題區塊，提供完整的 SEO 與設計控制。

*   **用途：** 建立頁面中的各級標題 (H1, H2, H3...)，並進行細緻的視覺設計。

*   **重要參數:**
    *   **一般 (General) 頁籤：**
        *   **Alignment:** 文字對齊方式。
        *   **HTML Tag:** **SEO 關鍵**。設定這個標題是 H1 到 H6 的哪一個層級，或是只是一個普通的 `div` 或 `p` 段落。請務必遵循頁面的語意結構。
        *   **Link Settings:** 為標題加上超連結。

    *   **樣式 (Style) 頁籤：**
        *   **Color Settings:** 設定文字顏色。
        *   **Highlight:** 為標題中的**特定文字**加上底線、背景色等醒目提示，非常適合用來強調關鍵字。
        *   **Typography:** 控制字體大小、行高、字重 (粗細)、字母間距等。
        *   **Text Shadow:** 為文字加上陰影效果。

    *   **進階 (Advanced) 頁籤：**
        *   **Padding/Margin:** 同上，控制標題與其他元素的間距。
        *   **Spacing Settings:** 控制標題本身的上下間距。

---

## 3. Advanced Button (進階按鈕)

功能完整的按鈕區塊，適用於各種「行動呼籲 (Call to Action, CTA)」。

*   **用途：** 建立引導使用者點擊的按鈕，例如「立即購買」、「了解更多」、「聯絡我們」。

*   **重要參數:**
    *   **一般 (General) 頁籤：**
        *   **Button Count:** 可在一個區塊中建立多個按鈕，並設定它們的排列方式。
        *   **Button Text:** 按鈕上顯示的文字。
        *   **Link:** 按鈕要連結到的網址。
        *   **Icon Settings:** 在按鈕文字旁加入圖示 (Icon)，並可設定圖示位置 (左/右)。

    *   **樣式 (Style) 頁籤：**
        *   **Typography:** 設定按鈕文字的字體樣式。
        *   **Text Color:** 設定文字顏色 (**一般狀態**與**滑鼠懸停 (Hover) 狀態**)。
        *   **Background:** 設定背景顏色 (**一般狀態**與**滑鼠懸停 (Hover) 狀態**)。
        *   **Border:** 設定框線樣式 (**一般狀態**與**滑鼠懸停 (Hover) 狀態**)。
        *   **Shadow & Opacity:** 為按鈕加上陰影，製造立體感。

    *   **進階 (Advanced) 頁籤：**
        *   **Padding/Margin:** 調整按鈕大小 (透過 Padding) 與外部間距 (透過 Margin)。

---

## 4. Info Box (資訊框)

一個多功能的複合式區塊，整合了**圖示/圖片、標題、內文和連結**，非常實用。

``

*   **用途：** 快速建立「產品特色」、「服務項目」、「公司優勢」等列表。

*   **重要參數:**
    *   **一般 (General) 頁籤：**
        *   **Media Settings:** 選擇頂部要顯示的是**圖示 (Icon)、圖片 (Image) 還是數字 (Number)**。
        *   **Media Align:** 設定媒體 (圖示/圖片) 的位置 (左、中、右)。
        *   **Title Settings:** 設定標題的文字與 HTML 標籤。
        *   **Learn More Settings:** 設定底部的「了解更多」連結文字與網址。

    *   **樣式 (Style) 頁籤：**
        *   **Media Style:** 控制圖示/圖片的顏色、大小、背景、邊框等。
        *   **Title/Text Style:** 分別控制標題與內文的顏色與字體樣式。
        *   **Container Style:** 控制整個 Info Box 區塊的背景、邊框與陰影。**Moksa 技巧：** 透過設定 Container 的 Hover 陰影，可以做出滑鼠移上去時，卡片浮起來的互動效果。

---

## 5. Accordion (手風琴/摺疊面板)

常用於將大量內容收合，讓使用者可以點擊標題來展開或收合內容。

*   **用途：** FAQ (常見問答)、產品規格說明、課程章節列表等。

*   **重要參數:**
    *   **一般 (General) 頁籤：**
        *   **Panes:** 管理每一個摺疊面板的標題與內容。
        *   **Initial Collapsed:** 載入頁面時，是否預設將所有面板收合。
        *   **Link to Anchor ID:** 允許透過網址直接展開並跳轉到指定的面板。

    *   **樣式 (Style) 頁籤：**
        *   **Pane Title Styles:** 設定標題的背景、顏色、字體 (可分別設定**一般、滑鼠懸停、啟用中**的狀態)。
        *   **Content Styling:** 設定展開後內容區域的背景、顏色與內距。

---

## 6. Tabs (頁籤)

將內容分組到不同的頁籤中，讓使用者點擊切換，節省頁面空間。

*   **用途：** 產品介紹 (商品描述、規格、評價)、多步驟說明、服務方案比較。

*   **重要參數:**
    *   **一般 (General) 頁籤：**
        *   **Tab Count & Titles:** 管理頁籤的數量與標題。
        *   **Initial Tab:** 載入頁面時，預設顯示第幾個頁籤。

    *   **樣式 (Style) 頁籤：**
        *   **Title Styles:** 設定頁籤標題的背景、顏色、字體 (可分別設定**一般、滑鼠懸停、啟用中**的狀態)。
        *   **Content Styling:** 設定頁籤內容區域的背景、顏色與內距。

## 總結

以上是我們在 Moksa 實務課程中，認為最核心且必須熟練的幾個 Kadence Blocks 區塊。掌握它們的參數設定，你就能夠組合出絕大多數的商業網站頁面設計。

**Moksa 技巧：** 學習 Kadence Blocks 最快的方式，就是**動手實作**。打開一個空白頁面，將這些區塊一個個拉進來，試著去調整每一個你看到的參數，觀察頁面上發生的即時變化。當你將這些參數與視覺變化連結起來後，未來在設計頁面時，就能夠更直覺地知道該去哪裡調整了。
