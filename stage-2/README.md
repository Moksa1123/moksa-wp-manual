# MoksaWP 第二階段課程：網站設計與進階客製化

哈囉！歡迎來到 MoksaWP 的第二階段課程：「網站設計與進階客製化」。

首先，恭喜你完成了第一階段的課程！你現在已經掌握了 WordPress 與 WooCommerce 的核心操作，能夠獨立建立一個功能完整的電子商務網站。這是一個非常了不起的成就！

在課程一，我們專注於「從 0 到 1」，讓你學會如何使用現成的強大工具，快速打造出商業網站的骨架。

而在這 40 小時的進階課程中，我們將帶你探索「從 1 到 100」的無限可能。我們將會深入 WordPress 的核心，學習如何不再受限於外掛或佈景主題的功能，真正打造出獨一無二、符合特定商業邏輯、兼具美感與效能的客製化網站。

準備好從一位「網站操作者」蛻變為一位「網站設計師」與「網站開發者」了嗎？讓我們開始這趟精彩的旅程吧！

## 這門課，你將會學到什麼？

這門課的核心目標是讓你掌握「解決複雜問題」與「實現獨特設計」的能力。我們將圍繞以下幾個核心領域展開：

### 視覺化編輯器的極致應用

*   深入探索 Kadence Blocks 與 Elementor 的進階功能，學習如何打造複雜的頁面佈局、動態內容，以及實現像素級精準的響應式設計 (RWD)。
*   **Moksa 技巧：** 我們會教你如何建立可重複使用的「設計系統」，大幅提升你未來的網站建置效率。

### 客製化內容結構 (CPT & ACF)

*   這將是本課程的精髓！你將學會使用 Custom Post Types (CPT) 來建立 WordPress 原生沒有的內容類型，例如：「課程」、「作品集」、「團隊成員」或「實體店面」。
*   搭配 Advanced Custom Fields (ACF)，你將可以為任何內容類型（包含商品）新增客製化欄位，例如：為商品加上「成分表」、為課程加上「開課日期」等。這將徹底解放你對網站內容的管理能力。

### 進階 WooCommerce 客製化

*   我們將超越基本設定，學習如何客製化商品頁面、購物車與結帳頁面，以提升轉換率。
*   你將學會如何結合 ACF 與頁面編輯器，打造獨一無二的商品展示方式。

### 自動化行銷與 CRM 整合

*   網站不只是門面，更是自動化行銷的利器。我們將深入 FluentCRM，學習設定複雜的自動化流程，例如：顧客標籤分群、購買後自動感謝信與教學、棄單召回 Email 系列。
*   結合 OrderChatz，實現更進階的 LINE CRM 應用，例如：依據顧客購買的商品，自動推播不同的 LINE 訊息。

### 開發者工具與基礎程式碼

*   我們將引導你進入開發者的世界，學習使用 LocalWP 建立安全的本機開發環境。
*   你將會接觸到 `functions.php` 檔案、Code Snippets 外掛，並學會如何使用 WordPress 的 Hooks (Action & Filter) 來進行輕度的程式碼客製化，不再需要為了小功能而安裝一堆外掛。
*   我們會介紹 Query Monitor 等除錯工具，讓你學會如何分析網站的效能瓶頸。

## 課程章節規劃

為了讓你循序漸進地學習，我們將課程規劃為以下幾個主要階段：

### 第一部分：設計深水區

本部分將從網站的基礎建置開始，深入探索視覺化編輯器的高級應用，並學習如何將設計稿精確地轉換為具備響應式設計的網頁。

*   **1.1 網站基礎建置與頁面編輯器總覽**
    *   01.01 課程介紹 (從設計師到網站實作者)
    *   01.02 本地端開發環境建置 (LocalWP)
    *   01.03 頁面編輯器 (Page Builder) 總覽 (Elementor vs Kadence)
    *   01.04 高效能佈景主題的選擇 (Astra, Kadence, Blocksy)
*   **1.2 Figma 設計稿轉 WordPress 實作**
    *   02.01 UI/UX 設計稿基礎 (Grid, Spacing, Typography)
    *   02.02 (實務) 如何在 Figma 建立「品牌 UI Kit」
    *   02.03 (實作) 將 UI Kit 導入 WP (全域色彩、字體)
    *   02.04 (實作) 拆解 Figma 設計稿 (Layout, Section, Block)
    *   02.05 (實作) 使用 Kadence Blocks 實作 Figma 靜態頁面
    *   02.06 (實務) RWD 響應式設計 (平板/手機)
*   **1.3 Kadence Blocks 進階功能與全站打造**
    *   04.01 Kadence Blocks 核心區塊詳解
    *   04.02 (實作) 打造「可重複使用」的區塊 (Reusable Blocks)
    *   04.03 (實作) Kadence Elements (動態模板) 介紹
    *   04.04 (實作) 打造「全站頁首 (Header)」
    *   04.05 (實作) 打造「全站頁尾 (Footer)」
    *   04.06 (實務) 頁面編輯器的效能陷阱與優化
*   **1.4 CSS 核心觀念與前端設計實作**
    *   03.01 CSS 核心觀念 (Selector, Box Model, Flexbox)
    *   03.02 (實務) 如何在 WordPress 中安全地加入 CSS (客製化器、子主題)
    *   03.05 (實作) CSS 偽類 (Hover 效果)
    *   03.06 (實作) CSS 轉場 (Transition) 與動畫 (Animation)

本部分是課程的精髓，將帶您突破 WordPress 內容管理的限制，實現高度客製化的內容結構。

*   什麼是 CPT (Custom Post Type)？
*   實作：建立你的第一個 CPT
*   ACF (Advanced Custom Fields) 欄位全解析
*   實作：將 ACF 應用於文章、頁面與商品

### 第三部分：打造高轉換率的電商體驗

在掌握了基礎的 WooCommerce 設定後，本部分將引導您深入客製化，打造更具吸引力與高轉換率的電子商務流程。

*   **3.1 WooCommerce 電子商務實作與客製化**

    *   05.05 (實務) 提升轉換率的結帳頁面優化 (Checkout Field Editor)

### 第四部分：自動化行銷流程設計

本部分將教您如何將網站轉化為強大的自動化行銷工具，有效提升顧客關係管理與銷售效益。
