### 在這門課，你將會學到什麼？

這門課的核心目標是讓你掌握「**解決複雜問題**」與「**實現獨特設計**」的能力。我們將圍繞以下幾個核心領域展開：

1.  **視覺化編輯器的極致應用：**
    *   深入探索 **Kadence Blocks** 與 **Elementor** 的進階功能，學習如何打造複雜的頁面佈局、動態內容，以及實現像素級精準的響應式設計 (RWD)。
    *   **Moksa 技巧：** 我們會教你如何建立可重複使用的「設計系統」，大幅提升你未來的網站建置效率。

2.  **客製化內容結構 (CPT & ACF)：**
    *   這將是本課程的精髓！你將學會使用 **Custom Post Types (CPT)** 來建立 WordPress 原生沒有的內容類型，例如：「課程」、「作品集」、「團隊成員」或「實體店面」。
    *   搭配 **Advanced Custom Fields (ACF)**，你將可以為任何內容類型（包含商品）新增客製化欄位，例如：為商品加上「成分表」、為課程加上「開課日期」等。這將徹底解放你對網站內容的管理能力。

3.  **進階 WooCommerce 客製化：**
    *   我們將超越基本設定，學習如何客製化商品頁面、購物車與結帳頁面，以提升轉換率。
    *   你將學會如何結合 **ACF** 與頁面編輯器，打造獨一無二的商品展示方式。

4.  **自動化行銷與 CRM 整合：**
    *   網站不只是門面，更是自動化行銷的利器。我們將深入 **FluentCRM**，學習設定複雜的自動化流程，例如：顧客標籤分群、購買後自動感謝信與教學、棄單召回 Email 系列。
    *   結合 **OrderChatz**，實現更進階的 LINE CRM 應用，例如：依據顧客購買的商品，自動推播不同的 LINE 訊息。

5.  **開發者工具與基礎程式碼：**
    *   我們將引導你進入開發者的世界，學習使用 **LocalWP** 建立安全的本機開發環境。
    *   你將會接觸到 `functions.php` 檔案、Code Snippets 外掛，並學會如何使用 WordPress 的 Hooks (Action & Filter) 來進行輕度的程式碼客製化，不再需要為了小功能而安裝一堆外掛。
    *   我們會介紹 **Query Monitor** 等除錯工具，讓你學會如何分析網站的效能瓶頸。

### 課程章節規劃

為了讓你循序漸進地學習，我們將課程規劃為以下幾個主要階段：

*   **第一部分：設計深水區**
    *   深入 Kadence Blocks 與 Elementor
    *   響應式設計原理與實作
    *   動態內容與模板設計

*   **第二部分：解放內容的枷鎖 - CPT & ACF**
    *   什麼是 CPT (Custom Post Type)？
    *   實作：建立你的第一個 CPT
    *   ACF (Advanced Custom Fields) 欄位全解析
    *   實作：將 ACF 應用於文章、頁面與商品

*   **第三部分：打造高轉換率的電商體驗**
    *   客製化 WooCommerce 商品頁面版型
    *   優化結帳流程與欄位
    *   WooCommerce Email 模板客製化

*   **第四部分：自動化行銷流程設計**
    *   FluentCRM 進階：標籤、名單與自動化流程
    *   實作：設計一個完整的「新客歡迎系列」Email

*   **第五部分：開發者入門與網站維護**
    *   設定 LocalWP 本機環境
    *   WordPress Hooks 核心觀念
    *   CSS 與 PHP 程式碼片段入門
    *   進階網站效能優化與除錯技巧

*   **第六部分：畢業專案**
    *   整合所有學習到的技巧，從零開始打造一個高度客製化的網站。
