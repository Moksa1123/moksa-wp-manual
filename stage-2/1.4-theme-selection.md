# 01.04 高效能佈景主題的選擇 (Astra, Kadence, Blocksy)

嗨，歡迎來到 Moksa 的課程！我是你的導師。

在我們正式開始設計網站外觀之前，我們必須先做出一個至關重要的決定：選擇一個好的「**基底**」，也就是我們的「**佈景主題 (Theme)**」。

過去，許多佈景主題（特別是在 ThemeForest 這類市集上）喜歡把所有功能都包山包海地塞進去，導致網站變得非常臃腫、緩慢。這不僅傷害了**使用者體驗 (UX)**，更會嚴重影響你的 **SEO 分數**。

幸運的是，時代已經改變了。現在的主流是選擇一個**輕量化**、**高效能**、且**與頁面編輯器（如 Kadence Blocks 或 Elementor）完美整合**的「基礎型」佈景主題。它們就像房子的鋼筋骨架，穩固、可靠，但把裝潢的彈性完全交給你。

今天，我們將會深入探討目前市場上最受歡迎、也是我們 Moksa 團隊最推薦的三款高效能佈景主題：**Astra**、**Kadence** 與 **Blocksy**。

---

### 選擇佈景主題的四大核心標準

在比較這三者之前，讓我們先建立一個評估的標準。一個優秀的現代佈景主題，應該具備以下特質：

1.  **效能 (Performance):** 這是最重要的指標。佈景主題本身應該非常輕巧，載入速度快，HTTP 請求數少。這直接關係到 Google 的 Core Web Vitals 分數。
2.  **相容性 (Compatibility):** 必須與主流的外掛，特別是 WooCommerce 和頁面編輯器（我們課程核心是 **Kadence Blocks**）有良好的整合度。
3.  **客製化彈性 (Customization):** 應該提供強大且直覺的「自訂」選項（通常在 WordPress 後台的 `外觀` > `自訂` 中），讓我們可以輕鬆調整全域的顏色、字體、頁首 (Header)、頁尾 (Footer) 等。
4.  **支援與社群 (Support & Community):** 當你遇到問題時，是否有活躍的社群或官方提供支援，這對於長期經營網站來說非常重要。

---

### 三大高效能佈景主題深度解析

現在，讓我們來逐一分析這三位主角。

#### 1. Astra

Astra 是目前 WordPress 生態系中安裝量最大的非官方佈景主題，擁有超過百萬的活躍用戶。它的成功在於極致的輕量化與龐大的範本庫。

*   **優點:**
    *   **極致輕量:** 預設安裝的檔案大小小於 50KB，速度飛快。
    *   **龐大的範本庫 (Starter Templates):** Astra 提供上百個預製好的網站範本，可以一鍵匯入，對於想快速上線的初學者來說非常方便。
    *   **高度相容性:** 與 Elementor、Beaver Builder 等主流頁面編輯器整合度非常高。

*   **缺點:**
    *   **免費版功能較陽春:** 許多進階的頁首/頁尾、版面配置選項都需要升級到 Pro 版。
    *   **Pro 版功能分散:** Pro 版是以「模組」方式提供，有時會讓使用者感到功能選項有點分散。

*   **Moksa 團隊點評:**
    Astra 是一個非常優秀的選擇，特別是如果你打算使用 Elementor 並且希望有大量的現成範本可以套用。它的穩定性與社群基礎無庸置疑。

#### 2. Blocksy

Blocksy 雖然是後起之秀，但憑藉其強大且創新的功能，迅速累積了大量的忠實用戶。它在「免費版」中提供了許多其他佈景主題需要付費才有的功能。

*   **優點:**
    *   **最強大的頁首/頁尾編輯器:** Blocksy 的拖放式頁首/頁尾編輯器功能非常強大且直覺，即使在免費版也幾乎沒有限制。
    *   **免費版功能豐富:** 提供了進階的色彩系統、字體選項、文章版面配置等，CP 值非常高。
    *   **原生整合 Gutenberg:** 與 WordPress 內建的區塊編輯器（Gutenberg）整合得很好。

*   **缺點:**
    *   **社群規模較小:** 相較於 Astra，它的用戶基數和社群規模還在成長中。
    *   **進階 WooCommerce 功能需付費:** 雖然整合度不錯，但像「商品快速瀏覽」、「願望清單」等進階電商功能需要 Pro 版。

*   **Moksa 團隊點評:**
    如果你預算有限，但又希望擁有最大的客製化彈性，Blocksy 絕對是首選。它的免費版幾乎可以媲美其他佈景主題的付費版，特別是在頁首/頁尾的設計上。

#### 3. Kadence (Moksa 官方推薦)

Kadence 是我們 Moksa 團隊在絕大多數專案中，以及本課程的**官方指定佈景主題**。原因無他，就是它與我們教學核心 **Kadence Blocks** 頁面編輯器達成了「**人劍合一**」的境界。

*   **優點:**
    *   **與 Kadence Blocks 完美整合:** 兩者由同一個團隊開發，整合度天衣無縫。你可以在佈景主題「自訂」中設定好全域的**色彩調色盤**和**字體**，Kadence Blocks 會自動繼承這些設定，讓你的網站設計風格高度一致，管理起來非常方便。
    *   **功能強大且直覺:** 同樣擁有強大的頁首/頁尾編輯器，並且在「自訂」選項中提供了非常細緻的調整功能（例如按鈕樣式、捲動至頂端按鈕、社群連結等）。
    *   **效能卓越:** 效能表現與 Astra、Blocksy 在同一個頂尖水平。

*   **缺點:**
    *   **免費範本庫較少:** 相較於 Astra，Kadence 的免費入門範本數量較少。

*   **Moksa 團隊點評:**
    **這就是我們的選擇**。當你使用 Kadence Theme 搭配 Kadence Blocks 時，整個網站的設計流程會變得極度高效且流暢。從全域設定到單一頁面區塊的微調，體驗是一致的。這種整合性帶來的開發效率提升，是其他組合難以比擬的。**Moksa 技巧：這個「全域色彩與字體連動」的功能，是我們選擇 Kadence 的核心關鍵，它能為你未來省下數十個小時的設計與維護時間。**

### (實作) 安裝我們的選擇：Kadence 佈景主題

理論說完了，我們來動手安裝吧！

*   **步驟 1:** 登入你的 WordPress 後台。
*   **步驟 2:** 在左側選單，找到 `外觀` > `佈景主題`，並點擊它。
    ``
*   **步驟 3:** 點擊頁面上方的 `安裝佈景主題` 按鈕。
    ``
*   **步驟 4:** 在右方的搜尋框中，輸入 `Kadence`。
    ``
*   **步驟 5:** 你會看到 Kadence 佈景主題出現。點擊 `安裝` 按鈕，稍等一下讓它安裝完成。
    ``
*   **步驟 6:** 安裝完成後，原來的按鈕會變成 `啟用`。勇敢地給它按下去！
    ``

恭喜你！你已經為你的網站打下了最穩固、最高效能的基礎。在接下來的課程中，我們將會以 Kadence Theme 為基礎，搭配 Kadence Blocks，一步步打造出專業、美觀且速度飛快的電商網站。
