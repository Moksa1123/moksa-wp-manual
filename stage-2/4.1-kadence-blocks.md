# 04.01 Kadence Blocks 核心區塊詳解

嗨，歡迎來到 MoksaWP 網站設計與進階客製化課程！我是你的導師。

在上一個章節，我們已經了解了 WordPress 區塊編輯器 (Gutenberg) 的基本概念。從這個章節開始，我們將深入探討 Moksa 團隊最推薦的區塊增強外掛——**Kadence Blocks**。它不僅免費、輕量、功能強大，更是打造高效能與高質感網站的關鍵。

今天的課程目標，是帶你認識並熟練操作 Kadence Blocks 中最常用、最重要的幾個「**核心區塊**」。掌握了它們，你就掌握了 80% 的頁面排版能力。

---

### 為什麼選擇 Kadence Blocks？

在開始實作前，你可能會好奇，為什麼我們不用 Elementor 或其他頁面編輯器，而是選擇以 Gutenberg 為基礎的 Kadence Blocks？

*   **效能優先：** Kadence Blocks 產生的程式碼非常乾淨，這代表著更快的網站載入速度，對於使用者體驗和 **Rank Math SEO** 的評分都有正面影響。
*   **與核心無縫整合：** 它完全基於 WordPress 原生的編輯器，未來的相容性與穩定性最高。
*   **直覺且強大：** 它在原生編輯器的基礎上，提供了極為豐富的設計選項，從間距、顏色、背景到響應式設計，幾乎無所不能。
*   **Moksa 技巧：** 我們的技術棧（如 Kadence Theme, WP Rocket）與 Kadence Blocks 搭配使用，可以達到最佳的效能與設計平衡。

## 核心區塊深度解析

接下來，我們將逐一拆解幾個構成所有頁面設計基礎的核心區塊。我們會以蓋房子來比喻：

1.  **列佈局 (Row Layout):** 房子的鋼筋骨架，決定了整體的結構與隔間。
2.  **進階標題 (Advanced Heading):** 每個房間的名稱標示，引導訪客。
3.  **進階文字 (Advanced Text):** 房間內的詳細說明與內容。
4.  **進階按鈕 (Advanced Button):** 連接到其他房間或外部的門。

### 1. 列佈局 (Row Layout) - 網站結構的基石

**這是你最需要掌握的區塊，沒有之一。** 任何複雜的排版，都是從 Row Layout 開始的。它就像一個容器，你可以在裡面放置任何其他的區塊。

#### 如何新增與設定？

**步驟 1：新增區塊**
在編輯器中點擊 `+`，搜尋「Row Layout」並點擊新增。

``
*截圖：顯示在編輯器中搜尋並新增 Row Layout 區塊的畫面。*

**步驟 2：選擇欄位 (Columns)**
新增後，它會請你選擇初始的欄位佈局，例如：單欄、50/50 雙欄、三欄等。你可以隨時在右側的設定欄中調整。

``
*截圖：顯示選擇初始欄位佈局的選項，例如 1欄、2欄、3欄。*

**步驟 3：認識右側設定面板**
點擊 Row Layout 區塊後，右側的設定面板會出現三個頁籤：**一般 (General)**, **樣式 (Style)**, **進階 (Advanced)**。

``
*截圖：顯示 Row Layout 右側的設定面板，並標示出三個頁籤。*

#### 常用重要設定：

*   **一般 (General) > Layout:**
    *   **Columns:** 調整欄位數量。
    *   **Column Gutter:** 調整**欄位之間**的間距。
    *   **Align Middle:** 控制欄位內容的垂直置中，非常實用。

*   **進階 (Advanced) > Padding/Margin:**
    *   **Padding (內距):** 區塊**內部**的留白空間。
    *   **Margin (外距):** 區塊**外部**的留白空間。
    *   **Moksa 技巧：** **優先使用 Row Layout 的 Padding 來製造區塊的上下空間感**，而不是用很多空白區塊。這樣在手機版 (響應式) 的調整上會更方便、更統一。你可以點擊旁邊的「裝置」圖示，分別設定桌面、平板、手機的內距。

*   **樣式 (Style) > Background Settings:**
    *   你可以在這裡設定整個 Row Layout 的背景顏色、背景圖片或漸層色。
    *   **Background Overlay:** 可以在背景圖片上疊加一個半透明的顏色圖層，讓前景的文字更清晰，這是製作質感 Banner 的必備技巧。

### 2. 進階標題 (Advanced Heading) - 抓住目光的重點

純文字是枯燥的，你需要用標題來劃分段落、強調重點。

#### 常用重要設定：

*   **一般 (General) > Text Alignment & HTML Tag:**
    *   **Text Alignment:** 設定文字對齊方式（左、中、右）。
    *   **HTML Tag:** **這是 SEO 的關鍵！** 請依據頁面結構選擇 H1, H2, H3... 等。
    *   **Moksa 技巧：** 一個頁面**只應該有一個 H1 標籤**，通常是頁面的主標題。這對 **Rank Math SEO** 的評分至關重要。其他的副標題則依序使用 H2, H3。

*   **樣式 (Style):**
    *   **Color Settings:** 設定標題顏色。
    *   **Font Sizing:** 分別設定桌面、平板、手機的字體大小。
    *   **Highlight Settings:** 這是一個超棒的功能！你可以為標題中的特定文字加上底線、背景色等醒目提示。

``
*截圖：展示使用 Highlight Settings 製作出的特殊標題效果。*

### 3. 進階文字 (Advanced Text) - 內容的核心載體

這個區塊就是你放置主要段落內容的地方。它的設定與「進階標題」類似，但更專注於內文的排版。

#### 常用重要設定：

*   **一般 (General) > Text Alignment:** 設定段落對齊。
*   **樣式 (Style) > Font Sizing & Line Height:**
    *   **Font Sizing:** 設定內文的字體大小。
    *   **Line Height (行高):** 設定文字行與行之間的距離。**適當的行高（建議 1.6 至 1.8）可以大幅提升文章的易讀性。**

### 4. 進階按鈕 (Advanced Button) - 引導訪客採取行動 (CTA)

按鈕是網站中最重要的互動元素，引導使用者前往購買頁、聯絡頁或了解更多。一個好的按鈕設計能有效提升轉換率。

#### 常用重要設定：

*   **一般 (General):**
    *   **Button Text:** **必填欄位**，輸入按鈕上顯示的文字，例如「立即購買」。
    *   **Link:** **必填欄位**，輸入點擊按鈕後要前往的網址。
    *   **Button Size:** 快速設定按鈕大小 (SM, MD, LG)。
    *   **Icon Settings:** 可以在文字旁加上一個小圖示，增加視覺吸引力。

*   **樣式 (Style):**
    *   **Color Settings:** 這是最重要的設定！你可以分別設定**一般狀態 (Normal)** 和**滑鼠懸停 (Hover)** 時的文字顏色與背景顏色。
    *   **Moksa 技巧：** **Hover 效果是提升使用者體驗的關鍵**。一個簡單的顏色變化就能讓使用者明確知道這是一個可以點擊的元素。
    *   **Border Settings:** 設定按鈕的邊框樣式與圓角 (Border Radius)。

## 實作練習：打造一個英雄區塊 (Hero Section)

現在，讓我們結合以上所學，動手打造一個網站首頁最常見的「英雄區塊」。

**步驟 1：建立結構**
*   新增一個 **Row Layout** 區塊，選擇**單欄**。

**步驟 2：設定背景**
*   點選該 Row Layout，在右側的 **樣式 (Style) > Background Settings** 中，上傳一張高品質的背景圖片。
*   在 **Background Overlay** 中，設定一個黑色、透明度 50% 的疊加層，讓背景變暗。

**步驟 3：設定空間感**
*   在 **進階 (Advanced) > Padding/Margin** 中，設定 **Padding (內距)** 的上方 (Top) 和下方 (Bottom) 各為 `100px`。這樣可以讓區塊內容不會緊貼著上下邊緣。

**步驟 4：新增內容**
*   在 Row Layout 內，點擊 `+`，依序新增：
    1.  一個 **進階標題 (Advanced Heading)**，內容輸入「MoksaWP 企業方案」，HTML Tag 設為 **H1**，顏色設為白色，置中對齊。
    2.  一個 **進階文字 (Advanced Text)**，內容輸入一段簡介文字，顏色設為淺灰色，置中對齊。
    3.  一個 **進階按鈕 (Advanced Button)**，文字輸入「了解更多方案」，並設定一個你喜歡的顏色與 Hover 效果。將按鈕置中對齊。

**步驟 5：檢查響應式設計**
*   點擊編輯器左下角的裝置預覽圖示，切換到「平板」和「手機」模式。
*   你可能會發現 `100px` 的內距在手機上太大了。點選 Row Layout，回到 Padding 設定，點擊「手機」圖示，將上下的 Padding 修改為 `50px`。
