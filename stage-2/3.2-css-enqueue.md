# 03.02 (實務) 如何在 WordPress 中安全地加入 CSS (客製化器、子主題)

嗨，歡迎來到這個章節！在設計網站時，我們經常會遇到一個情況：佈景主題或頁面編輯器提供了 95% 我們想要的功能與樣式，但總有那麼 5% 的小細節想要微調，例如「把這個按鈕的圓角改得更圓一點」、「讓這段文字的行距再寬一些」或是「在手機版上隱藏某個元素」。

這些微調的魔法，就是透過 **CSS (Cascading Style Sheets，層疊樣式表)** 來實現的。CSS 是網頁的「化妝師」，專門用來定義網頁元素的視覺表現。

然而，**「如何」以及「在哪裡」加入這些 CSS 碼，是一門非常重要的學問**。錯誤的方式可能會導致你在更新佈景主題後，所有辛苦寫的客製化樣式全部消失，那絕對是每個站長的惡夢。

在這個單元，我們將會以 Moksa 的專業標準，教你三種最安全、最推薦的 CSS 加入方式，並解釋它們各自的適用情境。

---

### 方法一：使用 WordPress 內建的「附加 CSS」功能 (最簡單)

這是最快、最簡單，也最適合初學者或只需要少量微調的方法。WordPress 核心就內建了這個方便的功能。

*   **適用情境：** 只需要加入幾行到幾十行的 CSS 程式碼，進行全站性的小範圍樣式調整。
*   **優點：** 操作直覺、可即時預覽、不需碰觸任何檔案。
*   **缺點：** 如果 CSS 碼過多會變得難以管理；此處的 CSS **跟隨佈景主題**，若更換佈景主題，這些 CSS 碼不會跟著過去。

#### 操作步驟

**步驟 1：進入客製化器**

從 WordPress 後台儀表板，點擊左側選單的「**外觀**」>「**自訂**」。

`[截圖：WordPress 後台儀表板，滑鼠指標停在「外觀」>「自訂」上] `

**步驟 2：找到「附加 CSS」選項**

進入「自訂」模式後，你的網站會顯示在右側，左側則是設定選單。請在左側選單中找到並點擊「**附加 CSS (Additional CSS)**」選項。

`[截圖：WordPress 客製化器介面，左側選單中「附加 CSS」選項被圈選] `

**步驟 3：貼上你的 CSS 程式碼**

點擊後，你會看到一個可以輸入程式碼的文字區域。直接將你想要加入的 CSS 程式碼貼到這裡。

**Moksa 技巧：** 右側的畫面會**即時預覽**你加入的 CSS 效果，非常方便你確認樣式是否正確。

**範例：** 假設我們想把網站中所有 H2 標題的顏色，都改成 Moksa 的品牌紫色。

```css
/* 將所有 H2 標題改為品牌紫色 */
h2 {
  color: #5B31E6; 
}
```

`[截圖：在「附加 CSS」文字區塊中貼上範例 CSS 碼，右側預覽畫面的 H2 標題顏色已改變] `

**步驟 4：發佈變更**

確認樣式無誤後，點擊最上方的藍色「**發佈**」按鈕，你的變更就會正式儲存並生效了。

### 方法二：建立並使用「子主題 (Child Theme)」(最專業)

這是 **Moksa 團隊強烈推薦** 的專業做法，也是業界的標準作業流程 (SOP)。當你需要加入大量 CSS，或者未來有可能需要修改佈景主題的版型檔案 (PHP templates) 時，使用子主題是唯一正確的選擇。

*   **適用情境：** 需要加入大量、有組織的 CSS 程式碼；未來有客製化版型檔案的需求；追求網站的長期穩定性與可維護性。
*   **優點：** **網站客製化內容與佈景主題本身完全分離**。這代表你可以隨時安全地更新你的主佈景主題 (父主題)，而**不用擔心任何客製化內容會被覆蓋或遺失**。
*   **缺點：** 首次設定比方法一稍稍複雜。

#### 什麼是子主題？

你可以把「父主題」(例如 Kadence) 想像成一棟房子的原始設計藍圖。而「子主題」就像是在這張藍圖上加蓋的一層透明描圖紙。你所有的客製化修改 (無論是 CSS 樣式或功能) 都畫在這張描圖紙上。

當房子的原始設計藍圖 (父主題) 更新時（例如建商說要升級水管線路），你只需要換掉底下的藍圖，而你在描圖紙上的所有客製化設計都還在，完全不受影響。

#### 操作步驟

**步驟 1：建立並啟用子主題**

手動建立子主題需要建立資料夾與 `style.css`、`functions.php` 兩個檔案，對於初學者來說有些門檻。

**Moksa 技巧：** 我們推薦使用外掛來快速生成子主題。你可以安裝「**Child Theme Configurator**」這個外掛，跟隨它的引導步驟，為你當前使用的佈景主題（例如 Kadence）產生一個子主題。許多優質的佈景主題 (如 Kadence Pro) 也會直接提供子主題的下載連結，更加方便。

`[截圖：安裝「Child Theme Configurator」外掛的畫面，或是 Kadence 官網提供子主題下載的畫面] `

生成後，到「**外觀**」>「**佈景主題**」，你會看到一個名為「Kadence Child」或類似名稱的新主題。請直接「**啟用**」這個子主題。

`[截圖：佈景主題頁面，顯示了 Kadence 父主題和 Kadence Child 子主題，且子主題已被啟用] `

**步驟 2：編輯子主題的 style.css 檔案**

啟用子主題後，我們就可以把 CSS 加到子主題的樣式表中了。

1.  前往「**外觀**」>「**佈景主題檔案編輯器**」。
2.  系統可能會跳出一個警告，提醒你直接編輯檔案的風險，請理解後點擊「我瞭解了」。
3.  在右上角的「**選取要編輯的佈景主題**」下拉選單中，**務必確認你選擇的是你的「子主題」** (例如：Kadence Child)，而不是父主題！
4.  在右側的「佈景主題檔案」列表中，點擊 `style.css` (樣式表)。
5.  將你的 CSS 程式碼貼在檔案的最下方。
6.  完成後，點擊「**更新檔案**」。

`[截圖：佈景主題檔案編輯器介面，已選取子主題，並在 style.css 中加入 CSS 碼] `

**Moksa 技巧 (進階)：** 雖然內建的編輯器很方便，但在生產環境的網站上，我們更推薦透過 **FTP** 或主機的 **cPanel 檔案管理員**來編輯 `style.css` 檔案。這樣做更安全，萬一不小心打錯字導致網站出錯，也更容易還原。若使用 **LocalWP** 在本機開發，則可以直接用 **VS Code** 等編輯器開啟檔案，這是最有效率的做法。

### 方法三：在頁面編輯器中加入 CSS (針對性強)

當你的客製化樣式**只需要應用在特定頁面或特定元素**上時，直接在頁面編輯器 (如 Kadence Blocks 或 Elementor) 中加入 CSS 是一個很棒的選擇。

*   **適用情境：** 樣式只針對單一頁面、單一區塊 (Section) 或單一元素 (Block/Widget) 生效。
*   **優點：** 針對性強、管理方便 (樣式跟著元素走)、不污染全域 CSS。
*   **缺點：** 如果同樣的樣式需要在很多頁面重複使用，會變得沒有效率。

#### 以 Kadence Blocks 為例

Kadence Blocks Pro 允許你在每一個獨立的區塊 (Block) 中加入客製化的 CSS。

**步驟 1：選取要套用樣式的區塊**

在編輯頁面時，點選你想要客製化的那個區塊，例如一個「進階按鈕 (Advanced Button)」。

**步驟 2：找到 Custom CSS 設定**

在右側的區塊設定欄中，切換到「**進階 (Advanced)**」分頁，然後展開「**Custom CSS**」面板。

`[截圖：Kadence Blocks 的區塊設定畫面，顯示「進階」分頁下的「Custom CSS」面板] `

**步驟 3：撰寫 CSS**

在 Custom CSS 的輸入框中，你可以直接撰寫 CSS。

**Moksa 技巧：** Kadence Blocks 很貼心地使用 `selector` 這個關鍵字來代表你當前選取的這個區塊。這讓你可以很精準地只對這個區塊本身進行樣式修改，而不會影響到網站上其他的按鈕。

**範例：** 讓這個按鈕在滑鼠移上去時，產生一個輕微放大的動畫效果。

```css
selector:hover {
  transform: scale(1.05);
  transition: transform 0.2s ease-in-out;
}
```

`[截圖：在 Kadence Blocks 的 Custom CSS 面板中輸入範例程式碼] `

這個 CSS 只會對你選取的這「一個」按鈕生效，非常乾淨俐落！

### 總結與 Moksa 推薦流程

到底該用哪一種方法？我們為你整理了一個簡單的決策樹：

1.  **我只需要改幾個全站通用的樣式 (例如連結顏色、標題大小)，而且程式碼不超過 30 行。**
    *   **答案：** 使用**方法一：附加 CSS**。最快最簡單。

2.  **我需要針對某個特定頁面的某個特定按鈕或圖片做樣式調整。**
    *   **答案：** 使用**方法三：頁面編輯器的 Custom CSS**。最精準。

3.  **我預計會加入大量的 CSS 來客製化網站外觀，或者我是一個有長遠規劃的站長，希望網站架構清晰、易於維護。**
    *   **答案：** **務必使用方法二：子主題**。這是最專業、最能保障你心血結晶的做法，也是 Moksa 為所有客戶建置網站的標準流程。

學習如何「安全地」加入 CSS，是從新手邁向專業玩家的關鍵一步。它能確保你的網站在不斷成長和更新的過程中，依然保持穩定與健壯。現在，就試著用這些方法來為你的網站做些獨一無二的微調吧！
