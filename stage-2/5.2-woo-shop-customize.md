# 05.02 (實作) 客製化「商店頁面」 (篩選器、排版)

嗨，歡迎來到 Moksa 課程！我是你的導師。

「商店頁面」可以說是整個購物網站的門面，顧客在這裡瀏覽你的所有商品。一個清晰、美觀、容易篩選的商店頁面，能大幅提升使用者體驗 (UX)，進而有效拉高轉換率。反之，一個雜亂無章、難以找到目標商品的頁面，則會快速地流失潛在顧客。

在這堂實作課中，我們將學習如何使用 **Moksa 技術棧**中的核心工具——**Kadence Theme** 的自訂功能與 **Kadence Pro** 的進階模組，來徹底客製化你的商店頁面，包含調整商品陳列的排版，以及建立專業級的 AJAX 即時商品篩選器。

---

### 課前準備

在開始之前，請確保你已經完成以下事項：

1.  安裝並啟用 **WooCommerce** 外掛。
2.  安裝並啟用 **Kadence Theme** (免費版) 以及 **Kadence Pro** (付費版)。
3.  已經新增了至少 10-15 個商品，並為這些商品設定了**商品分類**、**商品標籤**以及**商品屬性** (例如：顏色、尺寸)。這是建立篩選器的基礎。

---

### 第一部分：調整商店頁面基礎排版 (Kadence Customizer)

首先，我們先來處理最基本的排版問題：一頁要顯示多少商品？一排要顯示幾個？預設的排序方式是什麼？這些都可以透過佈景主題的「自訂」功能輕鬆完成。

#### 步驟 1：進入 WordPress 自訂器

從 WordPress 後台，點擊左側選單的「**外觀**」>「**自訂**」。

``
*（示意圖：WordPress 後台側邊欄，滑鼠指標停在「外觀 > 自訂」上）*

#### 步驟 2：找到 WooCommerce 商品目錄設定

進入自訂器後，你會在左邊看到設定選單，右邊則是網站的即時預覽畫面。

1.  點擊「**WooCommerce**」。
2.  接著點擊「**商品目錄 (Product Catalog)**」。
3.  在右側的預覽畫面，請手動導覽到你的「商店」頁面，這樣才能即時看到變更效果。

``
*（示意圖：自訂器畫面，左邊顯示 WooCommerce > 商品目錄的選項，右邊預覽商店頁面）*

#### 步驟 3：調整核心排版選項

在「商品目錄」的設定面板中，你會看到幾個關鍵選項，讓我們一一調整：

*   **商店頁面顯示 (Shop Page Display):** 預設是「**顯示商品**」。你也可以選擇「顯示分類」或「顯示分類和商品」。**Moksa 技巧：** 對於大多數電商網站，我們建議維持「**顯示商品**」，讓顧客能第一時間看到產品。
*   **分類顯示 (Category Display):** 當顧客點進某個商品分類頁時，要顯示什麼。同樣建議維持「**顯示商品**」。
*   **預設商品排序 (Default product sorting):** 顧客剛進入商店時，商品的預設排序方式。你可以選擇「熱門程度 (銷量)」、「平均評價」、「依最新商品排序」或「依價格排序」。
*   **每列商品數量 (Products per Row):** 這是最重要的排版設定之一。你可以設定在桌面、平板、手機上，每一行要顯示幾個商品。例如，桌面設為 `4`，平板設為 `3`，手機設為 `2`。
*   **每頁顯示行數 (Rows per Page):** 設定在顧客需要換頁前，總共要顯示幾「行」商品。例如，如果你設定 `4` 行，且每列商品數量為 `4`，那一頁就會顯示 `4 x 4 = 16` 個商品。

完成設定後，別忘了點擊右上角的藍色按鈕「**發佈**」來儲存變更。

**Moksa 技巧：** 在自訂器下方有三個裝置圖示（桌面、平板、手機），善用它們來切換預覽畫面，確保你的排版在所有裝置上都看起來很棒。

### 第二部分：建立專業級 AJAX 商品篩選器 (Kadence Pro)

傳統的篩選器在顧客每點選一個選項後，都會重新整理頁面，體驗非常差。我們要使用的是 **AJAX (Asynchronous JavaScript and XML)** 技術，讓篩選結果可以「**即時**」呈現在頁面上，無需等待頁面刷新。

#### 步驟 1：啟用 Kadence Pro 的 Product Filter 模組

1.  從 WordPress 後台，點擊「**外觀**」>「**Kadence**」。
2.  在 Pro Addons 的清單中，找到「**Product Filter (WooCommerce)**」。
3.  確保它右邊的開關是**啟用**狀態。

``
*（示意圖：Kadence Addons 頁面，Product Filter 模組被啟用）*

#### 步驟 2：將篩選器小工具加入側邊欄 (Sidebar)

啟用模組後，我們就可以在「小工具」區域使用這些新的篩選器了。

1.  從 WordPress 後台，點擊「**外觀**」>「**小工具**」。
2.  找到名為「**商店側邊欄 (Shop Sidebar)**」的小工具區域。如果沒有，你可以透過 `自訂 > WooCommerce > 商品目錄` 中的設定來啟用商店側邊欄。
3.  點擊「+」號按鈕，搜尋並加入以下幾個由 Kadence 提供的篩選器小工具 (Widget)：
    *   **Product Filter - Price:** 價格區間拉霸篩選器。
    *   **Product Filter - Attribute:** **最重要的篩選器**。你可以選擇一個已建立的「商品屬性」（如：顏色、尺寸），並設定其顯示樣式（如：色票、標籤、下拉選單）。
    *   **Product Filter - Category:** 商品分類篩選器。
    *   **Product Filter - Reset:** 提供一個按鈕讓顧客可以一鍵清除所有篩選條件。
    *   **Product Filter - Active Filters:** 顯示目前顧客已套用的所有篩選條件，讓他們可以清楚知道自己篩選了什麼。

你可以重複加入「**Product Filter - Attribute**」小工具，來分別建立「顏色」和「尺寸」的篩選器。

``
*（示意圖：小工具設定畫面，商店側邊欄中已加入了價格、顏色、尺寸等篩選器）*

**Moksa 技巧：** 在新增「**商品屬性**」時（`後台 > 商品 > 屬性`），請務必為每個屬性項目設定好對應的「色碼」或「圖片」，這樣在篩選器中才能顯示為漂亮的**色票 (Color Swatch)**，而不是單純的文字。

#### 步驟 3：前端測試與驗證

回到你的網站前台「商店」頁面，你現在應該會在側邊欄看到剛剛設定好的所有篩選器。

試著操作看看：
*   拖動價格拉霸。
*   點擊不同的顏色色票。
*   勾選一個或多個商品分類。

你會發現，每當你點擊一個篩選條件，右側的商品列表都會**立即更新**，而整個頁面並沒有重新整理。這就是專業電商網站該有的使用者體驗！

``
*（示意圖：最終的商店頁面，左側有漂亮的 AJAX 篩選器，右側是商品列表）*

### 第三部分：使用 Elementor 進行終極客製化 (進階選項)

雖然 Kadence 的自訂功能已經非常強大，但如果你需要一個完全打破常規、獨一無二的商店頁面版型（例如，特殊的 Banner、不對稱的商品格線），那麼你可以使用 **Elementor Pro** 的「**主題建構器 (Theme Builder)**」。

這是一個更進階的作法，它會**完全覆蓋**佈景主題預設的商店頁面範本。

#### 操作簡述：

1.  從後台進入「**Elementor**」>「**範本**」>「**Theme Builder**」。
2.  點擊「**新增**」，選擇範本類型為「**商品封存 (Product Archive)**」。
3.  為你的範本命名，例如「客製化商店頁面」。
4.  進入 Elementor 編輯器後，你可以從零開始拖拉元件來設計。最重要的兩個元件是：
    *   **封存標題 (Archive Title):** 用來顯示「商店」或分類頁的標題。
    *   **商品 (Products):** 用來顯示商品列表，你可以自訂欄位數、分頁樣式等。
5.  設計完成後，點擊「**發佈 (Publish)**」，並在「**顯示條件 (Display Conditions)**」中設定為「**商店頁面 (Shop Page)**」。

**Moksa 技巧：** 我們建議**優先使用 Kadence 自訂器**來調整排版與篩選器。它的整合度最高、效能最好，也最容易維護。只有在 Kadence 無法滿足你極度特殊的設計需求時，才考慮使用 Elementor Theme Builder 來覆蓋商店頁面。
