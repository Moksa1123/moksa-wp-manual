### 為什麼選擇 WordPress 作為開發框架？

在投入大量時間學習之前，我們必須先回答這個最重要的問題。為什麼在 2024 年，一個專業開發者還需要學習 WordPress？

1.  **驚人的市佔率與商業機會**
    *   全世界有**超過 43% 的網站**是使用 WordPress 建立的。這意味著一個龐大到難以想像的市場。無論是接案、求職，或是開發自己的商業產品 (佈景主題/外掛)，WordPress 開發者的需求量都非常大。
    *   Moksa 團隊的許多大型專案，客戶遍及各個產業，底層架構都是 WordPress + WooCommerce。這證明了它完全有能力應對複雜的商業邏輯。

2.  **無可比擬的彈性與擴充性**
    *   WordPress 的核心設計哲學之一就是「可擴充性」。這主要體現在它的**鉤點系統 (Hooks System)**，也就是 `Actions` 和 `Filters`。
    *   **Moksa 技巧：** 你可以把 WordPress 想像成一個樂高城堡，而 **Hooks 就是那些讓你可以隨意插拔、替換、增加新零件的「凸點」**。幾乎所有 WordPress 核心、佈景主題、外掛的行為，都可以透過 Hooks 來客製化，而不需要去修改原始碼，這讓系統升級與維護變得非常容易。
    *   除了 Hooks，**自訂內容類型 (Custom Post Types)**、**自訂分類 (Custom Taxonomies)** 以及強大的 **REST API**，都讓 WordPress 成為一個優秀的無頭後端 (Headless CMS) 解決方案。

3.  **成熟的社群與豐富的文件**
    *   遇到問題時，你幾乎永遠不是第一個遇到的人。從官方的 [Developer Resources](https://developer.wordpress.org/) 到 Stack Exchange，有著數十年的問題與解決方案積累。
    *   絕大多數知名的外掛，例如 WooCommerce、ACF、Rank Math，也都有非常詳盡的開發者文件，教你如何透過程式碼與它們互動。

### 我們將如何重新認識 WordPress？

這堂課不會教你如何在後台點擊按鈕。我們假設你已經熟悉 WordPress 的基本操作。我們要做的是，戴上開發者的「X光眼鏡」，透視 WordPress 的內部結構。

我們會重點關注 WordPress 的**核心四大金剛**：

1.  **資料庫結構 (Database Schema)**
    *   理解 `wp_posts`, `wp_postmeta`, `wp_users`, `wp_usermeta`, `wp_terms` 這些核心資料表的設計理念。**所有內容皆為文章 (Everything is a Post)** 是 WordPress 的一個核心概念，理解這一點至關重要。
    ``

2.  **檔案結構 (File Structure)**
    *   `wp-config.php` 的作用是什麼？`wp-content/themes/` 和 `wp-content/plugins/` 的關係是什麼？`mu-plugins` 又是什麼？我們會徹底搞懂 WordPress 的檔案是如何組織的。

3.  **核心載入流程 (Core Loading Process)**
    *   從使用者發出請求 (Request) 到頁面完全呈現 (Render)，WordPress 內部到底發生了什麼事？它會依序載入哪些檔案？`functions.php` 是在什麼時間點被載入的？理解這個流程，能幫助你把程式碼放在最正確的位置。

4.  **鉤點系統 (The Hook System)**
    *   這是本課程的重中之重。我們將深入學習如何使用 `add_action()` 和 `add_filter()` 來掛載自己的功能，以及如何尋找合適的鉤點來達成我們的客製化需求。**掌握 Hooks，你就掌握了 WordPress 開發的精髓**。

### 本課程的學習路徑與核心技術棧

這是一堂實戰課程，我們將會使用 Moksa 團隊在真實專案中使用的標準化技術棧。

*   **學習路徑概覽：**
    1.  **Part 1: 開發環境建置** - 打造一個專業、高效的本地開發環境。
    2.  **Part 2: 現代化佈景主題開發** - 從零開始，遵循官方標準，開發一個支援區塊編輯器 (Gutenberg) 的高效能佈景主題。
    3.  **Part 3: 物件導向外掛開發** - 學習如何組織你的程式碼，撰寫一個結構清晰、易於維護與擴充的專業外掛。
    4.  **Part 4: WooCommerce 深度客製化** - 包含自訂金流、物流、修改結帳流程、串接第三方 API 等商業實戰。
    5.  **Part 5: 進階主題與部署** - 涵蓋 REST API、WP-CLI、Git 版本控制、CI/CD 自動化部署等進階技巧。

*   **Moksa 核心開發工具棧：**
    *   **本地環境:** **LocalWP**
    *   **程式碼編輯器:** **VS Code** (搭配推薦的擴充套件)

    *   **指令列工具:** **WP-CLI**
    *   **除錯工具:** **Query Monitor**, Xdebug

### 給開發者的第一個建議

在我們正式開始寫程式碼之前，我想給你一個最重要的建議：

> **忘掉「外掛市集」，開始閱讀「原始碼」。**

當你遇到一個功能需求時，作為開發者的第一個反應不應該是「去找找看有沒有外掛可以做到」，而應該是「這個功能的核心邏輯是什麼？它應該掛載在哪個 WordPress 的鉤點上？我該如何自己把它寫出來？」

試著去閱讀 WordPress 核心、WooCommerce、或是你常用的外掛的原始碼。你會從中學到非常多關於「The WordPress Way」的程式碼撰寫風格與設計模式。

``
