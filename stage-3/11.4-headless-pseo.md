### 什麼是 Programmatic SEO？

簡單來說，**Programmatic SEO (程式化 SEO)** 是一種**利用「結構化資料」與「頁面範本」來大規模、自動化地生成大量目標明確的登陸頁面 (Landing Pages) 的策略**。

想像一下傳統的內容產出流程：
1.  研究一個關鍵字。
2.  撰寫一篇文章。
3.  發佈文章。
4.  重複以上步驟。

這個流程對於深度內容非常有效，但如果你要為全台灣 368 個鄉鎮市區都建立一個「[地區] + 推薦咖啡廳」的頁面，手動建立 368 次將會是一場惡夢。

Programmatic SEO 的流程則是：
1.  **收集資料**：建立一個包含所有鄉鎮市區名稱、特色、咖啡廳列表...等的資料庫 (例如一個 Excel/Google Sheet 表格)。
2.  **設計範本**：設計一個「咖啡廳推薦頁」的範本，裡面包含標題、內文、圖片等區塊，但關鍵字（如地區名稱）是動態的變數。
3.  **自動生成**：透過程式或工具，將資料庫中的每一行資料，自動套入範本中，一次性生成 368 個獨特的頁面。

``
*(示意圖：資料庫 (表格) + 頁面範本 = 大量生成的 SEO 頁面)*

這種策略的核心在於**鎖定「長尾關鍵字」**。雖然「台北 咖啡廳」的搜尋量很大，但競爭也非常激烈。而「大安區 安靜 咖啡廳」或「台南 中西區 老宅 咖啡廳」這類長尾關鍵字的搜尋量雖小，但轉換意圖非常高，且競爭較低。pSEO 正是捕捉這些海量長尾流量的最佳武器。

### Programmatic SEO 的核心組成要素

一個成功的 pSEO 專案，離不開以下三個核心要素。在 Moksa 的技術棧中，我們有對應的強大工具來實現它們。

#### 1. 結構化資料庫 (The Database)

這是 pSEO 的靈魂。資料的品質、豐富度和獨特性，直接決定了生成頁面的品質。

*   **來源**：可以是 CSV 檔案、Google Sheets、API 接口，或是你自己整理的資料庫。
*   **範例**：
    *   一個房地產網站的資料庫可能包含：建案名稱、地址、坪數、房型、價格、附近捷運站...
    *   一個旅遊網站的資料庫可能包含：景點名稱、城市、地址、門票價格、營業時間、特色描述...
*   **Moksa 技巧**：我們強烈建議使用 WordPress 原生的方式來管理這些結構化資料，也就是 **Custom Post Types (CPT)** 搭配 **Advanced Custom Fields (ACF)**。將每一筆資料（例如一個建案、一個景點）都當作一個 CPT 的文章，並用 ACF 來儲存它的結構化欄位（價格、坪數等）。這樣做不僅管理方便，也讓後續的範本製作和資料查詢更有彈性。

#### 2. 頁面範本 (The Template)

這是 pSEO 的骨架。範本決定了所有生成頁面的外觀、排版和內容結構。

*   **設計重點**：
    *   **使用者體驗 (UX)**：頁面必須清晰、易於閱讀且對使用者有真正的價值。
    *   **獨特性**：範本中除了動態變數外，應盡可能加入有價值的靜態內容、獨特的組合邏輯，避免讓 Google 判定為「樣板化內容 (Boilerplate Content)」。
    *   **CTA (Call to Action)**：明確的行動呼籲，引導使用者進行下一步（例如：預約看房、購買門票）。
*   **Moksa 技巧**：你可以使用 **Kadence Blocks** 或 **Elementor Pro** 的主題建構功能 (Theme Builder) 來為你的 CPT 打造頁面範本。它們強大的「**動態內容 (Dynamic Content)**」功能，可以讓你輕鬆地在頁面中插入來自 ACF 的欄位資料，無需撰寫任何程式碼。

``
*(示意圖：使用 Elementor 的動態標籤功能，將 ACF 欄位 `project_price` 插入到標題中)*

#### 3. 生成機制 (The Engine)

這是將資料與範本結合，並實際產出頁面的引擎。

*   **方法**：
    *   **外掛匯入**：使用像 `WP All Import` 這類外掛，將你的 CSV 或 XML 檔案匯入，並在過程中將資料欄位對應到 CPT 的標題、內文、ACF 欄位，甚至是 **Rank Math SEO** 的欄位。這是最適合初學者的入門方式。
    *   **客製化腳本**：對於更進階的開發者，可以撰寫自己的 PHP 腳本，透過 `wp_insert_post()` 函式搭配 `update_field()` (ACF 函式) 來程式化地建立文章。
    *   **WP-CLI**：這是在終端機環境下操作 WordPress 的命令列工具。你可以撰寫 Shell Script 結合 WP-CLI 指令來大量生成內容，效率極高，適合需要處理數十萬筆資料的超大型專案。

### 一個完整的 pSEO 專案思考流程

讓我們以建立一個「全台主題餐廳介紹網站」為例，走一遍標準的規劃流程。

*   **步驟 1: 關鍵字與資料研究**
    *   **目標關鍵字**：鎖定 `[城市] + [主題] + 餐廳` 的格式，例如：「台北 親子 餐廳」、「台中 寵物友善 餐廳」。
    *   **資料搜集**：建立一個 Google Sheet，欄位包含：**餐廳名稱**、**城市**、**行政區**、**主題分類** (親子/寵物/動漫...)、**地址**、**電話**、**平均消費**、**特色描述 (至少 150 字)**、**照片 URL**。
    *   **Moksa 技巧**：**資料品質是成敗關鍵！** 不要只複製貼上。特色描述盡量用自己的話重新組織，確保內容的獨特性和價值。

*   **步驟 2: 建立 WordPress 資料結構**
    *   使用 CPT UI 外掛，建立一個名為「主題餐廳 (restaurants)」的 CPT。
    *   使用 ACF 外掛，為這個 CPT 建立對應的欄位群組，包含：城市、主題分類、地址、電話...等。

*   **步驟 3: 設計頁面與 SEO 範本**
    *   使用 Elementor Pro 的 Theme Builder，為「主題餐廳」這個 CPT 建立一個「單一文章範本 (Single Post Template)」。
    *   在範本中，使用 Elementor 的動態內容功能，將標題設為 `[餐廳名稱] - [城市][主題]主題餐廳推薦`。
    *   將 ACF 建立的各個欄位，拉到頁面範本的對應位置。
    *   **Moksa 技巧**：進入 **Rank Math SEO** 的「標題與 Meta」設定，找到你的 CPT (主題餐廳)。在「單一標題」欄位中，設定一個公式，例如：`%title% - %custom_field(restaurant_city)% %custom_field(restaurant_theme)% 餐廳推薦 | 網站名稱`。這樣每一篇自動生成的文章，都會有獨一無二且高度優化的 SEO Title。

``
*(示意圖：在 Rank Math 的標題與 Meta 設定中，為 CPT 設定動態標題範本)*

*   **步驟 4: 生成頁面**
    *   將 Google Sheet 下載為 CSV 檔案。
    *   使用 `WP All Import` 外掛，上傳 CSV 檔，並將 CSV 的每一欄，拖拉對應到 WordPress 的欄位（文章標題、ACF 欄位、Rank Math 欄位等）。
    *   執行匯入，幾分鐘內，數百個介紹頁面就建立完成了！

*   **步驟 5: 內部連結與優化**
    *   **建立彙總頁 (Hub Page)**：建立一個「台北餐廳總覽」頁面，列出所有台北的餐廳。再建立一個「親子餐廳總覽」頁面，列出所有親子主題的餐廳。這有助於權重傳遞與使用者導覽。
    *   **監控成效**：使用 Google Search Console 觀察哪些頁面開始獲得曝光和點擊，並針對表現好的頁面進行內容的再優化。

### 常見陷阱與注意事項

Programmatic SEO 是一把雙面刃，用得好能帶你上天堂，用得不好可能會讓網站被 Google 懲罰。

1.  **避免低品質內容**：如果你的資料只是簡單的名稱和地址，生成的頁面將會非常空洞，這就是所謂的「Thin Content」。**確保你的資料庫有豐富且具備獨特價值的內容欄位**。
2.  **避免重複內容**：確保你生成的每個頁面，其核心內容（特別是標題和主要描述）都有顯著差異。不要只是替換一個地名，而其他 99% 的文字都相同。
3.  **注意伺服器負載與索引**：一次生成上萬個頁面，可能會對伺服器造成壓力。同時，Google 也需要時間來爬取和索引這些新頁面。**良好的網站架構、內部連結和提交 Sitemap 是讓頁面被順利索引的關鍵**。

### 結論

Programmatic SEO 是一種**思維模式**，它將內容創作從「手工業」轉變為「工業化生產」。它要求你從單點的內容創作，提升到系統性的**資料治理**和**範本設計**。

在 Moksa 的生態系中，透過 **CPT + ACF** 進行資料建模，**Kadence/Elementor** 進行視覺範本設計，以及 **Rank Math** 進行 SEO 範本優化，你已經擁有實踐 pSEO 的所有利器。
